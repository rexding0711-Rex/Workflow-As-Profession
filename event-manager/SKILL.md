# 活动管理（Event Management）

## 核心能力

活动是将品牌、产品、关系"打包为一场体验"。
本 Skill 覆盖从策划到复盘的全流程，适用于线上和线下活动。

## 工作流

### Step 1：目标与范围定义

回答三个问题：
1. **Why** — 为什么搞这个活动？（品牌曝光？获客？老客维护？融资路演？）
2. **Who** — 谁来？（目标参与人画像、预期人数、是否有VIP）
3. **What** — 产出什么？（品牌认知提升？N条销售线索？N份合作意向？）

### Step 2：活动方案设计（5W2H）

```
活动名称：
活动主题：
时间：____年__月__日（备选日期：____）
地点：____（线上平台：腾讯会议/Zoom/视频号）
规模：____人
形式：论坛 / 沙龙 / 发布会 / 晚宴 / 展会 / 线上直播 / 混合
预算：____元（场地__%、嘉宾__%、餐饮__%、物料__%、传播__%、应急__%）
```

### Step 3：活动策划清单

**2-4 周前**（策划期）：
- [ ] 确定活动目标、主题、形式
- [ ] 预定场地/线上平台
- [ ] 确认嘉宾/讲者（发邀请函、确认档期）
- [ ] 设计主视觉（KV）、物料清单
- [ ] 发布活动预告（公众号/朋友圈/EDM/社群）

**1 周前**（准备期）：
- [ ] 物料到位（背景板、桌牌、礼品、宣传册）
- [ ] 嘉宾PPT收集与审核
- [ ] 活动流程表定稿（精确到分钟）
- [ ] 现场分工表（签到/引导/摄影/控台/主持人）
- [ ] 发送参会提醒（含交通/停车/议程信息）

**活动当天**（执行期）：
- [ ] 提前 2h 到场布置
- [ ] 设备测试（投影/音响/麦克风/直播推流）
- [ ] 签到台就位、VIP接待就位
- [ ] 按流程表执行（每环节有人盯时间）
- [ ] 实时拍照/短视频素材采集

**活动后 3 天**（收尾期）：
- [ ] 活动回顾推文/新闻稿发布
- [ ] 参与者反馈收集（问卷）
- [ ] 销售线索跟进分配
- [ ] 供应商结款
- [ ] 活动复盘会（预算执行率、目标达成率、改进点）

### Step 4：预算模板

| 科目 | 占比 | 金额 |
|------|------|------|
| 场地费 | 15-25% | |
| 嘉宾费（出场+差旅） | 15-20% | |
| 餐饮 | 20-30% | |
| 物料（设计+制作） | 10-15% | |
| 传播（摄影/直播/推流） | 5-10% | |
| 应急储备 | 10% | |

### Step 5：风险评估

| 风险 | 预案 |
|------|------|
| 嘉宾临时取消 | 准备 2 位备选嘉宾 |
| 报名不足 | 活动前 1 周启动社群push |
| 现场技术故障 | 备用设备 + 技术彩排 |
| 天气原因（户外） | 室内备选方案 |

## 注意事项
- 活动的 ROI 不只看出席人数，看"有效触达"和"后续转化"
- 小活动（<30人）做深度，大活动（>100人）做广度
- 线上活动录播的回放价值 = 活动的长尾价值
