# 高难度对话话术指南

## 通用原则

1. **直接但不冷酷**：第一句话就说明来意，不要绕弯子
2. **面对面**：裁员/警告必须当面或视频，禁止邮件/短信通知
3. **准备充分**：提前准备所有实际问题的答案
4. **简短**：核心信息 5-8 分钟说完，不要展开讨论
5. **二次会谈**：给 1-2 天消化期，安排 follow-up 解答后续问题

## 场景 1：裁员沟通

### 话术框架

```
"[姓名]，很遗憾我今天要和你说一个艰难的决定。

经过公司慎重考虑，[原因一句话，不展开]。

这不是对你个人工作的评判。[肯定贡献的一句话]。

接下来关于过渡安排：
- 最后工作日：[日期]
- 补偿方案：[简要说明]
- 交接计划：[简要说明]
- 离职支持：[outplacement/EAP/推荐信等]

你现在有什么问题吗？我会解答所有实际问题。"
```

### 常见追问与回答

| 追问 | 回答要点 |
|------|---------|
| "为什么是我？" | 如果无法避免回答："这是基于 [业务/架构] 的决定，不是个人表现问题。" 如需保密不要展开 |
| "还有商量的余地吗？" | "这个决定已经是最终的。我理解这很难接受。" |
| "我能和 XXX 谈一下吗？" | "当然，我可以帮你安排。不过在正式流程完成前，建议你先消化一下。" |

### 绝对不要说的话

- ❌ "我能理解你的感受"（除非你真的经历过）
- ❌ "这对你也是个机会"（不是你说这话的时候）
- ❌ 把责任推给他人（"是老板/公司的决定"）
- ❌ 讨论其他受影响的人

## 场景 2：绩效警告

### 话术框架

```
"[姓名]，我今天想和你聊聊最近的一些观察。

在 [具体时间/项目] 中，我注意到 [具体行为/结果]。
这和我们对这个岗位的期望有差距，具体来说 [明确期望]。

我想帮你改进。接下来 [X 天/周]，我期待看到 [具体行为改变]。
我可以提供的支持有 [具体帮助]。

我们会做一个正式的 PIP（绩效改进计划），下周我们一起过一遍。
两周后我们再 check-in 看进展。

你有什么想说的吗？"
```

### 关键技巧

- 用数据/案例而非感觉："你在过去 4 次 sprint 中，有 3 次未按时完成分配的任务"
- 给具体期望而非抽象要求："我希望你每天站会前更新 Jira 状态"而非"希望你更主动"
- 提供支持而非只说问题："我可以帮你找 XXX 做代码 review buddy"

## 场景 3：性骚扰投诉接收

### 话术框架

```
"[姓名]，感谢你信任我，愿意和我分享这件事。

我想先确认：你现在感觉安全吗？你需要什么即刻的支持？

接下来我会认真听你说完，不做评判。然后我会告诉你我们会怎么处理。

[倾听...]

感谢你告诉我这些。以下是我们接下来的步骤：
1. [如果涉及可能的违法行为] 根据公司政策，我需要向 [法务/HR Director] 报告。
   我会保护你的隐私，只告知必要的人。
2. 我们会启动正式调查 [或：你希望先非正式处理吗？]
3. 调查期间，我们会确保 [保护措施，如：调整汇报线/工作安排]
4. 你会全程被告知进展。

这是我的联系方式，你随时可以找我。
此外，公司的 EAP（员工帮助计划）提供免费心理咨询 [如有]。
"
```

### 绝对原则

- 永远不要说"你是不是想多了"
- 永远不要建议"你们自己解决吧"
- 永远不要承诺"我保证不会告诉任何人"（法律义务可能需要报告）
- 如有任何不确定 → 咨询法务

## 场景 4：纪律处分通知

### 话术框架

```
"[姓名]，基于我们之前的讨论和调查结果，我今天需要正式告知你一个决定。

关于 [事件简述]，我们做了 [调查流程]。

调查结论是 [简述]。

根据公司政策，我们的决定是 [具体处分：书面警告/降级/停职]，
生效日期 [日期]，[如适用] 为期 [时长]。

这个决定的依据是 [公司政策条款]。

如果你对这个决定有异议，你可以在 [X 天] 内向 [HR Director / 申诉委员会] 提出正式申诉。

你有什么问题吗？"
```
