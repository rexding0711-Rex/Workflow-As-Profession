# 内部调查协议

## 原则

1. **保密**：仅告知"需要知情"的人。调查期间禁止在非必要范围内讨论案件。
2. **中立**：调查人必须无利益冲突，非任何一方的直属汇报线。
3. **及时**：投诉后 48 小时内启动调查。
4. **书面**：所有访谈、证据、决策都必须有书面记录（假设未来在法庭上被审查）。
5. **禁止报复**：任何人因善意投诉而受到报复，报复者从重处理。

## 严重程度分级

### Level 1：低风险
- **类型**：人际冲突、沟通风格分歧、轻微不当言论
- **处理人**：直属 Manager + HRBP
- **流程**：调解 → 记录 → 跟进
- **时间线**：1-2 周

### Level 2：中风险
- **类型**：持续的不当行为、歧视性言论、违规但非违法
- **处理人**：HR Director + 法务（可选）
- **流程**：正式调查 → 纪律处分决定 → 反馈闭环
- **时间线**：2-4 周

### Level 3：高风险
- **类型**：性骚扰、职场暴力、贪污、违法行为、报复行为
- **处理人**：法务 + 外部律师 + 必要时报警
- **流程**：紧急行动（保护投诉人）→ 正式调查 → 法律建议 → 决策
- **时间线**：立即行动，调查 1-2 周
