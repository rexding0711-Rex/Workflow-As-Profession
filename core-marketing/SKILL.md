# 核心营销 Skill

## 何时触发

用户请求：
- 学习MBA市场营销课程
- 理解消费者行为和决策过程
- 品牌定位和营销策略设计
- 市场细分和目标市场选择
- "怎么理解消费者" / "品牌定位怎么做"

## 核心框架：STP + 4P + 消费者决策旅程

### 步骤1：市场细分（Segmentation）

**细分维度**：
- **地理**：国家、地区、城市、气候
- **人口**：年龄、性别、收入、教育、职业
- **心理**：生活方式、价值观、个性
- **行为**：使用频率、忠诚度、利益追求、购买时机

**细分有效性标准**：
- 可衡量（Measurable）
- 可到达（Accessible）
- 足够大（Substantial）
- 可区分（Differentiable）
- 可操作（Actionable）

**经典细分案例**：
- 宝洁洗发水：海飞丝（去屑）、飘柔（柔顺）、潘婷（营养）
- 汽车：经济型、家庭型、运动型、豪华型
- 航空：头等舱、商务舱、经济舱（价格细分）

### 步骤2：目标市场选择（Targeting）

**选择策略**：
- **无差异**：整个市场，一种产品（可口可乐早期）
- **差异化**：多个细分市场，不同产品（宝洁）
- **集中**：一个细分市场，深耕（lululemon早期瑜伽）
- **微营销**：个人定制（一对一）

**评估细分市场**：
- 市场规模和增长率
- 竞争强度
- 与公司能力和资源的匹配度
- 进入壁垒

### 步骤3：定位（Positioning）

**定位三角**（同Bullshit包，但更深）：
- 目标消费者（Who）
- 竞争框架（What category）
- 差异化点（Why different）
- 信任状（Why believe）

**定位声明模板**：
```
对于[目标消费者]，
[品牌]是[竞争框架]，
它能够[差异化利益]，
因为[信任状/原因]。
```

**定位地图（Perceptual Map）**：
- 选择两个关键维度
- 将品牌和竞品放在坐标上
- 寻找空白区域（定位机会）

### 步骤4：消费者行为分析

**消费者决策旅程**：
```
认知 → 考虑 → 购买 → 体验 → 忠诚 → 倡导
```

**影响消费者决策的因素**：
- **文化因素**：文化、亚文化、社会阶层
- **社会因素**：参照群体、家庭、角色与地位
- **个人因素**：年龄、职业、经济状况、生活方式
- **心理因素**：动机、感知、学习、信念、态度

**购买决策类型**：
- **复杂决策**：高介入、高差异（房子、车）
- **减少失调**：高介入、低差异（地毯、家具）
- **习惯性购买**：低介入、低差异（盐、牙膏）
- **寻求多样性**：低介入、高差异（零食、饮料）

### 步骤5：营销组合（4P）

**Product（产品）**：
- 核心产品、实体产品、 augmented产品
- 产品生命周期：引入→成长→成熟→衰退
- 产品组合：宽度、长度、深度、一致性
- 品牌：名称、标志、定位、延伸

**Price（定价）**：
- 成本加成：成本 + 目标利润
- 价值定价：基于顾客感知价值
- 竞争定价：参考竞争对手
- 撇脂定价：高价入市，逐步降价
- 渗透定价：低价入市，快速占领
- 心理定价：9.99 vs 10
- 价格歧视：一级（完全）、二级（数量）、三级（群体）

**Place（渠道）**：
- 渠道层级：零级（直销）→ 一级 → 二级 → 三级
- 渠道宽度：独家、选择、密集
- 渠道冲突：水平冲突、垂直冲突、多渠道冲突
- 全渠道（Omni-channel）整合

**Promotion（推广）**：
- 广告：reach, frequency, impact
- 销售促进：短期刺激（coupon, 折扣）
- 公共关系：新闻、事件、 sponsorship
- 人员推销：B2B关键
- 直销：邮件、电话、数字
- 整合营销传播（IMC）：统一信息，多渠道协同

### 步骤6：品牌管理

**品牌资产（Brand Equity）**：
- 品牌知名度（Brand Awareness）
- 品牌联想（Brand Associations）
- 感知质量（Perceived Quality）
- 品牌忠诚度（Brand Loyalty）
- 其他品牌资产（专利、渠道关系）

**品牌价值链**：
```
营销投入 → 顾客心智 → 市场绩效 → 股东价值
```

**品牌决策**：
- 品牌延伸：利用品牌进入新品类（风险：品牌稀释）
- 多品牌：同一公司多个品牌（宝洁）
- 新品牌：新品类新品牌（避免负面联想）
- 联合品牌：两个品牌合作（Sony Ericsson）

### 步骤7：服务营销

**服务特性**：
- 无形性（Intangibility）→ 有形展示
- 不可分离性（Inseparability）→ 服务提供者重要
- 可变性（Variability）→ 标准化和培训
- 易逝性（Perishability）→ 需求管理

**服务质量差距模型**：
```
顾客期望 → 管理者感知 → 服务标准 → 服务传递 → 外部沟通 → 顾客感知
```
- 差距1：管理者不了解顾客期望
- 差距2：服务标准未满足期望
- 差距3：服务传递未达标准
- 差距4：承诺过度，交付不足
- 差距5：顾客感知与期望的差异（总差距）

## 输出模板

```markdown
# [品牌/产品] 营销策略分析

## 1. STP分析
- 市场细分：[维度/细分群体]
- 目标市场：[选择/原因]
- 定位声明：[一句话]
- 定位地图：[维度1/维度2，品牌位置]

## 2. 消费者洞察
- 目标消费者：[画像]
- 决策旅程：[关键触点]
- 关键影响因素：[文化/社会/个人/心理]
- 未被满足需求：[是什么]

## 3. 4P策略
- Product：[产品策略/组合/生命周期]
- Price：[定价策略/方法]
- Place：[渠道策略/覆盖]
- Promotion：[推广组合/IMC]

## 4. 品牌分析
- 品牌资产：[知名度/联想/质量/忠诚度]
- 品牌策略：[延伸/多品牌/联合]
- 品牌价值链：[营销投入→顾客心智→市场绩效]

## 5. 服务营销（如适用）
- 服务质量差距：[差距1-5评估]
- 改进方向：[具体措施]

## 6. 竞争定位
- 主要竞品：[列表]
- 差异化优势：[是什么]
- 防御壁垒：[品牌/渠道/成本/技术]

## 7. 建议
- 短期：[行动]
- 中期：[行动]
- 长期：[行动]
```

## 经典案例

- **可口可乐 vs 百事**：百年可乐战争，定位差异（经典 vs 年轻）
- **星巴克**：从咖啡到"第三空间"，体验营销
- **苹果**：品牌极简主义，生态系统锁定
- **耐克**："Just Do It"，情感营销+运动员代言
- **宝洁**：多品牌战略，品类管理
- **小米**：互联网营销，粉丝经济，性价比定位
- **海底捞**：服务差异化，口碑营销

## 使用示例

**用户输入**：
> "帮我分析星巴克的营销策略，它是怎么从卖咖啡变成卖体验的？"

**AI执行**：
1. STP：从"咖啡爱好者"到"寻求第三空间的人"
2. 定位：从"优质咖啡"到"工作和家之间的第三空间"
3. 4P：产品标准化+定制化，溢价定价，位置至上，体验式营销
4. 品牌：从功能品牌到生活方式品牌
5. 服务：环境+服务+产品的整体体验
6. 讨论品牌延伸（咖啡→食品→音乐→空间）的成功和风险
