# 核心战略 Skill

## 何时触发

用户请求：
- 学习MBA战略管理课程
- 分析竞争环境和战略选择
- 理解战略的理论基础和历史演进
- 公司战略和多元化决策
- "战略到底是什么" / "怎么制定战略"

## 核心框架：战略层次 + 竞争分析 + 战略选择 + 执行

### 步骤1：战略层次

**公司层战略（Corporate Strategy）**：
- 我们在什么行业竞争？（行业选择）
- 我们如何在这些行业之间配置资源？（资源配置）
- 我们如何创造协同效应？（价值创造）
- 战略类型：增长、稳定、收缩、组合

**业务层战略（Business Strategy）**：
- 我们在特定行业如何竞争？（竞争定位）
- 如何建立和维持竞争优势？
- 战略类型：成本领先、差异化、聚焦

**职能层战略（Functional Strategy）**：
- 各职能部门如何支持业务战略？
- 营销战略、运营战略、财务战略、人力资源战略

**战略层级关系**：
```
公司层战略 → 业务层战略 → 职能层战略
（做什么）    （怎么做）     （具体执行）
```

### 步骤2：战略分析工具

**外部环境分析**：
- **PESTEL**：政治、经济、社会、技术、环境、法律
- **五力模型（Porter）**：
  - 现有竞争者
  - 潜在进入者（进入壁垒）
  - 替代品威胁
  - 供应商议价能力
  - 买方议价能力
- **战略群组（Strategic Groups）**：行业内按战略相似性分组
- **关键成功因素（KSF）**：在该行业成功必须具备的条件

**内部环境分析**：
- **VRIO框架**：资源是否 valuable（有价值）、rare（稀缺）、inimitable（不可复制）、organized（组织化利用）
- **资源基础观（RBV）**：企业是资源组合，竞争优势来自独特资源
- **能力（Capability）**：整合资源完成活动的能力
- **核心竞争力（Core Competence）**：
  - 能为多个业务提供价值
  - 难以模仿
  - 不可替代
  - 如：本田的发动机技术、3M的材料科学、苹果的工业设计
- **价值链（Value Chain）**：
  - 主要活动：进料物流→生产→出货物流→营销→服务
  - 支持活动：采购、技术开发、HR、基础设施
  - 分析每个环节的成本和差异化潜力

**SWOT分析**：
- 不是列表，而是匹配
- SO策略：利用优势抓住机会
- WO策略：克服弱点抓住机会
- ST策略：利用优势规避威胁
- WT策略：减少弱点并规避威胁

### 步骤3：竞争战略

**波特竞争战略**：
- **成本领先**：全行业最低成本（沃尔玛、富士康）
- **差异化**：独特价值，溢价定价（苹果、特斯拉）
- **成本聚焦**：细分市场的低成本（区域性折扣店）
- **差异化聚焦**：细分市场的差异化（豪华跑车）
- ** stuck in the middle**：没有明确战略，成本不低，差异不大 → 最危险

**竞争定位**：
- **战略时钟（Bowman）**：价格-价值定位
  - 路线1：低价低值（集中成本领先）
  - 路线2：低价（成本领先）
  - 路线3：混合（低价+差异化）→ 通常不可持续
  - 路线4：差异化（同价高值）
  - 路线5：聚焦差异化（高价高值）
  - 路线6：高价低值（收割）→ 短期不可持续
  - 路线7：高价标准值（垄断/品牌溢价）
  - 路线8：标准价低值（失败）

**竞争动态**：
- **蓝海战略（Kim & Mauborgne）**：
  - 红海：现有市场空间，竞争激烈
  - 蓝海：新市场空间，无竞争
  - 价值创新：同时追求差异化和低成本
  - 工具：四步动作框架（剔除-减少-增加-创造）
- **竞争优势的持续性**：
  - 模仿壁垒：规模经济、学习曲线、专利、品牌、网络效应
  - 竞争对手分析：未来目标、假设、当前战略、能力

### 步骤4：公司战略

**增长战略**：
- **一体化**：
  - 纵向一体化：前向（向客户端）/后向（向供应商端）
  - 横向一体化：收购竞争对手
- **多元化**：
  - 相关多元化：进入相关产业（协同效应）
  - 非相关多元化：进入无关产业（分散风险）
  - 多元化动机：增长、风险分散、协同、资源利用、管理者利益
- **国际化**：
  - 进入模式：出口、许可、特许经营、合资、全资子公司
  - 全球化战略：全球一体化 vs 本地响应
  - 跨国战略：同时追求全球效率和本地适应

**战略联盟**：
- 动机：资源互补、风险分担、市场进入、学习
- 类型：合资、股权联盟、非股权联盟（合同）
- 风险：合作伙伴依赖、冲突、知识泄露
- 成功因素：互补性、信任、沟通、承诺

**并购战略**：
- 动机：协同效应、市场力量、多元化、资源获取、管理者利益
- 类型：横向、纵向、混合
- 估值：DCF、可比交易、溢价分析
- 整合：文化整合、组织整合、人力资源整合
- 成功率：多数并购失败或未达到预期（50-70%失败率）
- 失败原因：高估协同、文化冲突、整合不善、支付过高

**收缩战略**：
- 剥离（Divestiture）：出售业务单元
- 清算（Liquidation）：关闭并出售资产
- 收缩重组（Retrenchment）：削减成本、裁员、精简
- 原因：业务不再适合、财务压力、战略调整

### 步骤5：战略执行

**战略与结构**：
- Chandler定律：结构跟随战略
- 简单结构（创业期）→ 职能结构（成长期）→ 事业部结构（多元化）→ 矩阵结构（复杂）→ 网络结构（现代）

**战略控制系统**：
- 财务控制：ROI、预算、利润目标
- 行为控制：规则、SOP、标准化
- 文化控制：价值观、共享信念、社会化
- 平衡计分卡（Balanced Scorecard）：
  - 财务：股东如何看待我们？
  - 客户：客户如何看待我们？
  - 内部流程：我们需要在哪些方面卓越？
  - 学习成长：我们如何持续改进和创造价值？

**战略变革**：
- 变革阻力：习惯、安全感、经济因素、信息选择性处理
- 变革推动者：外部顾问（客观但有距离）vs 内部管理者（了解但可能有偏见）
- 变革步骤：解冻→改变→再冻结（Lewin）

**战略创新**：
- 动态能力（Dynamic Capabilities）：感知、捕捉、重构
- 敏捷战略：快速适应变化
- 开放式创新：利用外部创意和路径
- 商业模式创新：改变价值创造和捕获方式

## 经典案例

- **通用电气（GE）**：杰克·韦尔奇的#1或#2战略，多元化到专业化转型
- **苹果公司**：从PC到iPod到iPhone，生态系统战略，差异化极致
- **沃尔玛**：成本领先战略，规模经济，供应链效率
- **西南航空**：聚焦战略（短途、点对点、单一机型），低成本+高服务
- **IBM**：从硬件到服务到云计算，多次战略转型
- **诺基亚**：从造纸到橡胶到电信到手机到失败，技术战略失败
- **柯达**：发明数码相机但失败，战略惯性
- **亚马逊**：从在线书店到万货商店到AWS，长期主义战略
- **华为**：从代理到自研到全球化，农村包围城市战略

## 使用示例

**用户输入**：
> "帮我分析特斯拉的竞争战略，它是怎么在汽车行业建立差异化优势的？"

**AI执行**：
1. 战略层次：公司层（多元化：车+能源+AI），业务层（差异化+技术领先）
2. 五力分析：汽车行业竞争格局，新进入者（电动车），替代品（公共交通）
3. VRIO：品牌价值、技术能力、超级工厂、数据网络
4. 竞争战略：差异化聚焦（高端电动车），逐步向成本领先迁移（Model 3/Y）
5. 动态能力：快速迭代、软件定义汽车、垂直整合
6. 战略风险：竞争加剧、规模化挑战、品牌稀释
