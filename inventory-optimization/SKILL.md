# Inventory Optimization — 库存优化

库存是"必要的恶"——太多占用资金和仓库，太少导致停线和缺货。
库存优化的目标是找到那个"刚好"的点。

## 何时触发

用户提及：inventory optimization、库存优化、安全库存、EOQ、ABC 分析、
呆滞库存、库存周转、缺货、库存积压 等关键词时触发。

## ABC 分类

| 类别 | 占 SKU% | 占金额% | 管理策略 |
|------|---------|---------|---------|
| A 类 | 10-20% | 70-80% | 严格管控、频繁盘点、精确预测 |
| B 类 | 20-30% | 15-20% | 定期检查、适度安全库存 |
| C 类 | 50-70% | 5-10% | 简化管理、较高安全库存、两箱法 |

## 核心公式

### 安全库存
```
SS = Z × σ × √LT
Z = 服务水平系数（95%→1.65, 99%→2.33）
σ = 需求标准差
LT = 提前期（Lead Time）
```

### EOQ（经济订货批量）
```
EOQ = √(2DS/H)
D = 年需求量
S = 每次订货成本
H = 单位年持有成本
```

### 呆滞库存判定
- 超过 [X] 天无出库记录（X 因行业而异，制造通常 90-180 天）
- 库龄 > 保质期 50%
- 对应产品已 EOL

## 质量原则
1. A 类物料月度盘点、B 类季度、C 类半年
2. 库存周转率目标因行业而异（制造业 4-8 次/年）
3. 呆滞库存 > 货值 5% = 严重问题
