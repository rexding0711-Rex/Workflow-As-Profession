# HR 指标计算公式

## 流失指标

### 整体流失率 (Overall Attrition Rate)
```
离职人数 / ((期初人数 + 期末人数) / 2) × 100%
```
- 时间窗口：月度 / 季度 / 年度
- 分母用平均人数而非期末人数（更准确）

### 主动流失率 (Voluntary Attrition Rate)
```
主动离职人数 / 平均在职人数 × 100%
```
- 排除裁员、退休、死亡等非自愿离职
- 关注"可避免的主动离职"：因管理/薪酬/成长空间可改善而离职的人

### 新人首年流失率 (First-Year Attrition)
```
入职 <12 个月离职人数 / 过去 12 个月入职总人数 × 100%
```
- 如果 >30%，说明 onboarding 或招聘匹配度有问题

### 关键人才流失率 (Regretted Attrition)
```
高绩效 + 高潜离职人数 / (高绩效 + 高潜总人数) × 100%
```
- "可遗憾的离职"：公司其实想留但没留住的人
- 这个指标比整体流失率更有管理意义

## 招聘指标

### 招聘漏斗转化率
```
阶段通过人数 / 上一阶段人数 × 100%
```
- 逐阶段计算，找出瓶颈

### Time to Fill
```
Offer 接受日期 - JD 发布日期
```
- 包含 sourcing、筛选、面试、offer 全流程

### Time to Hire
```
Offer 接受日期 - 候选人进入流程日期
```
- 仅测量从候选人进入漏斗到接受的时间

### Offer Acceptance Rate
```
接受的 Offer 数 / 发出的 Offer 数 × 100%
```

### Cost per Hire
```
(内部招聘成本 + 外部招聘成本) / 录用人数
```
- 内部：招聘团队薪酬分摊 + 招聘工具 + 面试官时间成本
- 外部：猎头费 + 招聘广告 + 候选人差旅

### Source Effectiveness
```
各渠道录用人数 / 各渠道候选人总数 × 100%
```
- 评估哪个招聘渠道性价比最高

## 人效指标

### 人均营收 (Revenue per Employee)
```
总营收 / FTE（全职等效人数）
```
- FTE = 全职人数 + (兼职总工时 / 标准工时)
- 按季度/年度计算

### 人均利润 (Profit per Employee)
```
净利润 / FTE
```
- 净利润 = 总营收 - 总成本（含人力成本）

### 人力成本占比 (People Cost Ratio)
```
(总薪酬 + 福利 + 招聘成本 + 培训成本) / 总营收 × 100%
```
- SaaS 行业：成长期 50-65%，成熟期 35-50%

### 人才 ROI (Return on People Investment)
```
(总营收 - 非人力成本) / 总人力成本
```
- 非人力成本 = 基础设施、营销、行政等
- 健康值 >2.0

### 人均营收增长率 vs 人数增长率
```
营收增长率 / 人数增长率
```
- >1.5：人效在提升
- 0.8-1.2：同步增长
- <0.8：人效下降，需关注

## 员工敬业度指标

### eNPS (Employee Net Promoter Score)
```
推荐者% - 贬损者%
```
- "你有多大可能向朋友推荐我们公司？"（0-10 分）
- 9-10 = 推荐者；7-8 = 中立；0-6 = 贬损者
- 行业基准：科技公司中位数 10-30

### 缺勤率 (Absenteeism Rate)
```
未计划缺勤天数 / 总工作日 × 100%
```
- 仅计算未计划的病假和事假（不含年假、法定假日）
- 健康范围：1.5-3%

## 数据质量问题

计算前确认：
- 人数统计口径一致（含/不含外包？含/不含实习生？）
- 时间窗口一致（自然月 vs 滚动 30 天？）
- 离职分类标准明确（主动/被动/退休/死亡）
- FTE 折算标准明确（兼职怎么算？half-time = 0.5 FTE）
