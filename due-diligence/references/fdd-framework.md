# 财务尽调框架（FDD）

## 对标四大方法论

四大会计师事务所（德勤、普华永道、安永、毕马威）的财务尽调通常遵循以下框架：

## 一、尽调范围界定

### 1. 确定尽调范围

| 维度 | 选项 | 说明 |
|------|------|------|
| 时间范围 | 历史3年+当年预测 | 验证历史趋势，评估预测合理性 |
| 业务范围 | 合并报表/单体报表 | 是否包含子公司、关联公司 |
| 科目范围 | 全科目/重点科目 | 根据交易金额和风险确定 |
| 深度 | 全面尽调/有限尽调 | 全面=详细核查，有限=依赖管理层 |

### 2. 数据需求清单

| 类别 | 资料清单 | 用途 |
|------|----------|------|
| 基础财务 | 审计报告、财务报表、科目余额表 | 基础分析 |
| 税务 | 纳税申报表、税务检查报告、税收优惠批文 | 税务合规 |
| 银行 | 银行对账单、银行询证函、贷款合同 | 现金验证 |
| 收入 | 销售台账、合同台账、收入确认政策 | 收入验证 |
| 成本 | 成本明细、采购台账、供应商清单 | 成本验证 |
| 资产 | 固定资产清单、存货盘点表、应收账款账龄 | 资产验证 |
| 负债 | 借款合同、担保合同、应付账款账龄 | 负债验证 |
| 关联方 | 关联交易清单、关联方往来 | 关联交易 |
| 预算 | 年度预算、预算执行情况 | 预测验证 |
