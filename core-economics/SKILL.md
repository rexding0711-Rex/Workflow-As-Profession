# 核心经济学 Skill

## 何时触发

用户请求：
- 学习MBA经济学课程
- 理解市场机制、定价和竞争
- 宏观经济对商业的影响
- 博弈论和决策分析
- "供需怎么影响定价" / "经济周期怎么看"

## 核心框架：微观经济学 + 宏观经济学 + 博弈论

### 步骤1：微观经济学基础

**供需原理**：
- 需求定律：价格↑ → 需求量↓（其他条件不变）
- 供给定律：价格↑ → 供给量↑
- 均衡：供需交点，市场出清价格
- 弹性：
  - 需求价格弹性 = 需求量变化% / 价格变化%
  - 弹性 > 1：弹性需求（降价增加总收入）
  - 弹性 < 1：非弹性需求（涨价增加总收入）
  - 弹性 = 1：单位弹性
- 收入弹性、交叉弹性

**消费者理论**：
- 效用最大化：在预算约束下最大化满足
- 边际效用递减：额外消费带来的额外满足递减
- 无差异曲线：给消费者相同效用的商品组合
- 预算线：消费者能负担的所有组合
- 最优消费：无差异曲线与预算线切点

**生产者理论**：
- 生产函数：投入（劳动、资本）→ 产出
- 边际产量递减：增加一单位投入带来的产出增加递减
- 成本曲线：
  - 固定成本（FC）
  - 可变成本（VC）
  - 总成本（TC = FC + VC）
  - 平均成本（AC = TC / Q）
  - 边际成本（MC = ΔTC / ΔQ）
- 利润最大化：MC = MR（边际成本 = 边际收益）

**市场结构**：

| 市场结构 | 企业数量 | 产品差异 | 进入壁垒 | 定价权 | 例子 |
|---------|----------|----------|----------|--------|------|
| 完全竞争 | 很多 | 同质 | 无 | 无 | 农产品 |
| 垄断竞争 | 很多 | 差异 | 低 | 有限 | 餐饮、零售 |
| 寡头垄断 | 少数 | 同质或差异 | 高 | 有 | 汽车、航空 |
| 完全垄断 | 唯一 | 唯一 | 极高 | 完全 | 公用事业、专利 |

- **完全竞争**：P = MC，长期零经济利润
- **垄断**：MR = MC，P > MC，有经济利润（除非监管）
- **寡头**：博弈互动，相互依赖（纳什均衡）
- **垄断竞争**：短期有利润，长期零利润（自由进入）

**市场失灵**：
- **外部性**：行为对第三方的影响（正外部性：教育；负外部性：污染）
- **公共物品**：非排他、非竞争（国防、路灯）
- **信息不对称**：
  - 逆向选择：事前信息差（柠檬市场）
  - 道德风险：事后信息差（保险、代理问题）
  - 信号传递：用信号解决信息差（教育、广告、保修）
- **垄断势力**：市场权力导致效率损失
- **政府干预**：税收、补贴、管制、反垄断、提供公共物品

### 步骤2：宏观经济学基础

**GDP（国内生产总值）**：
- 定义：一国境内一定时期内生产的所有最终商品和服务的市场价值
- 支出法：GDP = C + I + G + (X - M)
  - C = 消费，I = 投资，G = 政府支出，X = 出口，M = 进口
- 收入法：GDP = 工资 + 利息 + 租金 + 利润 + 折旧 + 间接税
- 名义GDP vs 实际GDP（剔除通胀）
- GDP增长率：经济健康度

**通货膨胀**：
- CPI（消费者价格指数）：一篮子消费品价格变化
- PPI（生产者价格指数）：生产端价格变化
- 通胀原因：需求拉动、成本推动、货币超发、结构性
- 通胀影响：购买力下降、财富再分配、不确定性
- 通缩：更危险，导致延迟消费、债务负担增加

**失业**：
- 失业率 = 失业人数 / 劳动力总数
- 自然失业率：摩擦性 + 结构性失业（无法消除）
- 周期性失业：经济周期导致的失业
- 菲利普斯曲线：通胀与失业的短期替代关系

**货币政策**：
- 央行工具：利率、准备金率、公开市场操作、量化宽松（QE）
- 利率传导：政策利率 → 市场利率 → 投资/消费 → GDP
- 泰勒规则：利率 = 中性利率 + 0.5×(通胀缺口) + 0.5×(产出缺口)
- 流动性陷阱：利率接近零，货币政策失效

**财政政策**：
- 工具：政府支出、税收、转移支付
- 乘数效应：政府支出增加 → 收入增加 → 消费增加 → 收入再增加
- 预算：赤字（支出>收入）vs 盈余（收入>支出）
- 公债：债务/GDP比率，可持续性
- 自动稳定器：失业救济、累进税（经济衰退时自动增加支出/减少税收）

**国际经济学**：
- 汇率：名义汇率、实际汇率、购买力平价（PPP）
- 国际收支：经常账户 + 资本账户 + 金融账户
- 贸易顺差/逆差
- 汇率制度：固定、浮动、管理浮动
- 蒙代尔不可能三角：固定汇率、资本自由流动、独立货币政策，只能三选二

**经济周期**：
- 繁荣（Peak）→ 衰退（Recession）→ 谷底（Trough）→ 复苏（Recovery）
- 领先指标：股市、新订单、货币供应、建筑许可
- 同步指标：GDP、就业、工业产出
- 滞后指标：CPI、失业率、企业利润

### 步骤3：博弈论与商业策略

**基本博弈**：
- **囚徒困境**：个体理性 → 集体非理性（纳什均衡 ≠ 帕累托最优）
- 商业应用：价格战、广告竞争、研发竞争
- 重复博弈：合作可能在长期中出现（以牙还牙策略）
- 有限重复博弈：最后一次不合作，逆向归纳导致全程不合作

**纳什均衡**：
- 在给定其他参与者策略的情况下，每个参与者都没有动力改变策略
- 可能不唯一，可能非最优
- 求解方法：最优反应分析

**竞争博弈**：
- **古诺模型（Cournot）**：产量竞争，同时决策
- **伯特兰模型（Bertrand）**：价格竞争，同时决策
- **斯塔克尔伯格模型（Stackelberg）**：产量竞争，序贯决策（先动优势）
- **先动优势 vs 后动优势**：
  - 先动：抢占市场、建立标准、学习曲线
  - 后动：学习先动者、避免错误、技术后发优势

**合作博弈**：
- 战略联盟：互补性、信任、承诺
- 拍卖理论：英式、荷兰、第一价格密封、第二价格密封（维克里拍卖）
- 机制设计：激励相容、个体理性、效率

**行为经济学**：
- 有限理性：人不是完全理性
- 前景理论（Kahneman & Tversky）：
  - 损失厌恶：损失的痛苦 > 获得的快乐（约2倍）
  - 框架效应：同一信息不同呈现导致不同决策
  - 锚定效应：过度依赖第一个信息
  - 禀赋效应：拥有的东西估值更高
- 商业应用：定价策略、产品设计、营销沟通

## 经典案例

- **OPEC石油定价**：寡头博弈，产量配额，卡特尔稳定性
- **航空公司定价**：价格歧视，收益管理，寡头竞争
- **专利药 vs 仿制药**：垄断 → 竞争，价格战
- ** Uber vs 滴滴**：双边市场网络效应，补贴大战，囚徒困境
- **淘宝 vs eBay**：免费 vs 收费，平台竞争策略
- **iPhone定价**：价格歧视，版本控制，行为经济学应用
- **2008金融危机**：房地产泡沫、次级贷款、系统性风险、政策应对
- **中国供给侧改革**：去产能、去库存、去杠杆、降成本、补短板

## 使用示例

**用户输入**：
> "为什么航空公司在起飞前会降价卖剩余座位，但酒店不会？"

**AI执行**：
1. 解释易腐性库存：飞机座位和酒店房间都是易腐品（时间过了就没了）
2. 对比：航空公司收益管理（yield management）更精细，因为：
   - 需求波动更大（商务vs休闲）
   - 竞争更激烈（同一航线多家公司）
   - 边际成本极低（多一个乘客几乎零成本）
3. 酒店为什么不降价：
   - 品牌损害（降价影响感知价值）
   - 老客户不满（看到降价）
   - 需求预测相对容易
   - 但也在改变（OTA平台、最后一分钟预订）
4. 用经济学框架：价格歧视、库存管理、消费者行为
