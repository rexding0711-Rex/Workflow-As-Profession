## 核心框架：时间价值 + 风险回报 + 资本配置

### 步骤1：货币时间价值（TMV）

**基础公式**：
- 现值（PV）= FV / (1+r)^n
- 终值（FV）= PV × (1+r)^n
- 年金现值 = PMT × [1 - (1+r)^-n] / r

**应用**：
- 投资决策：NPV = ∑现金流/(1+r)^t - 初始投资
- NPV > 0 → 接受项目
- IRR：使NPV=0的折现率，IRR > 要求回报率 → 接受

**关键理解**：
- 今天的一块钱 > 明天的一块钱（通货膨胀+机会成本+风险）
- 复利是世界第八大奇迹（爱因斯坦可能没说过，但道理对）
- 永续年金：PV = C / r

### 步骤2：风险与回报

**单一资产**：
- 预期回报 = E(R) = Σ P_i × R_i
- 风险（标准差）= σ = √[Σ P_i × (R_i - E(R))^2]
- 风险越高，要求回报越高

**投资组合**：
- 组合回报 = 各资产回报的加权平均
- 组合风险 ≠ 各资产风险的加权平均（因为相关性）
- 分散化可以降低风险（不相关资产）
- 有效前沿：风险-回报最优组合集合

**CAPM模型**：
```
E(R_i) = R_f + β_i × (E(R_m) - R_f)
```
- R_f = 无风险利率
- β_i = 资产相对市场的系统性风险
- E(R_m) - R_f = 市场风险溢价
- β < 1：防御型（市场跌1%，它跌<1%）
- β > 1：激进型（市场涨1%，它涨>1%）

### 步骤3：资本成本（WACC）

**WACC公式**：
```
WACC = (E/V) × R_e + (D/V) × R_d × (1 - T_c)
```
- E = 权益市值，D = 债务市值，V = E + D
- R_e = 权益成本（用CAPM计算）
- R_d = 债务成本（利息率）
- T_c = 公司税率（利息抵税）

**关键理解**：
- WACC是公司的"平均融资成本"
- 项目回报 > WACC → 创造价值
- 债务便宜但增加风险，权益贵但更安全
- 最优资本结构：平衡税盾收益和财务困境成本

**权益成本计算**：
- CAPM法：R_e = R_f + β × (R_m - R_f)
- DDM法：R_e = (D_1 / P_0) + g（股利增长模型）
- 债券收益率加风险溢价：R_e = R_d + RP

### 步骤4：资本预算（投资决策）

**NPV法（首选）**：
- 计算所有未来现金流的现值（用WACC折现）
- 减去初始投资
- NPV > 0 → 接受

**IRR法**：
- 找到使NPV=0的折现率
- IRR > WACC → 接受
- 问题：非常规现金流可能有多个IRR；再投资假设不现实

**回收期法**：
- 简单回收期：几年回本
- 贴现回收期：用贴现现金流计算
- 问题：忽略回收期后的现金流

**盈利指数**：
- PI = PV(未来现金流) / 初始投资
- PI > 1 → 接受（等价于NPV > 0）

**互斥项目选择**：
- 规模差异：用NPV，不用IRR
- 时间差异：用NPV或MIRR
- 现金流模式差异：NPV最可靠

### 步骤5：估值方法

**DCF法（绝对估值）**：
- 自由现金流（FCF）预测（5-10年）
- 终值计算（Gordon增长模型或退出倍数）
- 用WACC折现到现值
- 企业价值 = PV(预测期FCF) + PV(终值)
- 股权价值 = 企业价值 - 净债务

**可比公司法（相对估值）**：
- 选择可比上市公司
- 计算倍数：P/E, EV/EBITDA, P/B, EV/Sales
- 应用到目标公司

**先例交易法**：
- 找类似并购交易
- 计算交易倍数
- 应用到目标公司

**资产基础法**：
- 清算价值（资产变现 - 负债）
- 重置成本（重建资产的成本）
- 账面价值（不太可靠）

### 步骤6：资本结构理论

**MM定理（无税）**：
- 资本结构不影响企业价值（完美市场假设）
- 现实：有税、有财务困境成本、有信息不对称

**MM定理（有税）**：
- 债务增加企业价值（利息抵税）
- 最优 = 100%债务（不现实）

**权衡理论**：
- 债务税盾收益 vs 财务困境成本
- 最优资本结构在平衡点

**优序融资理论（Pecking Order）**：
- 内部融资 > 债务 > 权益
- 原因：信息不对称（发行权益被解读为坏消息）
- 解释了为什么公司不遵循最优资本结构

**信号理论**：
- 分红增加 → 信号：公司有信心
- 股票回购 → 信号：股价被低估
- 发行新股 → 信号：管理层认为股价被高估（或需要钱）

### 步骤7：公司金融特殊主题

**并购估值**：
- 协同效应估值（收入协同/成本协同）
- 收购溢价（控制权溢价）
- 商誉和会计处理
- 支付方式选择（现金 vs 股票 vs 混合）

**杠杆收购（LBO）**：
- 用大量债务收购公司
- 现金流用于偿债
- 退出：IPO/出售/再融资
- IRR驱动因素：买入价、杠杆倍数、退出倍数、偿债速度

**实物期权**：
- 传统NPV忽略灵活性价值
- 扩张期权、放弃期权、延迟期权
- 用期权定价模型（Black-Scholes）估值

## 经典案例

- **可口可乐资本结构**：低债务，利用品牌优势
- **特斯拉融资**：多轮股权+可转债，成长型公司融资策略
- **KKR收购RJR Nabisco**：史上最大LBO，高杠杆风险
- **迪士尼收购21世纪福克斯**：协同效应估值，内容整合
- **伯克希尔资本配置**：不派息，再投资，最优资本配置
- **WeWork IPO失败**：估值泡沫，公司治理问题

## 使用示例

**用户输入**：
> "帮我用DCF方法估值一家SaaS公司，年收入5000万，增长率30%，毛利率80%"

**AI执行**：
1. 预测未来5年FCF
2. 选择合适的WACC（SaaS通常12-15%）
3. 计算终值（永续增长或退出倍数）
4. 折现到现值
5. 给出企业价值和股权价值
6. 做敏感性分析（增长率、WACC变化）
7. 讨论估值的不确定性
