# 相关度评分算法

## 评分维度

相关度评分基于以下维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 关键词匹配 | 40% | 文档关键词与用户标签的匹配度 |
| 主题匹配 | 25% | 文档主题与用户项目的相关性 |
| 行业匹配 | 20% | 文档行业与用户行业的重叠度 |
| 时效性 | 10% | 文档时效性（越新越相关） |
| 来源可靠性 | 5% | 来源可信度（高可靠来源加分） |

## 关键词匹配算法

```
段落关键词提取：
- 提取段落中的实体名词（公司、技术、产品、市场）
- 提取段落中的行业术语
- 提取段落中的数字和单位（如"50亿"、"30%"）

匹配计算：
- 精确匹配：关键词完全一致（权重1.0）
- 同义匹配：关键词同义（如"陶瓷基板"="陶瓷散热基板"）（权重0.8）
- 上下位匹配：关键词上下位关系（如"陶瓷基板"⊂"陶瓷材料"）（权重0.6）
- 关联匹配：关键词在知识图谱中有关联（权重0.4）

段落得分 = Σ(匹配关键词权重 × 匹配类型权重) / 段落长度归一化
```

## 主题匹配算法

```
文档主题识别：
- 标题主题提取
- 高频关键词聚类
- 段落主题分类（使用预定义主题库）

用户主题库：
- 从用户项目类型提取（新赛道调研、竞品分析、供应商评估）
- 从用户关注领域提取（技术、市场、资本、政策）

匹配计算：
- 主题直接匹配：权重1.0
- 主题相关：权重0.7
- 主题无关：权重0.0
```

## 行业匹配算法

```
行业分类体系：
- 一级：制造业、信息技术、金融、能源...
- 二级：半导体、新材料、软件服务、投资...
- 三级：功率半导体、陶瓷材料、SaaS、风险投资...

匹配规则：
- 同级匹配：权重1.0
- 上下游匹配：权重0.8（如"半导体"与"半导体设备"）
- 相邻行业：权重0.5（如"半导体"与"电子制造"）
- 无关行业：权重0.0
```

## 时效性评分

```
文档日期：D_doc
当前日期：D_now

时效性得分 = max(0, 1 - (D_now - D_doc) / 衰减周期)

衰减周期：
- 新闻/快讯：7天
- 行业报告：90天
- 技术论文：365天
- 政策文件：180天（但需标注政策有效期）
- 财务数据：90天（季报）/365天（年报）
```

## 来源可靠性评分

```
来源评级：
- A级（高可靠）：顶级券商、顶级期刊、政府官方、上市公司年报
- B级（可靠）：知名媒体、知名智库、行业龙头
- C级（一般）：普通媒体、自媒体、未验证来源
- D级（低可靠）：匿名来源、无出处的数据

评分：
- A级：1.0
- B级：0.8
- C级：0.5
- D级：0.2
```

## 综合评分公式

```
相关度得分 = 
  关键词匹配 × 0.40 +
  主题匹配 × 0.25 +
  行业匹配 × 0.20 +
  时效性 × 0.10 +
  来源可靠性 × 0.05

得分范围：0-100%

分级：
- 高相关（>70%）：🟢 优先阅读
- 中相关（30-70%）：🟡 快速浏览
- 低相关（<30%）：⚪ 可跳过
```

## 段落级评分示例

```
用户标签：陶瓷散热、半导体、AI算力

段落1："陶瓷基板在AI服务器散热中的应用"
- 关键词匹配：陶瓷基板(1.0) + AI(1.0) + 散热(1.0) = 高
- 主题匹配：技术应用 = 相关(0.8)
- 行业匹配：半导体/数据中心 = 相关(0.9)
- 时效性：2024年报告 = 高(0.95)
- 来源可靠性：中金 = A级(1.0)
- 得分：0.40×1.0 + 0.25×0.8 + 0.20×0.9 + 0.10×0.95 + 0.05×1.0 = 0.945 → 95% 🟢

段落2："陶瓷材料在汽车电子中的应用"
- 关键词匹配：陶瓷材料(0.6) + 汽车电子(0.0) = 中
- 主题匹配：应用场景不同 = 部分相关(0.4)
- 行业匹配：汽车行业 vs 半导体 = 低相关(0.2)
- 得分：0.40×0.3 + 0.25×0.4 + 0.20×0.2 + ... = 0.35 → 35% 🟡

段落3："房地产投资趋势分析"
- 关键词匹配：无匹配 = 0
- 主题匹配：无关 = 0
- 行业匹配：无关 = 0
- 得分：0.05 → 5% ⚪
```

## 用户反馈优化

用户可对评分进行反馈，优化算法：

```
用户反馈：
- "这段对我很有用" → 提升该段落关键词权重
- "这段不相关" → 降低该段落关键词权重
- "应该关注成本" → 新增"成本"标签
- "不需要汽车相关内容" → 降低"汽车"标签权重

算法更新：
- 每次反馈后，调整关键词权重
- 每月回顾，优化评分参数
- 用户标签库持续更新
```
