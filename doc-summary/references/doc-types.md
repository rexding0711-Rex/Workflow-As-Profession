# 支持的文档类型

## 券商/投研报告

**特点：**
- 结构固定：摘要→行业概览→公司分析→财务预测→投资建议→风险提示
- 数据密集：大量表格、图表、财务数据
- 观点明确：有明确的投资评级和目标价
- 时效性强：通常季度更新

**摘要策略：**
- 提取投资观点（买入/持有/卖出，目标价）
- 提取核心假设（关键变量）
- 提取财务预测（收入/利润/现金流）
- 提取风险提示（3-5个主要风险）
- 标记数据时效性

**常见来源：**
- 国内：中金、中信、华泰、招商、国君
- 国际：摩根士丹利、高盛、JP摩根、瑞银

## 技术论文/专利

**特点：**
- 结构固定：摘要→引言→方法→实验→结果→讨论→结论→参考文献
- 创新点突出：明确声明"本文贡献"
- 数据精确：实验参数、样本量、统计显著性
- 局限明确：通常有Limitations章节

**摘要策略：**
- 提取创新点（1-3个核心贡献）
- 提取方法（技术路线、实验设计）
- 提取关键结果（定量数据）
- 提取局限性（适用范围、未验证假设）
- 评估技术成熟度（TRL等级）

**常见来源：**
- 学术：Nature、Science、IEEE、arXiv、CNKI
- 专利：Google Patents、WIPO、中国专利局

## 政策/法规文件

**特点：**
- 条款精确：逐条列出，不容歧义
- 影响明确：通常会说明适用范围和执行时间
- 配套措施：常与补贴、标准、审批流程配套
- 时效性强：有明确的有效期和修订计划

**摘要策略：**
- 提取核心条款（5-10条）
- 提取适用范围（谁受影响）
- 提取时间节点（何时生效、何时截止）
- 提取配套措施（补贴、标准、审批）
- 提取合规要求（必须做什么、禁止做什么）
- 评估影响程度（高/中/低）

**常见来源：**
- 国务院、各部委、行业协会、地方政府

## 新闻/快讯

**特点：**
- 时效性极强：通常当天或隔天
- 信息密度低：单篇信息量有限
- 观点混杂：事实+解读+预测
- 来源多样：专业媒体、自媒体、社交媒体

**摘要策略：**
- 提取核心事件（谁、做了什么、何时）
- 提取影响评估（对谁、多大影响）
- 提取关联信号（后续可能引发什么）
- 标记来源可信度
- 标记信息完整性（是否已确认）

**常见来源：**
- 专业媒体：财联社、36氪、彭博、路透
- 行业媒体：各垂直领域媒体
- 社交媒体：Twitter/X、LinkedIn、知乎

## 会议纪要/演讲稿

**特点：**
- 口语化：非正式，可能有重复、口头禅
- 观点发散：可能涉及多个话题
- 决策导向：通常包含决策、行动项、责任人
- 时间敏感：会议内容有时效性

**摘要策略：**
- 提取决策（达成了什么共识）
- 提取行动项（谁、做什么、何时）
- 提取争议点（未达成一致的问题）
- 提取关键观点（有洞察的发言）
- 提取待跟进事项

## 合同/协议

**特点：**
- 条款精确：法律用语，不容歧义
- 权利义务明确：谁做什么、谁得什么
- 风险点隐蔽：可能在附件或条款细节中
- 结构固定：定义→权利义务→违约责任→争议解决

**摘要策略：**
- 提取核心条款（定义、权利、义务）
- 提取关键数字（金额、比例、时间）
- 提取风险条款（违约责任、不可抗力、终止条件）
- 提取争议解决机制（仲裁/诉讼、管辖地）
- 标记需要法务重点关注的条款

**特殊处理：**
- 条款对比：新旧版本对比，变更标记
- 模板匹配：与标准条款对比，标记异常

## 技术文档/API文档

**特点：**
- 结构清晰：功能列表、接口定义、参数说明
- 版本敏感：不同版本可能有差异
- 限制明确：性能限制、使用限制、兼容性
- 示例丰富：通常包含代码示例

**摘要策略：**
- 提取核心功能（3-5个主要功能）
- 提取接口概览（关键API）
- 提取限制条件（性能、并发、兼容性）
- 提取版本信息（当前版本、变更历史）
- 提取快速开始指南

## 财务报表

**特点：**
- 数据精确：审计过的数字
- 结构固定：利润表、资产负债表、现金流量表
- 趋势重要：同比、环比变化
- 附注关键：附注中可能包含重要信息

**摘要策略：**
- 提取关键指标（收入、利润、现金流、毛利率）
- 提取趋势变化（同比、环比）
- 提取异常项（与预期差异大的项目）
- 提取风险提示（应收账款、存货、债务）
- 提取管理层讨论（对未来展望）

## 特殊文档处理

### 扫描件/PDF图片
- 先OCR识别文字
- 再按对应类型处理

### 多语言文档
- 识别语言
- 优先翻译关键章节
- 保留原文引用

### 加密/受限文档
- 标记受限状态
- 提取可访问部分
- 建议获取完整权限
