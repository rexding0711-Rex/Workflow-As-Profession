# 核心分析框架库

根据问题类型，从以下框架库中选择最适用的 **2-4 个框架**进行深度分析。

---

## 战略分析

| 框架 | 核心要点 |
|------|----------|
| **五力模型** | 现有竞争、潜在进入、替代品、供应商、买方 |
| **价值链** | 主要活动（进料→生产→出货→营销→服务）+ 支持活动（采购/技术/HR/基础设施） |
| **VRIO** | 资源是否有价值、稀缺、不可复制、组织化利用 |
| **SWOT** | 优势、劣势、机会、威胁（必须匹配：SO/WO/ST/WT 策略） |
| **战略时钟** | 价格-价值定位，8条路径，避开中间路线 |
| **蓝海战略** | 四步动作框架（剔除/减少/增加/创造） |
| **商业模式画布** | 9要素（客户细分、价值主张、渠道、客户关系、收入、核心资源、关键活动、合作、成本） |

---

## 财务分析

| 框架 | 核心要点 |
|------|----------|
| **三表勾稽** | 利润→留存收益，折旧→现金流加回，期末现金→资产负债表 |
| **利润质量** | 经营现金流/净利润、非经常性损益、会计政策变更 |
| **杜邦分析** | ROE = 净利率 × 总资产周转率 × 权益乘数 |
| **现金流模式** | 奶牛型(+/-/+)、扩张型(+/-/+/-)、输血型(-/-/+)、衰退型(-/+/-) |
| **财务造假信号** | 毛利率异常、应收暴增、现金流利润背离、存货积压、在建工程挂账、频繁换审计师 |
| **估值方法** | DCF（现金流折现）、可比公司法、先例交易法、LBO模型 |

---

## 营销分析

| 框架 | 核心要点 |
|------|----------|
| **STP** | 细分(Segmentation)→目标(Targeting)→定位(Positioning) |
| **4P/4C** | 产品→客户方案、价格→客户成本、渠道→便利性、推广→沟通 |
| **定位三角** | 目标用户、差异化、信任状 → 品牌主张 |
| **AARRR漏斗** | 获客→激活→留存→收入→推荐 |
| **增长引擎** | 病毒式(K因子)、付费(LTV/CAC)、粘性(留存率/NPS) |
| **品牌资产** | 知名度、联想、质量、忠诚度 |

---

## 运营分析

| 框架 | 核心要点 |
|------|----------|
| **精益七浪费** | 过量生产、等待、运输、过度加工、库存、动作、缺陷 |
| **TOC五步法** | 识别瓶颈→挖尽瓶颈→迁就瓶颈→提升瓶颈→回到步骤1 |
| **OEE** | 可用率 × 性能率 × 良品率，世界级标准 > 85% |
| **EOQ** | 经济订货量 = √(2DS/H) |
| **牛鞭效应** | 需求端小波动向上游放大，对策：信息共享、缩短提前期 |
| **六西格玛** | DMAIC（定义→测量→分析→改进→控制） |
| **项目管理三角** | 范围 ↔ 时间 ↔ 成本（质量在中央） |

---

## 组织分析

| 框架 | 核心要点 |
|------|----------|
| **六盒子** | 目的→结构→关系→激励→支持→领导，评估每个盒子健康度 |
| **组织结构** | 职能型→事业部型→矩阵型→平台型→网络型→生态型 |
| **激励理论** | 马斯洛需求、赫茨伯格双因素、期望理论、公平理论、强化理论 |
| **领导力五级** | 高能力个人→贡献团队成员→胜任管理者→有效领导者→卓越领导者 |
| **变革管理** | 勒温三阶段（解冻→改变→再冻结）、科特八步 |
| **团队发展** | 形成→震荡→规范→执行→解散 |
| **冲突管理** | 任务冲突（有益）vs 关系冲突（有害），协作解决 |

---

## 数据分析

| 框架 | 核心要点 |
|------|----------|
| **DIKW金字塔** | 数据→信息→知识→智慧 |
| **OSM模型** | 目标(Objective)→策略(Strategy)→度量(Measurement) |
| **北极星指标** | 1个核心指标，反映产品核心价值 |
| **A/B测试** | 假设→随机分组→样本量计算→运行→统计检验→决策 |
| **因果推断** | 随机实验(金标准)、自然实验、DID、工具变量、断点回归 |
| **假设检验** | H₀/H₁ → 显著性水平α → 检验统计量 → p值 → 决策 |
| **p值理解** | 不是H₀为真概率，而是观察到当前或更极端结果的概率 |

---

## 经济学分析

| 框架 | 核心要点 |
|------|----------|
| **供需原理** | 价格由供需决定，弹性决定总收入变化方向 |
| **市场结构** | 完全竞争→垄断竞争→寡头→垄断，定价权递增 |
| **市场失灵** | 外部性、公共物品、信息不对称（逆向选择/道德风险） |
| **博弈论** | 囚徒困境、纳什均衡、古诺/伯特兰/斯塔克尔伯格模型 |
| **行为经济学** | 损失厌恶(2倍)、锚定效应、框架效应、禀赋效应 |
| **宏观政策** | 货币政策(利率/流动性)、财政政策(支出/税收)、监管政策 |
| **经济周期** | 复苏→繁荣→衰退→萧条，领先/同步/滞后指标 |
| **情景规划** | 基准50% + 乐观20% + 悲观20% + 极端10% |

---

## 创业/融资分析

| 框架 | 核心要点 |
|------|----------|
| **精益创业** | 假设→MVP→验证→转型/坚持，创新核算 |
| **PMF验证** | 40%用户说"不能用了会失望"、留存曲线平稳、自然增长 |
| **增长引擎** | 病毒式(K因子>1)、付费(LTV/CAC>3)、粘性(月流失<5%) |
| **融资阶段** | 种子→天使→Pre-A→A→B→Pre-IPO，匹配不同阶段 |
| **估值方法** | 早期(可比/博克斯) → 成长期(PS/DCF) → 后期(倍数/DCF) |
| **Term Sheet红线** | >1x participating LP、full ratchet、创始人vesting<4年、个人连带责任 |
| **现金流模式** | 经营+投资+筹资，三种流的组合揭示企业类型 |

---

## 决策/伦理分析

| 框架 | 核心要点 |
|------|----------|
| **决策质量四维度** | 信息质量、分析质量、偏见控制、执行可行性 |
| **对抗偏见** | 红队机制、事前验尸、反事实思考、基础概率 |
| **决策类型** | 战略(不可逆/慢决策)、战术(可逆/快迭代)、操作(标准化/授权) |
| **伦理四象限** | 合法+道德(理想)、合法+不道德(灰色)、不合法+道德(特殊情况)、不合法+不道德(红线) |
| **伦理检验** | 公开测试、普遍性测试、换位思考、遗产测试 |
| **利益相关者** | 股东、员工、客户、供应商、社区、环境、未来世代 |
