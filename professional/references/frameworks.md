# 20个职业模块核心框架速查

## 商业战略
- **五力模型**：现有竞争者/潜在进入者/替代品/供应商/买方 → 行业吸引力
- **价值链**：进料→生产→出货→营销→服务，找利润池和卡脖子点
- **战略时钟**：8条路径，避开中间路线，选择成本领先或差异化
- **蓝海战略**：四步动作（剔除/减少/增加/创造），价值创新
- **商业模式画布**：9要素（客户/价值/渠道/关系/收入/资源/活动/合作/成本）
- **SWOT**：不是列表，是匹配（SO/WO/ST/WT策略）
- **VRIO**：资源有价值/稀缺/不可复制/组织化利用 → 竞争优势

## 金融分析
- **三表勾稽**：利润→留存收益，折旧→现金流加回，现金→资产负债表
- **利润质量**：经营现金流/净利润、非经常性损益、会计政策变更
- **杜邦分析**：ROE = 净利率 × 总资产周转率 × 权益乘数 → 驱动因素识别
- **现金流模式**：奶牛型(+/-/+) / 扩张型(+/-/+/-) / 输血型(-/-/+) / 衰退型(-/+/-)
- **财务造假信号**：应收暴增、现金流利润背离、存货积压、在建工程挂账、频繁换审计师
- **估值方法**：DCF（现金流折现）/ 可比公司法（P/E、EV/EBITDA）/ 先例交易法 / LBO模型
- **WACC**：权益成本(CAPM) + 债务成本 × (1-税率) → 项目回报 > WACC 才创造价值

## 咨询方法
- **Issue Tree**：大问题拆成MECE子问题，拆到可验证为止
- **MECE**：Mutually Exclusive（不重叠）+ Collectively Exhaustive（不遗漏）
- **假设驱动**：先猜答案（具体/可证伪/有影响力）→ 收集验证数据（80/20）→ 证实/证伪/调整
- **金字塔原理**：结论先行 + 以上统下 + 归类分组 + 逻辑递进
- **SCQA故事线**：Situation（背景）→ Complication（冲突）→ Question（问题）→ Answer（答案）
- **高管沟通**：30秒Hook → 2分钟Core → 10分钟完整（Support→Implication→Ask）
- **数据收集80/20**：Must-have（20%数据）→ Nice-to-have → Context，从Insight倒推数据需求
- **行业速写**：市场规模(TAM/SAM/SOM) → 竞争结构(CR3/壁垒) → 价值链(利润池) → 趋势(3年后)
- **标杆分析**：选标杆 → 定维度(5-7个) → 量化差距 → 追赶路径(复制/超越/差异化)
- **变革管理**：科特八步（紧迫感→联盟→愿景→沟通→授权→短期胜利→巩固→文化）+ 7S诊断

## 产品管理
- **用户洞察**：用户任务(Jobs) → 痛点(Pains) → 收益(Gains)
- **价值主张**：解决什么痛点、创造什么收益、凭什么相信
- **MVP**：最小可行产品，验证核心假设，不是半成品
- **产品生命周期**：引入→成长→成熟→衰退，不同阶段不同策略
- **Roadmap**：现在(维护) → 近期(优化) → 中期(扩展) → 远期(创新)
- **PRD结构**：背景/目标/用户故事/功能描述/验收标准/非功能性需求
- **需求优先级**：MoSCoW（Must/Should/Could/Won't）或 RICE（Reach/Impact/Confidence/Effort）
- **迭代循环**：构建→测量→学习（Build-Measure-Learn），数据驱动决策

## 数据科学
- **假设检验**：H₀/H₁ → 显著性α → 检验统计量 → p值 → 决策（p<0.05拒绝H₀）
- **回归分析**：Y = β₀ + β₁X + ε，解释系数、R²、显著性、残差诊断
- **分类模型**：逻辑回归、决策树、随机森林，看准确率/召回率/F1/AUC
- **聚类分析**：K-means、层次聚类，看轮廓系数、业务可解释性
- **可视化原则**：目的明确、选对图表、减少认知负担、突出异常
- **因果推断**：随机实验(金标准)、DID、工具变量、断点回归，相关≠因果
- **A/B测试**：假设→随机分组→样本量计算→运行→统计检验→决策，不要偷看数据
- **特征工程**：缺失值处理、异常值、标准化、编码、降维，Garbage In Garbage Out
- **过拟合 vs 欠拟合**：训练集好测试集差=过拟合，训练集差=欠拟合，交叉验证调参

## 增长营销
- **STP**：细分(Segmentation) → 目标(Targeting) → 定位(Positioning)
- **4P/4C**：产品→客户方案、价格→客户成本、渠道→便利性、推广→沟通
- **定位三角**：目标用户、差异化、信任状 → 品牌主张
- **AARRR漏斗**：获客→激活→留存→收入→推荐，找瓶颈环节
- **增长引擎**：病毒式(K因子>1) / 付费(LTV/CAC>3) / 粘性(月流失<5%, NPS>50)
- **品牌资产**：知名度、品牌联想、感知质量、品牌忠诚度、其他资产
- **内容营销**：Hook(3秒) → 冲突 → 反转 → CTA，3秒法则
- **CRM**：获客成本(CAC) → 客户生命周期价值(LTV) → LTV/CAC > 3 可持续

## 精益运营
- **精益七浪费**：过量生产、等待、运输、过度加工、库存、动作、缺陷
- **TOC五步法**：识别瓶颈 → 挖尽瓶颈 → 迁就瓶颈 → 提升瓶颈 → 回到步骤1
- **OEE**：可用率 × 性能率 × 良品率，世界级>85%
- **EOQ**：经济订货量 = √(2DS/H)，平衡持有成本和订货成本
- **六西格玛**：DMAIC（定义→测量→分析→改进→控制），3.4ppm缺陷率
- **项目管理三角**：范围 ↔ 时间 ↔ 成本（质量在中央），一个变至少另一个变
- **关键路径(CPM)**：最长路径决定项目工期，关键路径延迟=项目延迟
- **敏捷/Scrum**：Sprint(2-4周) → 每日站会 → 评审 → 回顾，用户故事+燃尽图
- **牛鞭效应**：需求端小波动向上游放大，对策：信息共享、缩短提前期、减少批量

## 组织设计
- **六盒子**：目的 → 结构 → 关系 → 激励 → 支持 → 领导，评估每个盒子健康度
- **激励理论**：马斯洛需求层次 → 赫茨伯格双因素 → 期望理论 → 公平理论 → 强化理论
- **领导力五级**：高能力个人 → 贡献团队成员 → 胜任管理者 → 有效领导者 → 卓越领导者
- **变革管理**：勒温三阶段（解冻→改变→再冻结）/ 科特八步
- **团队发展**：形成 → 震荡 → 规范 → 执行 → 解散
- **冲突管理**：任务冲突（有益）vs 关系冲突（有害），协作解决
- **组织文化**：表层(行为/仪式) → 中层(价值观) → 深层(基本假设)，文化洋葱
- **邓巴数**：人类稳定社交关系上限~150人，超过需要制度替代人情

## 技术管理
- **技术生命周期**：萌芽→成长→成熟→衰退，S曲线，增长期早期进入最优
- **创新类型**：渐进式(小改进) / 突破式(全新) / 颠覆式(从低端侵蚀)
- **创新流程**：创意生成 → 筛选 → 概念开发 → 产品开发 → 商业化，漏斗模型
- **开放式创新**：利用外部创意和路径，API/众包/黑客松/加速器
- **数字化成熟度**：启蒙→采纳→适应→转型→原生，5阶段
- **技术治理**：IT架构(单体vs微服务/云vs本地) + 数据治理(质量/安全/隐私) + AI治理(偏见/解释性)
- **DevOps/SRE**：开发运维一体化，CI/CD，监控，可靠性工程
- **知识产权**：专利(防御/进攻) + 商业秘密 + 商标 + 著作权 + 开源策略

## 谈判沟通
- **BATNA**：Best Alternative to a Negotiated Agreement，谈判破裂的最佳替代方案
- **原则式谈判**：对人温和，对事强硬；关注利益，不是立场；创造双赢；用客观标准
- **谈判准备**：了解对方文化/利益/BATNA → 设定reservation price和target price
- **锚定效应**：先出价的一方设定锚点，高开/低开策略
- **非暴力沟通**：观察 → 感受 → 需要 → 请求，不带评判地表达
- **金字塔汇报**：结论先行 + 以上统下 + 归类分组 + 逻辑递进
- **高管沟通**：30秒Hook → 2分钟Core → 10分钟完整（Support→Implication→Ask）
- **电梯演讲**：30秒，"我们发现...因为...导致...建议...预期...需要..."
- **会议纪要**：决策 + 行动项(任务/负责人/截止) + 风险 + 下次会议

## 创业融资
- **精益创业**：假设 → MVP → 验证 → 转型/坚持，创新核算
- **PMF验证**：40%用户说"不能用了会失望"、留存曲线平稳、自然增长
- **商业模式画布**：9要素，客户细分/价值主张/渠道/关系/收入/资源/活动/合作/成本
- **融资阶段**：种子(10-300万) → 天使(300-1000万) → Pre-A → A → B → Pre-IPO
- **估值方法**：早期(可比/博克斯) → 成长期(PS/DCF) → 后期(倍数/DCF)
- **Term Sheet红线**：>1x participating LP、full ratchet、创始人vesting<4年、个人连带责任
- **股权分配**：按贡献+未来价值+承诺度，动态股权(4年vesting，1年cliff)
- **现金流模式**：经营+投资+筹资，三种流的组合揭示企业类型

## 投资并购
- **LBO模型**：假设(收购价/债务比例/利率) → 预测(EBITDA增长) → 计算IRR，目标>20-25%
- **尽职调查**：财务(历史/预测/税务) + 法律(合同/诉讼/IP) + 商业(市场/竞争/客户) + 技术(架构/代码/团队) + 人力(组织/文化/薪酬)
- **交易结构**：资产收购vs股权收购、现金vs股票、earn-out、对赌、管理层保留
- **协同效应**：收入协同(交叉销售/市场扩展) + 成本协同(规模/效率)，通常高估
- **投后管理**：董事会参与、运营支持、财务优化、人才支持、战略增长、治理改善
- **退出策略**：IPO(最高估值但复杂) / 战略出售(协同溢价) / 二次出售(给新PE) / 回购

## 全球化
- **CAGE框架**：文化距离(Cultural) + 行政距离(Administrative) + 地理距离(Geographic) + 经济距离(Economic)
- **进入模式**：出口(低控制) → 许可/特许经营 → 合资 → 全资/并购(高控制)
- **全球战略**：国际(国内创新海外销售) → 多国(高度本地化) → 全球(标准化) → 跨国(兼顾)
- **全球价值链**：设计(高附加值) → 核心部件 → 组装(低附加值) → 品牌/服务(高附加值)，微笑曲线
- **跨文化管理**：Hofstede六维度(权力距离/个人主义/男性化/不确定性规避/长期导向/放纵)
- **汇率管理**：交易风险(应收应付) + 折算风险(报表) + 经济风险(长期竞争力)，对冲工具

## 品牌策略
- **定位三角**：目标用户 + 差异化 + 信任状 → 品牌主张
- **品牌资产**：知名度(Awareness) + 品牌联想(Associations) + 感知质量(Quality) + 忠诚度(Loyalty)
- **品牌价值链**：营销投入 → 顾客心智 → 市场绩效 → 股东价值
- **品牌延伸**：利用品牌进入新品类，风险是品牌稀释
- **品牌架构**：单一品牌(华为) vs 多品牌(宝洁) vs 背书品牌(万豪/喜来登) vs 独立品牌
- **品牌承诺**：功能性(产品做什么) + 情感性(让我感觉如何) + 社会性(代表我是什么人)

## 设计思维
- **同理心(Empathize)**：观察、访谈、沉浸，理解用户真实需求
- **定义(Define)**：POV(Point of View) = 用户 + 需求 + 洞察，把问题聚焦
- **构思(Ideate)**：头脑风暴， quantity over quality，先发散再收敛
- **原型(Prototype)**：快速、低成本、可触摸，失败要早、失败要便宜
- **测试(Test)**：真实用户测试，观察行为而非听意见，迭代改进
- **用户旅程**：认知 → 考虑 → 购买 → 使用 → 推荐，找痛点和机会点
- **服务蓝图**：前台(用户可见) + 后台(内部流程) + 支持(系统/资源)

## 项目管理
- **WBS**：工作分解结构，把大项目拆成小任务，拆到可交付、可分配、可估算
- **甘特图**：时间轴上的任务条，可视化进度、依赖、里程碑
- **关键路径(CPM)**：最长路径决定工期，关键路径浮动时间为0
- **风险矩阵**：概率(高/中/低) × 影响(高/中/低) → 优先级，制定应对策略
- **敏捷/Scrum**：Sprint(2-4周) → 产品待办列表 → Sprint待办 → 每日站会 → 评审 → 回顾
- **PDCA**：Plan-Do-Check-Act，持续改进循环
- **干系人管理**：识别干系人 → 分析影响/利益 → 制定沟通策略 → 管理期望
- **范围蔓延**：防止无控制的需求增加，变更必须走流程

## 决策科学
- **决策质量四维度**：信息质量、分析质量、偏见控制、执行可行性
- **认知偏见清单**：确认偏见、锚定效应、沉没成本、幸存者偏差、过度自信、群体思维
- **对抗偏见工具**：红队机制(专人挑毛病)、事前验尸("假设失败，为什么")、反事实思考、基础概率
- **决策类型**：战略(不可逆/慢决策) / 战术(可逆/快迭代) / 操作(标准化/授权)
- **情景规划**：基准50% + 乐观20% + 悲观20% + 极端10%，每个情景有触发条件和行动计划
- **决策矩阵**：标准 × 权重 × 评分，敏感性分析，给出明确推荐
- **博弈思维**：纳什均衡、囚徒困境、重复博弈、先动/后动优势

## 商业伦理
- **伦理四象限**：合法+道德(理想) / 合法+不道德(灰色) / 不合法+道德(特殊情况) / 不合法+不道德(红线)
- **利益相关者**：股东、员工、客户、供应商、社区、环境、未来世代
- **伦理检验**：公开测试(公开后是否骄傲)、普遍性测试(所有人都做会怎样)、换位思考、遗产测试
- **底线原则**：不帮客户撒谎、不操纵数据、不利用信息优势损害弱势方、不接受超出合理范围的利益
- **吹哨人伦理**：发现严重违规/安全/环境问题时，内部报告优先，无效时考虑外部披露

## 宏观经济
- **经济周期**：复苏(PMI>50,利率低) → 繁荣(GDP高,通胀升) → 衰退(GDP降,失业升) → 萧条(低增长,通缩)
- **领先指标**：PMI、新订单、股票指数、货币供应；**同步指标**：GDP、就业、工业产出；**滞后指标**：CPI、企业利润
- **货币政策**：加息(抑制通胀,美元强) → 降息(刺激增长,美元弱) → 降准(释放流动性)
- **财政政策**：扩张(基建/减税/补贴) → 收缩(减支/增税)，乘数效应
- **行业映射**：利率敏感型(房地产/金融) vs 出口导向型(汇率影响) vs 消费驱动型(收入/信心) vs 周期型(大宗/工业) vs 防御型(医药/食品)
- **情景规划**：基准50% + 乐观20% + 悲观20% + 极端10%

## 职业决策
- **MBA ROI**：总成本(学费+生活费+机会成本=$200k-500k) vs 收益(薪资增幅+网络+信号+转型)
- **个人匹配度**：知识收益(80%可自学) + 网络收益(质量×广度×深度×主动投入) + 信号收益(行业差异大) + 转型收益(切换成功率)
- **替代方案**：自学+在线课程($0-2k) / 在职MBA/EMBA($50k-100k) / Executive Education($5k-20k) / 本包+实践($0)
- **学校选择**：职业目标匹配(Consulting→Kellogg/Booth, Finance→Wharton, Tech→Stanford, Entrepreneurship→Stanford/MIT) + 地理位置 + 文化适配 + 成本与奖学金
- **校友网络**：质量×广度×深度×频率，真诚给予>索取，跨界>同类，长期>短期
