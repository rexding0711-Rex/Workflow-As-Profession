# Search Writing Guide

> For sub-agents: When you are permitted to use search tools to expand content, you need to search based on the topic directions in the outline and write presentation page content.

## Workflow

### 1. Understand Page Direction
- Read the `outline.md` produced by the main agent to understand the title, content direction, and page type for each page you are responsible for
- Clarify what information needs to be searched for each page

### 2. Search Strategy
- Conduct targeted searches based on the topic directions in the outline
- Read `guideline/search/text_search.md` for text search strategies
- Prioritize data and viewpoints from authoritative sources
- Focus on the latest industry trends and data
- Collect specific cases as supporting evidence

### 3. Content Writing Principles
1. **Information reliability**: Ensure that cited data and viewpoints come from reliable sources, and annotate data sources
2. **Per-page focus**: Each page should focus on one clear information point, supported by collected data and cases
3. **Substantive content**: Collect sufficient information to give each page substantive depth, rather than staying at a superficial level
4. **Refine after searching**: After collection, refine and organize the information rather than simply piling up search results
