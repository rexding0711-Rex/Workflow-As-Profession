# Technical Spec Selling — 技术规格驱动销售

工业品销售最大的错误：对着采购经理念数据手册。

采购经理不关心你的材料密度是 1.14 还是 1.18。
他关心的是："用你的材料替换现在的，每件成本能降多少？质量会不会出问题？"

技术规格是你的武器，但不是你的子弹。**翻译**才是。

## 何时触发

用户提及以下内容时触发：
- "怎么让客户的工程师认可我们的材料？"
- "我们的技术参数比竞品好，但客户说'差不多就行'"
- "帮我做一个材料替换的成本分析"
- "客户说另一家便宜 20%，但我们性能更好，怎么讲？"
- "How to explain technical advantages to non-technical buyers?"
- "Technical selling for engineering materials"

**排除：** 此技能 NOT for：
- 投标文件写作 → 用 `rfq-response`
- 销售流程管理 → 用 `industrial-sales-cycle`

## 技术参数翻译公式

### 黄金公式

```
技术参数 → 工程影响 → 商业价值 → 用客户的语言说
```

**❌ 技术语言：**
> "我们的尼龙材料热变形温度 245°C。"

**✅ 商业语言：**
> "因为热变形温度 245°C，你们的零件能在发动机舱里稳定工作，
> 不用担心夏天堵车时过热变形。这意味着更少的售后索赔和更高的客户满意度。"

### 常见技术参数翻译表

| 技术参数 | 工程影响 | 商业价值 | 客户语言 |
|---------|---------|---------|---------|
| 拉伸强度高 10% | 零件壁厚可减 5-10% | 每件材料成本降低 X 元 | "帮你们省材料钱" |
| 熔融指数更高 | 注塑周期缩短 15% | 每台机器每天多产 20% 的零件 | "你们的产能提升 20%" |
| 密度更低 | 同样重量做更多零件 | 每公斤材料多产 X 个零件 | "同样的钱买到更多产能" |
| 收缩率更稳定 | 模具调试更快，废品少 | 新品导入周期缩短 2 周 | "比对手快两周上市" |
| 耐老化更好 | 使用寿命延长 3 年 | 售后索赔减少 X% | "用户投诉变少" |

## 材料替换的 ROI 计算

当你想让客户替换现有材料时，不要只说"我们的材料更好"。
算一笔 Total Cost 的账：

### Total Cost 模型

```
每件 Total Cost = 材料成本 + 加工成本 + 废品成本 + 售后成本
```

| 成本项 | 现有方案 | 你的方案 | 差异 |
|--------|---------|---------|------|
| 材料单价（¥/kg） | ¥X | ¥Y | +¥Z（你的更贵） |
| 零件重量（g） | A | B（你的更轻） | -C g |
| 每件材料成本 | ¥X×A/1000 | ¥Y×B/1000 | **¥D（更便宜）** |
| 加工成本（¥/件） | ¥X | ¥Y（周期更短） | -¥Z |
| 废品率 | X% | Y%（更稳定） | -¥Z |
| 每件 Total Cost | ¥X | ¥Y | **-¥Z** |

> 即使你的材料单价贵 15%，如果零件更轻、加工更快、废品更少，
> 每件 Total Cost 可能反而低 8%。这才是你要讲的故事。

### VAVE 降本路径

帮客户做价值工程降本（Value Analysis / Value Engineering）：

| 方向 | 方法 | 典型降本幅度 |
|------|------|-----------|
| 壁厚减薄 | 材料强度提升 → 结构可更薄 | 5-15% |
| 周期缩短 | 流动性更好 → 注塑更快 | 5-10% |
| 废品降低 | 批次一致性更好 → 良率提升 | 2-5% |
| 免喷涂 | 材料自带颜色/纹理 → 省去喷涂工序 | 10-20% |
| 多部件一体化 | 材料性能允许复杂结构 → 多个零件合并 | 15-30% |

## 与不同类型客户的技术沟通

### 对工程师
- **讲什么：** 技术参数、测试数据、CAE 模拟结果、加工窗口
- **怎么讲：** 给他们数据，让他们自己验证。"这是我们的拉伸测试报告，按 ASTM D638 标准做的，你们可以参考"
- **不要：** 夸大性能、回避缺陷

### 对采购
- **讲什么：** Total Cost、供应安全、付款灵活性
- **怎么讲：** "单价贵 10%，但废品率低 3%，加上 VAVE 降本空间，Total Cost 低 8%"
- **不要：** 念技术参数

### 对高管
- **讲什么：** 供应风险、战略关系、创新合作
- **怎么讲：** "如果你们只用一家供应商，一旦断供 = 停线。我们作为 second source，给你们议价权和安全感"
- **不要：** 讲细节、吐槽客户工程师

```markdown
**目标客户：** [公司名]
**应用场景：** [具体零件/工况]
**要替代的方案：** [现有材料/供应商]

## 1. 关键参数对比

| 参数 | 客户要求 | 现有方案 | 我方方案 | 优势 |
|------|---------|---------|---------|------|
| [参数] | [值] | [值] | [值] | [说明] |

## 2. Total Cost 对比

| 成本项 | 现有方案 | 我方方案 | 差异 |
|--------|---------|---------|------|
| 材料单价 | ¥X/kg | ¥Y/kg | +¥Z |
| 每件材料成本 | ¥A | ¥B | -¥C |
| 每件加工成本 | ¥D | ¥E | -¥F |
| **每件 Total Cost** | **¥G** | **¥H** | **-¥I** |

**年化节省：** 年用量 [N] 件 × 每件节省 ¥I = **¥[总额]**

## 3. 非量化价值
- [更低的供应风险]
- [更快的技术支持响应]
- [免费的 CAE 模拟支持]

## 4. 验证计划
| 阶段 | 内容 | 时间 |
|------|------|------|
| 小样 | 实验室性能验证 | 2 周 |
| 中样 | 产线试模 | 4 周 |
| 大样 | 小批量验证 | 6 周 |
```

## 质量原则

1. **每个技术参数都要有对应的"so what"** — 拉伸强度 80 MPa → 壁厚可减 10% → 每件省 2.3 元
2. **不夸大，但也不谦虚** — 你材料的一个性能优势可能帮客户解决他们一直头疼的问题
3. **算账用客户的数据** — 问客户要他们的产量、废品率、节拍时间，用他们的数据算 ROI
4. **技术资料给工程师，商业故事给采购** — 同一批数据，不同讲法
5. **如果性能确实不如别人，不要硬比** — 换一个维度（服务、交期、灵活度）

## 使用示例

**用户输入：**
> 我们有一种新的 PPS 材料，热变形温度比竞品高 20°C，但贵了 15%。怎么说服客户值这个溢价？

**AI 执行过程：**
1. 用黄金公式翻译：热变形温度 260 → 可用于更靠近发动机的零件 → 替代金属件的可能性 → 减重 40%
2. 计算 Total Cost：材料贵 15% 但减重 40%，加上替代金属免去机加工成本，总成本低 30%
3. 对比金属件 vs PPS 件的全部成本项
4. 给工程师讲耐热数据和长期老化性能，给采购讲 Total Cost，给高管讲减重对新能源车续航的价值
