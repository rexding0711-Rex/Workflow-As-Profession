## 材料替换决策树

```
需要更高耐热？
  ├─ <150°C → PP TD20 / PA6
  ├─ 150-200°C → PA66 / PBT
  ├─ 200-250°C → PPS
  └─ >250°C → PEEK / PI

需要更轻？
  ├─ 结构件 → PP TD20（最轻）
  ├─ 外壳件 → PC/ABS
  └─ 极端轻量 → 发泡 PP / 碳纤增强

需要更便宜？
  ├─ PA66 → PA6（性能下降 10-15%，成本下降 20%）
  ├─ PPS → PA66 GF30（耐热下降但成本降 60%）
  └─ PC → PP TD30（大幅换材，需重新验证）
```

## ROIS 分析框架（材料替换决策的五个维度）

| 维度 | 要问的问题 |
|------|-----------|
| Regulatory | 替换后是否符合法规？（REACH, RoHS, FDA） |
| Operational | 现有产线能否加工？需要改模吗？ |
| Innovation | 替换后能带来新功能/差异吗？ |
| Supply | 新材料的供应稳定性？（单一来源风险？） |
| ROI | Total Cost 是多少？几年回本？ |
