# QBR 模板集

## 30 分钟版（月度 Check-in）

```
1. 执行摘要 (3 min) — 1 页、3 个数字
2. 价值亮点 (8 min) — 2-3 个使用案例 + ROI
3. 改进机会 (8 min) — 1-2 个建议
4. 下一步 (3 min) — 双方行动
5. 开放讨论 (8 min) — 客户反馈
```

## 45 分钟版（标准季度 QBR）

```
1. 执行摘要 (5 min)
2. 价值回顾 (10 min)
3. 洞察与建议 (10 min)
4. 路线图对齐 (10 min)
5. 下一步 (5 min)
6. 开放讨论 (5 min)
```

## 60 分钟版（高管到场/年度回顾）

```
1. 执行摘要 (5 min)
2. 价值回顾 (15 min) — 更多案例、更详细的 ROI
3. 行业趋势 (5 min) — 客户所在行业的 insights
4. 洞察与建议 (10 min)
5. 产品路线图 (10 min) — 对未来 6-12 月的展望
6. 联合创新 (5 min) — "我们还可以一起做什么"
7. 下一步 (5 min)
8. 开放讨论 (5 min)
```

## Slide 画布模板

### Slide 1: 执行摘要
```
[标题]：[客户名] × [公司名] — Q[N] 回顾

[副标题]：过去一个季度我们一起达成了...

[3 个大数字]
[数字 1]         [数字 2]         [数字 3]
[一句话含义]     [一句话含义]     [一句话含义]

[底部 1 行]：下一季度目标 → [一句话]
```

### Slide: ROI 展示
```
[标题]：[功能/模块] 在过去 6 个月帮你们节省了 [X] 小时

[左栏]
使用前：平均每个 [任务] 耗时 [X] 分钟
使用后：平均每个 [任务] 耗时 [Y] 分钟
提升：[Z]%

[右栏]
换算：
× [人数] 人 × [频率] 次/周 × 26 周
= 节省 [X] 小时
= ¥[Y] （按平均时薪 ¥[Z]/h）

[底部]：放大到全公司 → [更大的数字]
```

## 每页检查清单

- [ ] 标题是结论而非主题（"响应速度提升 40%" 而非 "响应速度数据"）
- [ ] 每页有且仅有一个核心观点
- [ ] 数据有来源标注
- [ ] 所有数字有 "so what" 解读
- [ ] 客户 logo/名称在每页右上角
- [ ] 无内部敏感信息（如其他客户数据、内部定价）
- [ ] 不超过 10 页（45 分钟版）
