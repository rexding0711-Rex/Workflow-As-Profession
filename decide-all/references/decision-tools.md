## 决策框架

### 快速决策框架（< 30分钟）

```
1. 明确决策目标（1分钟）
2. 列出选项（2分钟）
3. 确定关键标准（2分钟）
4. 快速评分（10分钟）
5. 选择最优（5分钟）
6. 记录决策（10分钟）
```

### 标准决策框架（< 4小时）

```
1. 问题定义（30分钟）
2. 信息收集（60分钟）
3. 方案生成（30分钟）
4. 评估分析（90分钟）
5. 决策选择（30分钟）
6. 计划制定（30分钟）
7. 文档记录（30分钟）
```

### 重大决策框架（1-3天）

```
1. 问题定义与边界（2小时）
2. 深度信息收集（4小时）
3. 多方案设计（4小时）
4. 详细评估分析（8小时）
5. 风险识别与应对（4小时）
6. 决策会议（2小时）
7. 执行计划（4小时）
8. 文档与沟通（2小时）
```

## 投资决策工具

| 工具 | 公式 | 说明 |
|------|------|------|
| NPV | Σ(CF_t / (1+r)^t) | 净现值，>0则投资 |
| IRR | NPV=0时的r | 内部收益率，>要求回报率则投资 |
| ROI | (收益-成本)/成本 | 投资回报率 |
| 回收期 | 累计现金流=0的时间 | 静态/动态回收期 |
| TCO | 总拥有成本 | 包含采购、运营、维护、处置 |
| 盈亏平衡 | 收入=成本时的产量 | 盈亏平衡分析 |

## 决策质量检查

| 检查项 | 问题 | 标准 |
|--------|------|------|
| 目标清晰 | 决策目标是否明确？ | 可衡量、可验证 |
| 信息充分 | 信息是否足够？ | 关键信息无缺失 |
| 方案全面 | 是否考虑了所有选项？ | 至少3个选项 |
| 标准合理 | 评估标准是否恰当？ | 完整、独立、可衡量 |
| 偏见检查 | 是否存在认知偏见？ | 多角度验证 |
| 风险考虑 | 风险是否充分识别？ | 关键风险有应对 |
| 可执行性 | 决策是否可执行？ | 有明确执行计划 |
| 时间合适 | 决策时机是否合适？ | 不过早/过晚 |

## 常见决策陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| 分析瘫痪 | 过度分析，无法决策 | 设定决策截止时间 |
| 群体思维 | 团队意见趋同 | 引入魔鬼代言人 |
| 沉没成本 | 被已投入影响 | 关注未来收益 |
| 锚定效应 | 被初始信息影响 | 多角度获取信息 |
| 框架效应 | 表述方式影响判断 | 多框架呈现 |
| 过度自信 | 高估判断准确性 | 强制寻找反面证据 |
| 近因效应 | 重视近期信息 | 考虑长期历史 |
| 可得性偏差 | 重视易回忆信息 | 系统性数据收集 |
