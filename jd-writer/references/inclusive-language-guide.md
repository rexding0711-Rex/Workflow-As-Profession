# 包容性语言指南

## 为什么重要

- 研究显示：性别化的 JD 会让 40% 的潜在候选人不敢投递
- "他/她"暗示特定性别偏好
- "年轻有活力"暗示年龄歧视
- 包容性语言让合格候选人自筛入而非自筛出

## 词汇替换表

### 性别

| ❌ 避免 | ✅ 推荐 |
|--------|--------|
| 他/她 | 他们、候选人、你 |
| manpower | workforce, team, headcount |
| guys | team, everyone, folks |
| chairman | chair, chairperson |
| salesman | sales representative |
| waitress | server |

### 年龄

| ❌ 避免 | ✅ 推荐 |
|--------|--------|
| 年轻有活力 | 充满热情、积极主动 |
| 应届生优先 | 欢迎各阶段候选人 |
| 35 岁以下 | 不做年龄限制 |
| 资深（作为年龄暗示） | 用技能描述替代："能独立负责..." |
| 10 年以上经验 | "精通 XXX 领域，能独立设计..." |

### 能力

| ❌ 避免 | ✅ 推荐 |
|--------|--------|
| 身体健康、五官端正 | 删除（除非是真实的职业要求） |
| 正常视力 | 删除 |
| 能承受高强度工作 | "能适应快速迭代的节奏" |
| 吃苦耐劳 | "有韧性，能在不确定性中找到方向" |

### 文化

| ❌ 避免 | ✅ 推荐 |
|--------|--------|
| culture fit | culture add（你能带来什么新东西） |
| 985/211 优先 | "计算机相关专业" 或删除 |
| 名校背景 | 删除 |
| 海归优先 | "有国际化视野"（如果确实是岗位需求） |

## 检查清单

每份 JD 写完后，逐项检查：

- [ ] 所有代词性别中立
- [ ] 无年龄限制性语言
- [ ] 经验要求描述能力而非年限
- [ ] 无身体/外貌要求（除非职业必须）
- [ ] 无学校/学历门槛（除非职业必须）
- [ ] 使用"culture add"而非"culture fit"
- [ ] 平等雇佣声明已包含

## 平等雇佣声明模板

```
[公司名] 是平等机会雇主。我们不会基于种族、肤色、宗教、性别、性取向、
性别认同或表达、国籍、年龄、残障状况、婚姻状况、退伍军人身份或任何其他
受法律保护的特征进行歧视。我们欢迎不同背景、经验和视角的候选人加入我们。
```
