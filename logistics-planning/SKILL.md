# Logistics Planning — 物流规划

物流成本通常占制造业营收的 5-15%。优化物流不是在 Excel 里找最低运费，
是在运输成本、库存成本和服务水平三者之间找到最优平衡。

## 何时触发

用户提及：logistics planning、物流规划、运输优化、仓储布局、3PL 选型、
物流成本、配送网络、货运、路线优化 等关键词时触发。

## 运输模式选择

| 模式 | 成本 | 速度 | 适用 |
|------|------|------|------|
| 公路 | 中 | 中 | <1000km、灵活配送 |
| 铁路 | 低 | 慢 | 大宗物资、>500km |
| 海运 | 最低 | 最慢 | 国际大宗、非紧急 |
| 空运 | 最高 | 最快 | 高价值、紧急、小件 |
| 多式联运 | 中低 | 中 | 铁路+公路、海运+铁路 |

### 决策公式
```
总物流成本 = 运输成本 + 在途库存成本 + 仓储成本 + 缺货成本
```
不是选运输成本最低的模式，是选总物流成本最低的模式。

## 仓储布局优化

### 网络设计问题
- 几个仓库？放在哪？
- 每个仓库覆盖多大半径？
- 集中式 vs 分布式？

| 模式 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| 单仓 | 低库存、低管理成本 | 末端配送慢 | 小市场、高价值 |
| 多仓 | 快速响应、低运费 | 高库存、管理复杂 | 全国覆盖、电商 |
| Hub-and-Spoke | 平衡成本和速度 | 转运增加时间和破损 | 快递物流 |

## 输出模板

```markdown
# [公司/产品线] — 物流优化方案

## 当前状态
- 年物流成本：¥[X]（占营收 [X]%）
- 网络：[X] 个仓库 / [X] 条干线
- 时效：平均 [X] 天到客户

## 优化机会
| 机会 | 年节省 | 实施成本 | ROI |
|------|--------|---------|-----|
| [方案] | ¥[X] | ¥[Y] | [X] 月回本 |

## 推荐方案
[具体方案 + 时间线 + 风险]
```

## 质量原则
1. 物流成本看 Total Landed Cost，不看运费单价
2. 每 2 年做一次网络优化分析（市场变了、客户变了）
3. 3PL 合同不要太长（2-3 年），保留重新谈判空间
