# 监测收集

一站式信息监测与收集中心：给一个URL或信息源，自动完成监测变化、抓取内容、汇总资讯、清洗数据、过滤噪音、提取行动项，输出结构化摘要。

## 定位

- **All-in-One**：不是多个零散技能，是一个统一的监测入口
- **输入灵活**：支持URL、网页、微信链接、文档、数据表、聊天记录
- **输出统一**：不管什么来源，输出都是结构化的摘要+行动项

## 执行规则

1. **触发时**：接收用户输入（URL/链接/文字/文件），确认来源类型和监测目标
2. **识别来源时**：判断输入类型：
   - 网页/网站 → 监测变化 + 抓取内容
   - 新闻/资讯 → 汇总摘要 + 重要性排序
   - 数据/表格 → 清洗 + 异常检测 + 趋势分析
   - 聊天记录/会议 → 提取要点 + 行动项
   - 多源混合 → 整合对比 + 去重
3. **执行时**：根据来源类型自动选择处理模块：
   - 网站监测：`references/site-monitor.md`
   - 资讯汇总：`references/news-digest.md`
   - 数据清洗：`references/data-clean.md`
   - 噪音过滤：`references/noise-filter.md`
   - 会议/聊天提取：`references/chat-extract.md`
4. **输出时**：统一格式：来源摘要 + 关键变化 + 行动项 + 数据可信度标注
5. **升级时**：如果用户需要深度分析，推荐调用 `mba-is-bullshit` 或 `meeting-to-action`
