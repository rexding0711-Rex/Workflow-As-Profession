# 噪音过滤模块

## 信息分类

| 类别 | 定义 | 处理方式 |
|------|------|----------|
| **关键信号** | 必须立即处理 | 立即推送 |
| **重要信息** | 需要关注 | 日报汇总 |
| **一般信息** | 有空再看 | 周报或归档 |
| **噪音** | 可忽略 | 自动过滤 |
| **误报** | 已知非问题 | 学习模式，持续优化 |

## 降噪规则

### 重复合并
- 同一来源5分钟内多次触发 → 合并为1条，记录次数
- 周期性变化（如每日定时更新）→ 标记为"周期性"，不重复推送

### 误报识别
- 时间戳变化 → 忽略
- 广告/推荐内容变化 → 忽略
- 格式变化（空格/换行/HTML标签）→ 忽略
- 第三方嵌入内容变化 → 忽略

### 关键词过滤
- 白名单：包含特定关键词才推送（如"政策"、"发布"、"合作"）
- 黑名单：包含特定关键词过滤（如"广告"、"推广"、"无关"）

### 重要性判定

| 信号 | 级别 | 说明 |
|------|------|------|
| 价格大幅变化>10% | 紧急 | 立即推送 |
| 政策重大更新 | 紧急 | 立即推送 |
| 竞品新功能/产品 | 重要 | 日报汇总 |
| 常规数据更新 | 一般 | 日报汇总 |
| 页面样式微调 | 可忽略 | 不推送 |
| 时间戳/广告变化 | 误报 | 自动过滤 |

## 输出

过滤后的关键信息列表，每条标注：来源、时间、重要级别、可信度。
