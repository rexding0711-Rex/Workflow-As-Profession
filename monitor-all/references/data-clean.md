# 数据清洗模块

## 常见数据源格式

| 来源 | 格式 | 常见问题 | 清洗动作 |
|------|------|----------|----------|
| **Excel/CSV** | 表格 | 列名不一致、合并单元格 | 统一列名、拆分单元格 |
| **数据库** | SQL表 | 编码问题、时间戳格式 | 编码转换、时间标准化 |
| **API** | JSON | 嵌套结构、空值 | 扁平化、空值填充 |
| **日志** | 文本 | 非结构化、格式混杂 | 正则提取、结构化 |
| **PDF** | 文档 | 图片型、扫描件 | OCR提取、文本解析 |
| **网页** | HTML | 标签混杂、动态渲染 | 去标签、提取正文 |

## 清洗步骤

### 1. 缺失值处理
| 策略 | 适用场景 | 示例 |
|------|----------|------|
| 删除整行 | 缺失率>50% | 删除关键字段全空记录 |
| 填充默认值 | 有明确默认值 | 状态缺失→"待确认" |
| 填充均值/中位数 | 数值型 | 年龄缺失→用中位数 |
| 前向/后向填充 | 时间序列 | 日报缺失→用前一天值 |
| 标记"未知" | 分类字段 | 地区缺失→"未知" |

### 2. 格式标准化
- 日期：统一为 YYYY-MM-DD
- 金额：统一为数字，去除货币符号
- 手机号：统一为11位数字
- 文本：去除前后空格，统一全角半角

### 3. 重复处理
- 完全重复：删除重复，保留一条
- 部分重复：标记冲突，待人工确认
- 近似重复：标准化后合并

### 4. 异常值标记
- 数值异常：超出3σ或IQR范围
- 日期异常：未来日期或早于业务起始日期
- 逻辑异常：离职日期早于入职日期
