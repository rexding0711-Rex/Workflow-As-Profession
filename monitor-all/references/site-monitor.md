# 网站监测模块

## 监测目标识别

| 来源类型 | 示例 | 监测维度 | 输出 |
|---------|------|---------|------|
| **竞品官网** | nvidia.com, tesla.com | 产品/价格/公告/页面 | 变化摘要 |
| **电商页面** | 淘宝/京东商品页 | 价格/库存/促销/评价 | 价格变动 |
| **政策页面** | 政府/监管公告 | 政策/法规/标准 | 政策摘要 |
| **媒体页面** | 新闻/研报页面 | 文章/数据/观点 | 资讯摘要 |
| **数据页面** | 统计/数据库 | 数据/指标/图表 | 数据快照 |

## 监测配置

```yaml
monitor_config:
  url: 目标URL
  frequency: [hourly/daily/weekly]
  dimension: [content/price/announcement/structure]
  keywords: 关注关键词列表
  threshold: 变化阈值（价格>1%, 文本>50字符）
  notification: [immediate/daily_digest]
```

## 变化类型识别

| 变化类型 | 判定方法 | 示例 |
|---------|----------|------|
| **文本变化** | 文本相似度<90% | 产品描述更新 |
| **数值变化** | 数字字段差值>阈值 | 价格、库存、评分 |
| **结构变化** | DOM节点增减 | 页面改版、新增板块 |
| **列表更新** | 列表项数量增加 | 新闻列表、产品列表 |
| **状态变化** | 可用性状态码变化 | 网站宕机 |

## 重要性分级

| 级别 | 条件 | 处理方式 |
|------|------|----------|
| **紧急** | 价格大幅变化>10%, 重大政策, 竞品新功能 | 立即推送 |
| **重要** | 价格小幅变化, 常规更新 | 日报汇总 |
| **一般** | 页面微调, 非关键文本 | 周报或忽略 |
| **误报** | 时间戳, 广告轮播 | 自动过滤 |

## 输出格式

```markdown
## [URL] 监测报告

### 变化摘要
- **变化时间**：YYYY-MM-DD HH:MM
- **变化类型**：[内容/价格/结构/列表]
- **变化前**：...
- **变化后**：...
- **重要性**：[紧急/重要/一般]

### 关键内容提取
- [要点1]
- [要点2]

### 数据可信度
- [信息]：来源/可信度
```
