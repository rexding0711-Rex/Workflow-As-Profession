# 聊天/会议提取模块

## 内容识别

### 议题识别
- 关键词："今天讨论..."、"关于..."、"议题..."、"第一个..."
- 列表结构：1. 2. 3. / 首先、其次、最后

### 决策识别
- 关键词："决定..."、"定下来..."、"通过了..."、"同意..."
- 条件句式："如果...就..."、"除非...否则..."
- 数字/日期：出现具体数字、日期、金额

### 行动项识别
- 关键词："负责..."、"跟进..."、"落实..."、"确认..."
- 人名+动词："[人名] 去..."、"[人名] 负责..."
- 时间词："本周"、"月底前"、"下周三"
- 模糊项："看看"、"了解一下" → 标记"待澄清"

### 信息同步
- 关键词："通知大家..."、"同步一下..."、"已知..."
- 无行动、无决策，仅陈述事实

## 结构化输出

```markdown
## 会议纪要

### 基本信息
- **时间**：
- **参会人**：
- **来源**：微信/钉钉/飞书/会议录音

### 议题与决策
- **议题1**：[名称]
  - 决策：...
  - 行动项：谁/做什么/截止

### 行动项汇总
| # | 事项 | 责任人 | 截止日期 | 优先级 |
|---|------|--------|----------|--------|
| 1 | | | | P0/P1/P2 |

### 待澄清
- [事项]：需确认

### 信息同步
- [信息]：大家已知，无需行动
```

## 提取质量

- 每个议题有明确的决策或行动项
- 模糊表述转化为具体待办或标记待澄清
- 责任人尽量明确，不确定的标记"待确认"
- 时间尽量明确，模糊的标记"待确认时间"

## 后续跟进

提取后自动生成跟进建议：
- 截止前2天：确认进度
- 截止当天：强调完成
- 超时：升级提醒
