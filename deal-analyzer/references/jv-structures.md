# 合资架构模板

## 方案A：控股型合资公司

### 架构图
```
我方（51%） + 对方（49%）
        ↓
    合资公司（100%）
        ↓
    运营业务
```

### 特点
- 我方控股，控制经营
- 对方参股，分享利润
- 简单直接，无额外架构

### 适用场景
- 我方有核心资源/技术/市场
- 对方提供资金/渠道/补充资源
- 双方信任度高，合作初期

### 关键参数
| 参数 | 标准 | 可谈判空间 |
|------|------|------------|
| 股权比例 | 51:49 | 可调整为55:45或60:40 |
| 董事会 | 我方3席，对方2席 | 可调整为4:3或5:4 |
| 总经理 | 我方委派 | 可协商联合任命 |
| 利润分配 | 按股权比例 | 可设置前期倾斜 |
| 退出机制 | 股权转让/回购 | 可设置IPO退出 |

### 风险
- 对方可能要求更多控制权
- 经营决策可能产生分歧
- 退出时股权转让需对方同意

## 方案B：参股型合资公司

### 架构图
```
我方（30%） + 对方（70%）
        ↓
    合资公司（100%）
        ↓
    运营业务
```

### 特点
- 对方控股，控制经营
- 我方参股，分享利润
- 我方风险有限，但控制力弱

### 适用场景
- 对方有核心资源/市场/品牌
- 我方提供资金/技术/补充资源
- 我方希望有限参与，不深度介入

### 关键参数
| 参数 | 标准 | 可谈判空间 |
|------|------|------------|
| 股权比例 | 30:70 | 可调整为40:60 |
| 董事会 | 我方1席，对方4席 | 可争取2席 |
| 总经理 | 对方委派 | 可协商副总经理 |
| 利润分配 | 按股权比例 | 可设置业绩挂钩 |
| 退出机制 | 股权转让/回购 | 可设置优先回购权 |

### 风险
- 我方控制力弱，可能被动
- 对方经营不善，我方难以干预
- 利润分配可能不透明

## 方案C：SPV（特殊目的公司）架构

### 架构图
```
我方（100%）          对方（100%）
    ↓                    ↓
SPV-A（控股）      SPV-B（参股）
    ↓                    ↓
    合资公司（SPV-A控股）
        ↓
    运营业务
```

### 特点
- 通过SPV隔离风险
- SPV-A控股合资公司
- SPV-B参股合资公司
- 母公司风险隔离

### 适用场景
- 项目风险较高
- 需要保护母公司资产
- 税务优化需求
- 跨境合作（外资准入限制）

### 关键参数
| 参数 | 标准 | 可谈判空间 |
|------|------|------------|
| SPV层级 | 1层SPV | 可多层（税务优化） |
| 股权比例 | 通过SPV间接控制 | 可调整SPV股权 |
| 风险隔离 | 债务/诉讼隔离 | 可设置担保上限 |
| 税务优化 | 利用SPV所在地 | 需税务顾问设计 |
| 退出机制 | 转让SPV股权 | 可设置分层退出 |

### 风险
- 架构复杂，管理成本高
- 税务合规风险
- 多层架构可能触发反避税

## 方案D：VIE架构（可变利益实体）

### 架构图
```
境外上市公司（开曼）
        ↓
    WFOE（外商独资企业）
        ↓
    VIE协议控制
        ↓
    境内运营公司（牌照持有方）
```

### 特点
- 境外上市/融资
- 境内运营公司持有关键牌照
- WFOE通过VIE协议控制境内公司
- 外资规避限制行业准入

### 适用场景
- 需要境外上市（美股/港股）
- 业务涉及外资限制行业（教育、互联网、媒体等）
- 需要引入外资美元基金

### 关键参数
| 参数 | 标准 | 可谈判空间 |
|------|------|------------|
| 上市地 | 美股/港股 | 可协商 |
| VIE协议 | 独家服务、股权质押、表决权委托 | 需律师设计 |
| 牌照持有 | 境内运营公司 | 不可变 |
| 利润转移 | 服务费/知识产权费 | 需税务合规 |
| 退出机制 | 境外股权转让 | 需外汇审批 |

### 风险
- VIE架构法律风险（中国不承认）
- 外汇管制风险
- 政策风险（随时可能禁止）
- 架构复杂，成本高

---

## 方案E：联营（Consortium）

### 架构图
```
我方（牵头） + 对方A + 对方B + ...
        ↓
    联营体（非法人）
        ↓
    联合运营（各自负责部分）
```

### 特点
- 多方合作，无独立法人
- 各自负责部分业务
- 按约定分配利润/风险
- 灵活，但责任不清

### 适用场景
- 大型项目，多方合作
- 各自有专精领域
- 不想设立独立公司

### 关键参数
| 参数 | 标准 | 可谈判空间 |
|------|------|------------|
| 责任划分 | 按业务模块 | 需明确边界 |
| 利润分配 | 按贡献/约定 | 可设置业绩挂钩 |
| 风险承担 | 各自承担部分 | 可设置连带责任上限 |
| 决策机制 | 联合体会议 | 可设置牵头方特权 |
| 退出机制 | 提前通知 | 可设置违约金 |

### 风险
- 责任不清，容易扯皮
- 利润分配可能不公
- 一方退出，项目可能中断

## 架构对比矩阵

| 维度 | 控股型 | 参股型 | SPV | VIE | 联营 |
|------|--------|--------|-----|-----|------|
| 控制权 | 高 | 低 | 中 | 中 | 分散 |
| 风险隔离 | 低 | 低 | 高 | 中 | 低 |
| 复杂度 | 低 | 低 | 高 | 极高 | 中 |
| 成本 | 低 | 低 | 中 | 高 | 中 |
| 税务优化 | 低 | 低 | 高 | 高 | 低 |
| 退出灵活性 | 中 | 中 | 高 | 低 | 高 |
| 适用信任度 | 高 | 中 | 中 | 低 | 中 |
| 外资准入 | 不适用 | 不适用 | 适用 | 适用 | 适用 |

## 架构选择指南

| 场景 | 推荐架构 | 原因 |
|------|----------|------|
| 我方有核心技术，对方有资金 | 控股型 | 保持技术控制权 |
| 对方有核心市场，我方有资金 | 参股型 | 借助对方市场 |
| 项目风险高，需保护母公司 | SPV | 风险隔离 |
| 需要境外上市/美元融资 | VIE | 规避外资限制 |
| 大型项目，多方合作 | 联营 | 灵活分工 |
| 跨境合作，税务优化 | SPV | 利用不同税制 |
| 快速试点，后期调整 | 控股型 | 简单高效 |
| 深度合作，长期绑定 | SPV | 风险隔离+灵活退出 |

## 退出机制设计

| 架构 | 退出方式 | 特点 |
|------|----------|------|
| 控股型 | 股权转让、回购、IPO | 需对方同意或优先购买 |
| 参股型 | 股权转让、回购、IPO | 控制力弱，退出被动 |
| SPV | 转让SPV股权、清算SPV | 灵活，不影响母公司 |
| VIE | 境外股权转让、退市 | 复杂，需外汇审批 |
| 联营 | 提前通知、解散 | 简单，但可能违约 |

## 关键谈判要点

1. **控制权**：董事会席位、重大事项表决、总经理任命
2. **利润分配**：按股权比例 vs 按贡献 vs 前期倾斜
3. **风险隔离**：债务上限、担保上限、连带责任
4. **退出机制**：触发条件、估值方法、优先权
5. **竞业禁止**：范围、时间、补偿
6. **知识产权**：归属、许可、后续开发
7. **保密**：范围、时间、例外
8. **争议解决**：仲裁 vs 诉讼、地点、适用法律
