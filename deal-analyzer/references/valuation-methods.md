# 估值方法

## 一、DCF（现金流折现法）

### 原理
企业价值 = 未来自由现金流的折现值

### 公式
```
Enterprise Value = Σ(FCF_t / (1+WACC)^t) + Terminal Value / (1+WACC)^n

其中：
- FCF_t = 第t年的自由现金流
- WACC = 加权平均资本成本
- Terminal Value = 终值（通常用永续增长模型或退出倍数）
- n = 预测期（通常5-10年）
```

### 关键参数
| 参数 | 说明 | 常见范围 | 敏感性 |
|------|------|----------|--------|
| 收入增长率 | 未来收入增长 | 10-50%（早期） | 极高 |
| 毛利率 | 收入-成本 | 30-80% | 高 |
| 运营费用率 | 销售/管理/研发费用 | 20-60% | 高 |
| 资本开支 | 维持和增长投资 | 收入的5-20% | 中 |
| 营运资本变动 | 应收/存货/应付 | 收入的-5%至5% | 中 |
| WACC | 折现率 | 10-20% | 极高 |
| 永续增长率 | 终值增长 | 2-5% | 高 |

### 适用场景
- 有稳定现金流的公司（成长期后）
- 盈利模式清晰
- 可预测性强

### 局限性
- 早期公司现金流为负，不适用
- 对增长率假设敏感
- 终值占比可能>50%，不确定性大

### 四大的做法
- 通常预测5-10年详细现金流
- 终值用退出倍数法（EV/EBITDA）或永续增长法
- 做敏感性分析（增长率±2%，WACC±2%）
- 做情景分析（乐观/基准/悲观）

## 二、可比公司法（Trading Comps）

### 原理
用同行业上市公司的估值倍数来估值目标公司

### 常用倍数
| 倍数 | 公式 | 适用场景 | 常见范围 |
|------|------|----------|----------|
| P/E | 股价/每股收益 | 盈利稳定公司 | 10-30x |
| EV/EBITDA | 企业价值/息税折旧前利润 | 资本密集型 | 8-20x |
| EV/Revenue | 企业价值/收入 | 早期/亏损公司 | 2-10x |
| P/S | 市值/收入 | SaaS/高增长 | 5-20x |
| P/B | 市值/净资产 | 资产重型公司 | 1-3x |

### 可比公司选择
1. **同行业**：业务相似、客户群体相似
2. **同规模**：收入规模、员工数相似
3. **同阶段**：成长阶段相似（早期/成长期/成熟期）
4. **同市场**：主要市场相似（中国/美国/全球）
5. **同模式**：商业模式相似（SaaS/平台/产品）

### 调整因素
- 流动性折扣（非上市公司折扣20-30%）
- 控制权溢价（控股权溢价10-20%）
- 规模折扣（小公司折扣）
- 成长溢价（高成长公司溢价）

### 适用场景
- 有成熟可比上市公司
- 行业估值体系成熟
- 公司有一定规模

### 四大的做法
- 选取5-10家可比公司
- 计算各倍数的均值、中位数、高低区间
- 对目标公司进行调整
- 给出估值区间（不是单一数值）

## 三、可比交易法（Transaction Comps）

### 原理
用同行业近期的并购/投资交易来估值目标公司

### 常用倍数
| 倍数 | 公式 | 适用场景 |
|------|------|----------|
| EV/Revenue | 交易价值/收入 | 早期公司 |
| EV/EBITDA | 交易价值/EBITDA | 盈利公司 |
| P/E | 交易价格/利润 | 盈利公司 |
| 每股价格 | 交易价格/股数 | 所有公司 |

### 可比交易选择
1. **同行业**：业务相似
2. **近期**：通常1-2年内
3. **同阶段**：成长阶段相似
4. **同规模**：规模相似
5. **同市场**：市场相似

### 关键信息
- 交易日期（市场环境影响）
- 交易结构（股权/资产）
- 支付对价（现金/股票/混合）
- 交易背景（战略/财务）
- 溢价水平（vs交易前估值）

### 适用场景
- 有近期可比交易
- 市场活跃
- 行业估值体系成熟

### 四大的做法
- 选取5-10笔可比交易
- 分析交易背景（战略vs财务）
- 计算各倍数
- 考虑市场变化（时间调整）
- 给出估值区间

## 四、VC法（风险投资法）

### 原理
从投资人退出时的目标回报，倒推当前估值

### 公式
```
当前估值 = 退出时估值 / (1 + 目标IRR)^n

其中：
- 退出时估值 = 退出时收入 × 退出倍数
- 目标IRR = 30-50%（早期）/ 20-30%（成长期）
- n = 投资年限（通常3-7年）
```

### 关键参数
| 参数 | 说明 | 常见范围 |
|------|------|----------|
| 退出时收入 | 退出时的年收入预测 | 基于增长假设 |
| 退出倍数 | 退出时的估值倍数 | 可比公司/交易 |
| 目标IRR | 投资人要求的回报率 | 30-50% |
| 投资年限 | 从投资到退出 | 3-7年 |
| 稀释率 | 后续融资稀释 | 20-50% |

### 适用场景
- 早期创业公司
- 无盈利/无现金流
- VC/PE投资

### 局限性
- 对退出假设敏感
- 对IRR假设敏感
- 不反映当前价值

### 四大的做法
- 早期投资通常用此方法
- 需要详细论证退出假设
- 敏感性分析（IRR±10%，退出倍数±2x）

## 五、成本法（Asset-Based）

### 原理
企业价值 = 净资产价值（资产 - 负债）

### 适用场景
- 资产重型公司（房地产、制造业）
- 清算价值
- 账面价值接近市场价值

### 局限性
- 不反映无形资产价值
- 不反映未来盈利能力
- 不适用于科技公司

## 六、方法选择指南

| 公司阶段 | 推荐方法 | 原因 |
|----------|----------|------|
| 种子/天使 | VC法 | 无收入/无盈利 |
| Pre-A/A | VC法 + 可比交易 | 有收入但无盈利 |
| B/C | 可比公司 + DCF | 有收入，接近盈利 |
| 成长期 | DCF + 可比公司 | 有稳定现金流 |
| 成熟期 | DCF + 可比公司 | 现金流稳定 |
| 重资产 | 成本法 + DCF | 资产价值重要 |

## 估值区间

四大的惯例是给出**估值区间**，而不是单一数值：

```markdown
# 估值分析

## 估值区间
| 方法 | 低值 | 中值 | 高值 | 权重 |
|------|------|------|------|------|
| DCF | 2.5亿 | 3.0亿 | 3.5亿 | 40% |
| 可比公司 | 2.8亿 | 3.2亿 | 3.8亿 | 30% |
| 可比交易 | 2.2亿 | 2.8亿 | 3.2亿 | 20% |
| VC法 | 2.0亿 | 2.5亿 | 3.0亿 | 10% |
| **综合** | **2.4亿** | **2.9亿** | **3.4亿** | **100%** |

## 关键假设
- 收入增长率：30%（基准），20%（悲观），40%（乐观）
- WACC：15%（基准），13%（乐观），17%（悲观）
- 退出倍数：8x EV/Revenue（基准），6x（悲观），10x（乐观）

## 敏感性分析
| 情景 | 增长率 | WACC | 估值 | 概率 |
|------|--------|------|------|------|
| 乐观 | 40% | 13% | 3.4亿 | 25% |
| 基准 | 30% | 15% | 2.9亿 | 50% |
| 悲观 | 20% | 17% | 2.4亿 | 25% |

## 建议估值
- 投前估值区间：2.5-3.0亿
- 谈判锚点：2.8亿
- 底线：2.5亿（低于此建议放弃）
```

## 谈判中的估值策略

| 策略 | 适用场景 | 操作方式 |
|------|----------|----------|
| 锚定效应 | 先报价 | 先报高值，预留谈判空间 |
| 可比交易 | 有近期交易 | 引用最近3笔交易的估值 |
| 成长故事 | 高增长 | 强调未来潜力，而非当前 |
| 多种方法 | 有争议 | 展示多种方法，取交集 |
| 敏感性分析 | 对方质疑 | 展示假设变化对估值的影响 |
| 分阶段估值 | 长期合作 | 本轮估值较低，下轮有对赌提升 |
