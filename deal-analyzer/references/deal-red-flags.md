# 交易红旗信号

## 必须否决的条款（Deal Breaker）

以下条款如果出现，必须坚决谈判，否则建议放弃交易。

### 1. 创始人无限连带责任
**条款内容**：创始人以个人资产担保公司债务或赔偿义务。
**风险**：创始人可能因公司问题个人破产。
**应对**：限制赔偿责任上限为投资额或公司资产，创始人个人不承担连带责任。
**底线**：如果投资人坚持，建议放弃交易。

### 2. 对赌惩罚超过投资额
**条款内容**：业绩不达标时，创始人需赔偿的金额超过投资款。
**风险**：本质是投资人无风险、创始人承担全部风险的高利贷。
**应对**：设置赔偿上限为投资额，或转为股权调整而非现金赔偿。
**底线**：赔偿上限不得超过投资额。

### 3. 投资人可随时要求回购
**条款内容**：投资人可在任何时间、无条件要求公司回购其股权。
**风险**：公司随时面临资金压力，创始人无法规划长期。
**应对**：限制回购触发条件（5年后、特定事件）和回购价格（按回报率计算）。
**底线**：回购必须有时间限制和合理回报率。

### 4. 终身竞业禁止
**条款内容**：创始人终身不得从事与公司竞争的业务。
**风险**：创始人即使离开公司，也无法在相关领域工作。
**应对**：限制为任职期间+离职2-3年，范围限制为公司直接竞争领域。
**底线**：竞业禁止时间不超过3年，范围必须明确限定。

### 5. A轮投资人股权比例>40%
**条款内容**：早期投资人（A轮）要求超过40%的股权。
**风险**：创始人失去控制权，后续融资空间不足，投资人动力下降。
**应对**：稀释后创始人保持>50%或至少保持控制权。
**底线**：A轮投资人不超过30%，创始人保持>50%。

### 6. 拖售权简单多数触发
**条款内容**：持有简单多数股权的投资人可强制出售公司。
**风险**：小投资人可强制创始人出售公司，创始人被扫地出门。
**应对**：要求 supermajority（如75%）或创始人同意。
**底线**：拖售权必须经创始人同意或 supermajority 批准。

### 7. 全部TS有约束力
**条款内容**：签署TS后，所有条款均具有约束力，不可退出。
**风险**：创始人失去谈判空间，无法接触其他投资人。
**应对**：TS中明确仅保密、排他、费用条款有约束力，其他条款无法律约束力。
**底线**：TS必须有"无约束力"条款（except for...）。

### 8. 排他期超过90天
**条款内容**：排他期超过90天，或自动续约。
**风险**：创始人被长期锁定，无法接触其他投资人，机会成本高。
**应对**：限制排他期为30-60天，明确到期不自动续约。
**底线**：排他期不超过60天。

### 9. 清算优先>1x或参与式
**条款内容**：清算优先超过1倍，或参与式（优先分配后仍参与剩余分配）。
**风险**：创始人清算时几乎无收益，投资人拿走全部。
**应对**：争取1x非参与（优先拿回1倍，不再参与剩余）。
**底线**：最多接受1x参与，不接受>1x。

### 10. 全棘轮反稀释
**条款内容**：后续融资估值低于本轮时，投资人股权按最新低价重新计算。
**风险**：创始人股权被大幅稀释，可能丧失控制权。
**应对**：改为加权平均（Broad-Based Weighted Average）。
**底线**：绝不接受全棘轮。

## 需要律师介入的条款

以下条款不一定会否决交易，但必须由专业律师审查。

### 11. 赔偿无上限
**风险**：创始人可能承担无限赔偿，个人资产不保。
**应对**：设置赔偿上限（如投资额或公司资产），设置赔偿期限（如交割后2-3年）。

### 12. 知识产权归属不清
**风险**：创始人可能丧失核心技术，或公司无法使用关键IP。
**应对**：明确区分职务发明（归公司）和个人发明（归个人），签署IP转让协议。

### 13. 争议解决地点在对方所在地
**风险**：诉讼/仲裁成本高，法律环境不熟悉，可能不利。
**应对**：争取中立地点（如香港、新加坡），或公司所在地。

### 14. 适用法律为对方所在地法律
**风险**：不熟悉法律环境，解释可能不利。
**应对**：争取公司所在地法律，或国际通用的法律（如英国法、纽约州法）。

### 15. 复杂对赌公式
**风险**：公式可能隐含不利条款，计算复杂容易出错。
**应对**：要求简化公式，明确每个参数的定义和计算方式。

### 16. 多层SPV架构
**风险**：税务/法律风险复杂，可能触发反避税调查。
**应对**：聘请专业税务顾问和律师设计架构，确保合规。

### 17. 创始人股份无Vesting或Vesting过长
**风险**：创始人无激励长期留任，或Vesting过长导致激励不足。
**应对**：标准4年Vesting，1年cliff，创始人已工作的时间可加速Vesting。

### 18. 强制购买权（Call Option）
**风险**：投资人可强制购买创始人股权，创始人可能被迫出售。
**应对**：限制触发条件（如创始人离职、违约），设置购买价格（公平市场价值）。

### 19. 最惠国待遇（MFN）
**风险**：投资人自动享受其他投资人更优条款，可能导致条款不断恶化。
**应对**：删除MFN，或限制适用范围（如仅适用于经济条款）。

### 20. 强制共同出售权（Tag-Along）
**风险**：创始人出售股权时，投资人可强制跟售，可能阻碍创始人出售。
**应对**：限制跟售比例（按比例），设置例外（如IPO、战略出售）。

## 红旗信号检查清单

在签署任何交易文件前，检查以下20项红旗信号：

- [ ] 1. 创始人无限连带责任
- [ ] 2. 对赌惩罚超过投资额
- [ ] 3. 投资人可随时要求回购
- [ ] 4. 终身竞业禁止
- [ ] 5. A轮投资人股权>40%
- [ ] 6. 拖售权简单多数触发
- [ ] 7. 全部TS有约束力
- [ ] 8. 排他期>90天
- [ ] 9. 清算优先>1x或参与式
- [ ] 10. 全棘轮反稀释
- [ ] 11. 赔偿无上限
- [ ] 12. 知识产权归属不清
- [ ] 13. 争议解决地点在对方所在地
- [ ] 14. 适用法律为对方所在地法律
- [ ] 15. 复杂对赌公式
- [ ] 16. 多层SPV架构（未论证）
- [ ] 17. 创始人股份无Vesting或过长
- [ ] 18. 强制购买权
- [ ] 19. 最惠国待遇（无限制）
- [ ] 20. 强制共同出售权（无限制）

**如果勾选超过3项，建议重新评估交易是否继续。**
**如果勾选超过5项，建议坚决拒绝交易。**

## 红旗信号应对优先级

| 优先级 | 红旗信号 | 应对方式 | 谈判难度 |
|--------|----------|----------|----------|
| P0 | 1-10（Deal Breaker） | 必须谈判，否则放弃 | 高 |
| P1 | 11-15（需律师介入） | 律师审查，争取修改 | 中-高 |
| P2 | 16-20（可协商） | 尝试谈判，可让步 | 中 |

## 红旗信号与行业的关系

| 行业 | 常见红旗 | 特别关注 |
|------|----------|----------|
| 科技/SaaS | 对赌、IP归属 | 技术估值合理性 |
| 制造业 | 赔偿上限、环保合规 | 资产估值、环保风险 |
| 消费/零售 | 清算优先、拖售权 | 品牌估值、渠道依赖 |
| 医疗/生物 | IP归属、监管合规 | 研发风险、审批时间 |
| 教育 | 政策风险、VIE | 政策变化、牌照 |
| 房地产 | 资产估值、债务 | 抵押、预售监管 |
| 金融 | 牌照、合规 | 监管风险、资本金 |
