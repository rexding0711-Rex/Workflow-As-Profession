# 谈判策略

## 一、让步顺序策略

### 原则
先谈重要条款，再谈次要条款。不要在次要条款上消耗太多精力。

### 优先级排序
1. **第一优先（Deal Breaker）**：估值、股权比例、控制权、清算优先、反稀释
2. **第二优先（重要）**：董事会席位、否决权、对赌、赎回权
3. **第三优先（一般）**：知情权、检查权、费用分担
4. **第四优先（可让）**：排他期、保密期限、争议解决地点

### 让步策略
| 条款 | 你的立场 | 投资人立场 | 交换策略 |
|------|----------|------------|----------|
| 估值 | 3亿 | 2亿 | 可以让步到2.5亿，但要求控制条款改善 |
| 清算优先 | 1x非参与 | 1x参与 | 可以让步到1x参与，但要求估值提高 |
| 董事会 | 3:2 | 2:3 | 可以让步到3:3，但要求一票否决权 |
| 反稀释 | 加权平均 | 全棘轮 | 绝不让步，必须加权平均 |
| 对赌 | 无或宽松 | 严格 | 可以让步到中等，但要求惩罚上限 |

## 二、BATNA（最佳替代方案）

### 定义
如果谈判破裂，你的最佳选择是什么？

### 应用场景
| 场景 | BATNA | 如何利用 |
|------|-------|----------|
| 有多个投资人 | 转向其他投资人 | "我们还在和其他2家沟通" |
| 有现金流 | 暂缓融资 | "我们也可以先不融，用现有资金跑6个月" |
| 有收入 | 自力更生 | "我们今年已经盈利，融资是加速而非必需" |
| 有债务能力 | 银行贷款 | "银行愿意给我们授信2000万" |

### 策略
- 不要让对方知道你的真实BATNA
- 但暗示你有好的BATNA
- BATNA越好，谈判地位越强

## 三、锚定效应

### 原理
第一个报价会成为谈判的锚点，影响后续谈判。

### 策略
| 场景 | 锚定策略 | 效果 |
|------|----------|------|
| 估值谈判 | 你先报3.5亿（高于预期） | 对方可能从2亿提到2.5亿 |
| 条款谈判 | 你先列出所有不可让步条款 | 对方知道你的底线 |
| 时间谈判 | 你先给短排他期（30天） | 对方可能接受 |

### 反锚定
如果对方先报价：
- 不要轻易接受对方的锚点
- 重新设定自己的锚点
- "我们理解您的估值逻辑，但基于我们的数据，我们认为..."

## 四、红脸白脸策略

### 场景
谈判团队中有不同角色：
- **红脸（强硬）**：创始人/CEO，坚持底线
- **白脸（温和）**：CFO/顾问，寻求妥协

### 操作
1. 红脸先出场，设定强硬立场
2. 白脸出场，表示理解对方，寻求中间方案
3. 红脸最终批准"白脸争取到的方案"

### 注意事项
- 不要过度使用，否则显得不真诚
- 红脸白脸要配合默契
- 最终决策权在红脸

## 五、分阶段谈判

### 策略
将复杂交易分阶段谈判，每阶段解决一部分问题。

### 阶段划分
| 阶段 | 内容 | 时间 | 目标 |
|------|------|------|------|
| 第一阶段 | 估值+金额 | 1-2周 | 确定大致估值区间 |
| 第二阶段 | 经济条款 | 1-2周 | 确定清算优先、反稀释等 |
| 第三阶段 | 控制条款 | 1-2周 | 确定董事会、否决权等 |
| 第四阶段 | 保护条款 | 1周 | 确定赔偿、陈述保证等 |
| 第五阶段 | 杂项 | 3-5天 | 收尾 |

### 好处
- 每阶段聚焦，避免信息过载
- 每阶段达成阶段性共识，增强信心
- 如果前期分歧太大，可以及时止损

## 六、条件交换（Trade-off）

### 原则
每个让步都必须换取对方的让步。

### 交换矩阵
| 你的让步 | 换取对方的让步 | 交换比例 |
|----------|----------------|----------|
| 估值降到2.5亿 | 清算优先改为1x非参与 | 1:1 |
| 接受对赌条款 | 对赌惩罚上限设为投资额 | 1:1 |
| 给投资人1个董事席位 | 否决权限制为保护性清单 | 1:1 |
| 延长排他期到60天 | 对方承诺尽职调查效率 | 1:1 |
| 承担部分费用 | 对方承诺快速交割 | 1:1 |

### 禁忌
- 不要无条件让步
- 不要连续让步而不换取
- 不要在没有交换的情况下让步多个条款

## 七、时间压力

### 策略
利用时间压力创造紧迫感。

| 场景 | 时间压力 | 使用方式 |
|------|----------|----------|
| 资金快耗尽 | 真实压力 | "我们现有资金够跑3个月" |
| 竞品在融资 | 竞争压力 | "我们的竞品也在接触投资人" |
| 市场窗口 | 市场压力 | "这个估值窗口可能只开3个月" |
| 对方时间紧 | 对方压力 | "我们还有其他选择，但您的时间更宝贵" |

### 注意事项
- 不要虚构时间压力（容易被识破）
- 真实的时间压力更有说服力
- 给自己留缓冲时间

## 八、沉默的力量

### 策略
在对方报价后，保持沉默，让对方感到不安，从而主动改善报价。

### 操作
1. 对方报价后，不要立即回应
2. 沉默3-5秒（感觉很长）
3. 对方可能会主动补充："当然，这个还可以谈"
4. 或者你可以说："这个比我们的预期低一些..."

### 适用场景
- 对方报价低于预期
- 对方条款过于苛刻
- 你想争取更多思考时间

## 九、书面确认

### 原则
所有谈判结果必须书面确认，避免口头承诺。

### 流程
1. 谈判中：记录所有讨论要点
2. 谈判后：24小时内发送邮件确认
3. 确认内容：达成的共识、待确认的事项、下一步行动
4. 最终：签署TS或正式协议

### 邮件模板
```
主题：关于[日期]谈判的确认

[对方姓名]，

感谢今天的沟通。根据讨论，我们确认以下共识：
1. 估值：投前2.5亿（待最终确认）
2. 融资金额：5000万
3. 清算优先：1x非参与（已确认）
4. 反稀释：加权平均（已确认）

待确认事项：
1. 董事会席位：我们提议3:2，待您确认
2. 对赌条款：待下周讨论

下一步：
- 我方本周五前发送修订版TS
- 您下周三前反馈意见

请确认以上是否准确。

[你的姓名]
[日期]
```

## 十、谈判底线

### 定义
低于此底线，宁愿不交易。

### 设定方法
| 维度 | 底线 | 如何设定 |
|------|------|----------|
| 估值 | 不低于2.5亿 | 基于可比交易+成本法 |
| 股权 | 创始人保持>50% | 基于控制权需求 |
| 条款 | 无全棘轮、无无限赔偿 | 基于风险承受能力 |
| 时间 | 排他期不超过60天 | 基于机会成本 |

### 坚守底线
- 在谈判前明确底线，写下来
- 谈判中不要临时降低底线
- 如果接近底线，暂停谈判，重新评估
- 如果突破底线，礼貌拒绝："这个条款超出了我们的承受范围，我们可能需要重新考虑合作"

## 十一、常见谈判话术

| 投资人话术 | 你的回应 | 目的 |
|------------|----------|------|
| "我们只能给2亿" | "我们理解，但基于可比交易，最近3家都是3亿+，我们希望能接近这个水平" | 锚定到更高值 |
| "这是我们的标准条款" | "我们理解，但每个公司情况不同，我们的[xxx]条款是基于我们的特殊情况" | 打破标准条款假象 |
| "其他创始人都接受了" | "我们尊重其他创始人的选择，但我们的底线是[xxx]" | 不比较，坚持底线 |
| "如果不接受，我们可能不投了" | "我们理解您的立场，我们也希望找到双方都能接受的方案。如果确实无法达成一致，我们尊重您的决定" | 不威胁，但展示BATNA |
| "我们需要更多控制权" | "我们理解您的保护需求，但我们可以在[保护性条款]上满足您，而不是[控制条款]" | 区分保护和控制 |
| "这个条款是律师要求的" | "我们理解，但我们可以让双方律师一起找更合理的方案" | 把律师变成共同解决问题的人 |

## 十二、谈判后的复盘

### 复盘清单
- [ ] 哪些条款谈判成功？为什么？
- [ ] 哪些条款谈判失败？为什么？
- [ ] 对方的核心关切是什么？
- [ ] 我们的核心关切是否得到满足？
- [ ] 最终协议与我们的底线相比如何？
- [ ] 如果重来，我们会怎么做不同？
- [ ] 对方谈判风格如何？下次如何调整？

### 经验归档
将谈判经验归档到知识库，供未来谈判参考：
- 对方机构/投资人的谈判风格
- 对方常见条款和让步空间
- 我们的成功/失败策略
- 行业基准条款变化
