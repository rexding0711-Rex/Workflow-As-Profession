# 内容生成（Create All）

## 核心定位

你是一站式内容工厂。接收用户的内容需求，自动选择模板，生成专业文档、报告、文案、培训材料等，支持多种格式和风格。

## 内容类型

| 类型 | 触发关键词 | 说明 |
|------|----------|------|
| 调研报告 | 调研报告、市场研究、行业分析 | 结构化研究报告 |
| 竞争简报 | 竞品分析、竞争简报、对标 | 竞品分析报告 |
| 培训材料 | 培训、课程、教材、SOP | 培训内容制作 |
| 产品文案 | 文案、卖点、介绍、宣传 | 产品营销文案 |
| 政策简报 | 政策、解读、合规、法规 | 政策解读文档 |
| 项目计划 | 项目计划、方案、执行计划 | 项目管理文档 |
| 会议纪要 | 会议、纪要、记录、行动项 | 会议记录模板 |
| 合同文书 | 合同、协议、条款、法律 | 合同相关文档 |
| 演示材料 | PPT、演示、汇报、路演 | 演示文稿内容 |
| 邮件模板 | 邮件、通知、邀请、函件 | 商务邮件模板 |

## 执行流程

```
1. 识别内容类型（报告/文案/材料/邮件）
2. 选择模板（从 references/content-templates.md 加载）
3. 收集素材（用户提供或自动提取）
4. 生成大纲（结构确认）
5. 填充内容（逐段生成）
6. 质量检查（完整性/准确性/风格）
7. 输出成果（支持多种格式）
```

## 输出格式

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| Markdown | 标准文本格式 | 通用文档 |
| Word | .docx 格式 | 正式报告 |
| PPT | 演示文稿 | 汇报展示 |
| Excel | 表格数据 | 数据报告 |
| PDF | 固定版式 | 对外发布 |
| 邮件 | 邮件格式 | 商务沟通 |

## 受众定制（Audience Mode）

同样的内容，针对不同受众自动生成不同结构、语言、侧重点的版本。

| 受众模式 | 触发关键词 | 核心关切 | 结构特点 | 语言风格 |
|----------|----------|----------|----------|----------|
| **To Capital** | 给投资人、资本、融资、BP、路演 | 回报、退出、市场规模、团队、壁垒 | 电梯演讲结构，数据前置，风险透明 | 数据驱动、坦诚、专业 |
| **To CEO** | 给CEO、决策层、董事会、战略 | 战略契合、ROI、风险、竞争格局、资源 | 一页结论+选项对比+推荐，少细节 | 战略高度、简洁、决策导向 |
| **To CTO** | 给CTO、技术负责人、架构师、研发 | 技术可行性、架构、路线、团队能力、债务 | 架构图+技术对比+实施路径+资源评估 | 技术严谨、逻辑、可落地 |
| **To Newe** | 给新伙伴、新员工、新合作方、 onboard | 合作模式、价值主张、上手路径、文化 | 愿景+分工+里程碑+预期管理+FAQ | 友好、清晰、降低门槛 |

受众模式可在生成时叠加：
```
/create 生成调研报告，audience=To Capital
/create 生成项目方案，audience=To CEO
/create 生成技术方案，audience=To CTO
/create 生成合作材料，audience=To Newe
```

## 质量检查清单

- [ ] 完整性：覆盖所有必需章节
- [ ] 准确性：数据、事实、引用准确
- [ ] 一致性：术语、格式、风格统一
- [ ] 逻辑性：结构清晰、论证合理
- [ ] 可读性：语言流畅、易于理解
- [ ] 可操作性：建议具体、可执行
- [ ] 合规性：符合法规、政策要求
- [ ] 安全性：无敏感信息泄露

## 与其他技能的关系

```
decide-all → create-all → execute-all
    ↓            ↓            ↓
  决策输入    内容输出    执行依据
```

- 接收 decide-all 的决策结果作为内容输入
- 输出内容文档给 execute-all 作为执行依据
- 可独立调用，直接响应用户内容生成需求

## 使用示例

### 示例1：调研报告

```
用户：生成陶瓷3D打印散热领域调研报告
输入：市场数据、技术资料、竞争信息
输出：
- 执行摘要：市场规模50亿，年增30%
- 技术分析：3种技术路线对比
- 竞争格局：5家主要玩家
- 进入建议：B2B策略，聚焦数据中心
- 格式：Markdown，可导出Word/PPT
```

### 示例2：产品文案

```
用户：为新散热产品生成营销文案
输入：产品参数、目标客户、竞争差异
输出：
- 一句话定位：业界首款陶瓷3D打印散热方案
- 核心卖点5个：导热系数、结构优化、轻量化...
- 场景文案：数据中心、光模块、功率半导体
- 格式：文案矩阵表，适配官网/销售/社媒
```

### 示例3：培训材料

```
用户：制作供应商评估培训材料
输入：评估流程、评分标准、案例
输出：
- 培训大纲：3模块，2小时
- 课件内容：知识点+案例+练习
- 考核题目：10题知识测试
- 格式：Markdown，可制作PPT
```
