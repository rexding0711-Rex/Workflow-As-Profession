# 创业融资 Skill

## 何时触发

用户请求：
- 准备融资或商业计划书
- 理解投资条款（Term Sheet）
- 评估估值是否合理
- 谈判融资条件
- "怎么融资" / "这个TS条款有没有坑"

## 核心框架：融资阶段 + 估值方法 + 条款红线

### 步骤1：融资阶段匹配

| 阶段 | 公司状态 | 典型融资额 | 资金来源 | 关键指标 |
|------|----------|----------|----------|----------|
| 种子期 | 想法/MVP | 10-300万 | 天使/家人/朋友 | 团队背景/市场规模 |
| 天使轮 | 产品验证 | 300-1000万 | 天使投资人/孵化器 | PMF信号/早期用户 |
| Pre-A | 模式跑通 | 1000-3000万 | VC | 增长率/单位经济模型 |
| A轮 | 规模化 | 3000万-1亿 | VC | 收入/市场渗透率 |
| B轮+ | 市场扩张 | 1亿+ | VC/PE | 规模化增长率/盈利路径 |
| Pre-IPO | 上市准备 | 数亿 | PE/战略投资 | 财务规范/市场地位 |

**融资节奏原则**：
- 钱是工具，不是目的（不要为了融资而融资）
- 融资要够18-24个月运营（到下一个里程碑）
- 市场热时多融，市场冷时少融但保留弹性

### 步骤2：估值方法论

**早期公司（无收入/少收入）**：
- **可比交易法**：看类似公司最近融资的估值
- **VC快速法**：创始人背景好、市场大、产品酷 → 拍脑袋
- **博克斯法**：为团队、产品、市场、商业模式各打分，加总

**成长期公司（有收入）**：
- **PS倍数法**：估值 = 年收入 × 行业PS倍数
- **DCF法**：自由现金流折现（需要预测，早期不准）
- **目标回报法**：投资人要求5-10倍回报，反推当前估值

**关键认知**：
- 早期估值是"谈"出来的，不是算出来的
- 估值高不一定是好事（可能下一轮融资困难）
- 稀释比估值更重要（10%的100亿 > 50%的10亿）

### 步骤3：Term Sheet核心条款

**必须理解的条款**：

1. **估值与股权结构**
   - Pre-money / Post-money 估值
   - 期权池（ESOP）是否从Pre-money里出
   - 创始人股份锁定（Vesting）

2. **控制权条款**
   - 董事会席位分配（不要让投资人占多数）
   - 保护性条款（Protective Provisions）范围
   - 否决权范围（日常经营还是重大事项）

3. **经济条款**
   - 清算优先权（Liquidation Preference）
     - 1x non-participating：最友好（拿回本金，剩余按股比）
     - 1x participating：中等（拿回本金+按股比分剩余）
     - >1x participating：高烈度（几倍优先）
   - 反稀释条款（Anti-dilution）
     - Full ratchet：投资人全保护（极端，避免）
     - Weighted average：加权平均（标准）
     - 无反稀释：最友好创始人

4. **退出条款**
   - 共同出售权（Tag-along）
   - 领售权（Drag-along，注意触发条件）
   - 优先购买权（Right of First Refusal）
   - 回购权（Redemption，注意时间窗口）

**红线清单**：
- 🚫 清算优先权 > 1x participating
- 🚫 Full ratchet 反稀释
- 🚫 创始人Vesting < 4年，其中<1年cliff
- 🚫 投资人单方可以强制公司回购
- 🚫 对赌业绩条款（早期公司避免）
- 🚫 创始人个人连带责任担保

### 步骤4：融资材料准备

**必备材料**：
- 一句话介绍（电梯演讲）
- 商业计划书（10-15页PPT）
- 财务模型（3年预测）
- 产品Demo/数据看板
- 团队介绍

**BP结构**：
1. 痛点与市场（为什么值得做）
2. 解决方案与产品（你怎么做）
3. 商业模式（怎么赚钱）
4. 市场规模（TAM/SAM/SOM）
5. 竞争格局（为什么是你）
6.  traction数据（证明你能做）
7. 团队（为什么你们能做）
8. 融资计划（要多少钱，做什么）

### 步骤5：谈判策略

**创始人视角**：
- 多个投资人同时谈（制造竞争）
- 先谈条款，再谈估值（条款有时比估值重要）
- 保留关键决策权（产品方向、后续融资、出售）
- 找律师审TS（花小钱避大坑）
- 好的投资人 > 高估值（资源、背书、后续轮）

**投资人视角**：
- 团队 > 市场 > 产品（早期）
- 单位经济模型 > 增长率（成长期）
- 退出路径清晰度（后期）

## 输出模板

```markdown
# [公司] 融资策略分析

## 1. 融资阶段定位
- 当前阶段：[种子/天使/A/B...]
- 公司状态：[描述]
- 推荐融资额：[X]万/亿
- 推荐用途：[18-24个月运营/扩张]

## 2. 估值评估
- 推荐估值方法：[可比交易/PS/博克斯]
- 合理估值区间：[X-Y]亿
- 稀释比例建议：出让 [10-20]%
- 期权池预留：[10-15]%

## 3. 条款红线检查
- 清算优先权：[是否安全]
- 反稀释条款：[类型/风险]
- 董事会控制：[创始人是否保留多数]
- 否决权范围：[是否合理]
- **总体条款风险**：[低/中/高]

## 4. 融资材料建议
- 一句话介绍：[建议文案]
- BP重点页：[优先准备哪几页]
- 数据准备：[需要哪些traction数据]

## 5. 投资人策略
- 目标投资人类型：[天使/VC/战略]
- 接触顺序：[先谁后谁]
- 竞争策略：[如何制造紧迫感]

## 6. 谈判要点
- 必须坚持的条款：[列出]
- 可以让步的条款：[列出]
- 底线：[如果突破则放弃]
```

## 使用示例

**用户输入**：
> "我们pre-A轮，TS里写了1.5x participating liquidation preference，有没有问题？"

**AI执行**：
1. 立即标注：🚫 红线！
2. 解释participating LP的含义：投资人先拿1.5倍本金，再按股比分剩余
3. 计算极端情景下的创始人剩余
4. 给出谈判建议：坚持1x non-participating
5. 解释替代方案：如果投资人坚持，可以谈判提高估值换取条款友好
