# 实体类型定义

## 公司（Company）

### 属性
| 属性 | 说明 | 示例 | 必填 |
|------|------|------|------|
| name | 公司全称 | 江苏富乐华半导体科技有限公司 | 是 |
| short_name | 简称 | 富乐华 | 是 |
| aliases | 别名/曾用名 | 富乐华科技 | 否 |
| industry | 所属行业 | 陶瓷基板 | 是 |
| sub_industry | 细分行业 | AMB陶瓷基板 | 否 |
| region | 地区 | 中国·江苏·盐城 | 是 |
| scale | 规模 | 员工500+，年收入5亿 | 否 |
| founded | 成立时间 | 2015 | 否 |
| stage | 发展阶段 | 成长期/B轮 | 否 |
| funding_status | 融资状态 | B轮，金额未披露 | 否 |
| website | 官网 | www.xxx.com | 否 |
| tags | 标签 | ["AMB", "陶瓷基板", "国产替代"] | 否 |
| description | 简介 | 国内领先的AMB陶瓷基板供应商 | 否 |

### 关键指标
| 指标 | 说明 | 来源 |
|------|------|------|
| 产能 | 产线产能 | 调研/访谈 |
| 收入 | 年收入 | 公开信息/访谈 |
| 市场份额 | 细分领域份额 | 行业报告/估算 |
| 技术路线 | 主导技术 | 产品分析 |
| 客户数 | 主要客户数量 | 访谈/公开 |
| 认证情况 | 客户认证/质量体系 | 访谈/证书 |

## 技术（Technology）

### 属性
| 属性 | 说明 | 示例 | 必填 |
|------|------|------|------|
| name | 技术名称 | AMB（活性金属钎焊） | 是 |
| type | 技术类型 | 工艺/材料/设备/设计 | 是 |
| maturity | 成熟度 | 量产/验证/研发/概念 | 否 |
| trl | 技术成熟度等级 | 8 | 否 |
| alternative | 替代技术 | DBC、DPC | 否 |
| advantages | 优势 | 导热好、可靠性高 | 否 |
| disadvantages | 劣势 | 成本高、工艺复杂 | 否 |
| applications | 应用场景 | 功率半导体、光模块 | 否 |
| key_params | 关键参数 | 导热系数、CTE、耐温 | 否 |
| tags | 标签 | ["陶瓷基板", "封装"] | 否 |

## 专家（Expert）

### 属性
| 属性 | 说明 | 示例 | 必填 |
|------|------|------|------|
| name | 姓名 | 张三 | 是 |
| title | 头衔/职位 | 教授/CTO/技术顾问 | 是 |
| organization | 所属机构 | 某大学/某公司 | 否 |
| field | 专长领域 | 陶瓷材料、热管理 | 是 |
| reliability | 可靠性 | 高/中/低 | 否 |
| contact | 联系方式 | 微信/电话（敏感，可选） | 否 |
| publications | 代表成果 | 论文20篇，专利10件 | 否 |
| tags | 标签 | ["陶瓷材料", "教授"] | 否 |
