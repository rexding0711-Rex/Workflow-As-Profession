# 关系类型定义

## 提供（SUPPLY）

### 说明
A向B提供产品/服务。最常见的产业链关系。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| product | 产品/服务名称 | AMB陶瓷基板 |
| price_range | 价格区间 | 100-200元/片 |
| volume | 供应规模 | 10万片/月 |
| start_date | 合作开始时间 | 2023-01 |
| status | 合作状态 | 合作中/洽谈中/已终止 |
| satisfaction | 满意度 | 高/中/低 |
| exclusivity | 是否独家 | 是/否 |

### 示例
- 富乐华 → 提供 → 某光模块厂商（AMB基板，100元/片，10万片/月）
- 某氮化铝粉厂商 → 提供 → 富乐华（氮化铝粉，50万/吨，2吨/月）

## 使用（USE）

### 说明
A使用B的技术/产品/服务。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| technology | 使用的技术 | DBC工艺 |
| version | 版本/型号 | 第2代 |
| start_date | 开始使用 | 2022-06 |
| satisfaction | 满意度 | 高/中/低 |
| usage_scope | 使用范围 | 全部产品/部分产品 |

### 示例
- 某光模块厂商 → 使用 → DBC工艺（用于低端产品）
- 某厂商 → 使用 → 富乐华的AMB基板（用于高端产品）

## 投资（INVEST）

### 说明
A投资B（股权/债权/项目）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| amount | 投资金额 | 5000万 |
| round | 轮次 | B轮 |
| date | 投资时间 | 2024-03 |
| equity | 股权比例 | 15% |
| valuation | 估值 | 3亿（投后） |
| type | 投资类型 | 股权/可转债/债权 |
| board_seat | 董事会席位 | 1席 |

### 示例
- 某产业基金 → 投资 → 富乐华（B轮，5000万，15%股权）
- 红杉 → 投资 → 某散热公司（A轮，3000万）

## 竞争（COMPETE）

### 说明
A与B在同一市场/技术/客户群竞争。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| domain | 竞争领域 | 陶瓷基板市场 |
| intensity | 竞争强度 | 高/中/低 |
| a_advantage | A的优势 | 价格低、响应快 |
| b_advantage | B的优势 | 技术好、品牌强 |
| market_share_a | A的市场份额 | 15% |
| market_share_b | B的市场份额 | 30% |

### 示例
- 富乐华 ↔ 竞争 ↔ 日本京瓷（陶瓷基板市场，京瓷技术强，富乐华价格低）
- 富乐华 ↔ 竞争 ↔ 罗杰斯（AMB市场，罗杰斯品牌强，富乐华国产替代）

## 合作（COOPERATE）

### 说明
A与B合作（联合研发、战略联盟、渠道合作等）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| cooperation_type | 合作类型 | 联合研发/渠道合作/战略联盟 |
| content | 合作内容 | 开发新一代AMB基板 |
| start_date | 开始时间 | 2023-06 |
| duration | 合作期限 | 3年 |
| outcome | 合作成果 | 申请专利3件，产品1款 |
| status | 合作状态 | 进行中/已完成/已终止 |

### 示例
- 富乐华 ↔ 合作 ↔ 某大学（联合研发，陶瓷材料，3年）
- 某光模块厂商 ↔ 合作 ↔ 某数据中心（联合测试，散热方案）

## 专长（EXPERTISE）

### 说明
A专长于B领域（技术/产品/行业）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| field | 专长领域 | 陶瓷材料 |
| depth | 专深度 | 专家级/资深/熟悉 |
| achievements | 代表成果 | 论文20篇，专利10件 |
| reputation | 行业声誉 | 高/中/低 |

### 示例
- 张三教授 → 专长 → 陶瓷材料（20年研究，论文50篇）
- 李四 → 专长 → AMB工艺（10年经验，专利15件）

## 隶属（BELONG）

### 说明
A隶属于B（子公司/部门/分支机构）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| relation_type | 隶属类型 | 子公司/分公司/部门 |
| equity_ratio | 股权比例 | 100%全资 |
| control_level | 控制程度 | 完全控制/重大影响 |
| establishment_date | 设立时间 | 2020-01 |

### 示例
- 富乐华子公司A → 隶属 → 富乐华（100%全资，负责研发）
- 富乐华分公司B → 隶属 → 富乐华（负责销售）

## 供应（SUPPLY_RAW）

### 说明
A向B供应原材料/零部件（上游关系）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| material | 原材料名称 | 氮化铝粉 |
| volume | 供应量 | 2吨/月 |
| price | 价格 | 50万/吨 |
| quality | 质量等级 | 高纯/普通 |
| stability | 供应稳定性 | 高/中/低 |

### 示例
- 某氮化铝粉厂商 → 供应 → 富乐华（氮化铝粉，2吨/月，50万/吨）
- 某设备商 → 供应 → 富乐华（AMB产线设备，1条/年）

## 采购（PURCHASE）

### 说明
A从B采购产品/设备/服务（下游关系）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| product | 采购产品 | AMB产线设备 |
| amount | 采购金额 | 2000万 |
| date | 采购时间 | 2023-06 |
| purpose | 用途 | 扩建产线 |

### 示例
- 富乐华 → 采购 → 某设备商（AMB产线设备，2000万）
- 富乐华 → 采购 → 某氮化铝粉厂商（氮化铝粉，100万/月）

## 影响（INFLUENCE）

### 说明
A影响B（政策影响行业、技术影响市场、事件影响公司）。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| influence_type | 影响类型 | 正面/负面/中性 |
| influence_mechanism | 影响机制 | 补贴/限制/标准/技术替代 |
| strength | 影响强度 | 强/中/弱 |
| duration | 影响持续时间 | 短期/中期/长期 |

### 示例
- 国产替代政策 → 影响 → 富乐华（正面，补贴+市场机会，长期）
- AI算力爆发 → 影响 → 陶瓷基板市场（正面，需求增长，长期）
- 日本地震 → 影响 → 氮化铝粉供应（负面，供应中断，短期）

## 参与（PARTICIPATE）

### 说明
A参与B项目/活动/事件。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| role | 参与角色 | 领投/跟投/联合研发/顾问 |
| contribution | 贡献 | 资金5000万/技术/资源 |
| start_date | 参与时间 | 2024-03 |
| status | 参与状态 | 进行中/已完成/退出 |

### 示例
- 某产业基金 → 参与 → 富乐华B轮（领投，5000万）
- 张三教授 → 参与 → 富乐华研发项目（顾问，技术咨询）

## 推出（LAUNCH）

### 说明
A推出B产品/技术/服务。

### 属性
| 属性 | 说明 | 示例 |
|------|------|------|
| product | 产品名称 | 新一代AMB基板 |
| launch_date | 推出时间 | 2024-06 |
| market | 目标市场 | 功率半导体 |
| performance | 性能指标 | 导热系数250W/mK |
| market_reception | 市场反馈 | 良好/一般/差 |

### 示例
- 富乐华 → 推出 → 新一代AMB基板（2024-06，导热系数250W/mK）
- 日本京瓷 → 推出 → 新型DBC基板（2024-03，成本降低20%）

## 关系强度

每条关系标注强度：

| 强度 | 说明 | 可视化 |
|------|------|--------|
| 强 | 深度合作/长期合作/核心关系 | 粗线（3px） |
| 中 | 一般合作/短期合作 | 中线（2px） |
| 弱 | 初步接触/传闻/一次性 | 细线（1px） |
| 潜在 | 有可能合作/正在洽谈 | 虚线 |
| 已终止 | 合作已终止 | 灰色线 |

## 关系方向

| 方向 | 说明 | 示例 |
|------|------|------|
| 单向 | A→B | 富乐华提供AMB基板给客户 |
| 双向 | A↔B | 富乐华与京瓷竞争 |
| 多向 | A→B, A→C | 富乐华供应给多家客户 |

## 关系验证

| 验证级别 | 说明 | 颜色 |
|----------|------|------|
| 已验证 | 有合同/访谈/公开信息确认 | 实线 |
| 待确认 | 有传闻/初步接触，未确认 | 虚线 |
| 推测 | 基于公开信息推测 | 浅灰色 |

## 关系时间

| 时间属性 | 说明 | 可视化 |
|----------|------|--------|
| 长期 | 合作>3年 | 深线 |
| 中期 | 合作1-3年 | 中线 |
| 短期 | 合作<1年 | 浅线 |
| 新建立 | 最近3个月 | 带标签"NEW" |
| 已终止 | 合作已终止 | 灰色线 |
