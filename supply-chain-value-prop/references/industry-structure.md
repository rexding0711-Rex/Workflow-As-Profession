# 汽车制造业供应链结构

## 典型 Tier 层级

```
OEM（整车厂）
  └─ 比亚迪 / 特斯拉 / 大众 / 丰田 / 吉利
     └─ Tier 1（系统供应商）
        └─ 博世 / 大陆 / 麦格纳 / 延锋 / 德赛西威
           └─ Tier 2（零部件/材料供应商）
              └─ 改性塑料 / 金属冲压 / 电子元件 / 橡胶密封
                 └─ Tier 3（原材料供应商）
                    └─ 树脂 / 钢材 / 铜线 / 天然橡胶
```

## Tier 2 供应商面临的核心压力

| 压力 | 来源 | 表现 |
|------|------|------|
| **降价压力** | Tier 1 + OEM | 年降 3-5% 是常态 |
| **认证壁垒** | OEM AVL | 一款材料进 AVL 需要 18-24 个月 |
| **被整合风险** | Tier 1 向上游延伸 | Tier 1 自建改性工厂 |
| **被替代风险** | 新材料/新工艺 | 金属替代塑料、热塑替代热固 |
| **碳合规压力** | OEM ESG 要求 | 需要 PCF/LCA 数据 |
| **多基地要求** | Tier 1 跟随 OEM 建厂 | 要求 near-site 供货 |

## 不同 OEM 的材料策略差异

| OEM | 材料偏好 | 供应商策略 | Tier 2 机会 |
|-----|---------|-----------|-----------|
| 比亚迪 | 成本优先，国产替代 | 供应商体系开放，扶持本土 | 机会大，但价格战激烈 |
| 特斯拉 | 创新优先，愿意试新材料 | 供应商体系开放，技术驱动 | 技术创新有机会，认证快 |
| 丰田 | 质量优先，长期稳定 | 供应商体系稳定，更换慢 | 进入难，但进入后稳定 |
| 大众 | 平台化，规格统一 | 体系成熟，要求多基地 | 需要匹配全球布局 |
| 新势力 | 速度优先，快速迭代 | 相对开放，认证周期短 | 进入门槛低，但车型生命周期短 |
