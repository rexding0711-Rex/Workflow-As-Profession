# Supply Chain Value Prop — 供应链价值主张

Tier 2 供应商面对的不是一个客户，是一整条链。
你的材料卖给了 Tier 1，但它最终要在 OEM 的终端产品里工作。
如果你只对 Tier 1 讲"我的材料便宜 5%"，你永远在比价。
如果你能对整条链讲"我的材料让终端产品更好"，你就在建立护城河。

## 何时触发

用户提及以下内容时触发：
- "我们的材料好，但 Tier 1 客户就是不换"
- "怎么让 OEM 知道我们 Tier 2 的价值？"
- "竞争对手比我们大 10 倍，怎么差异化？"
- "客户说'你们和 XX 差不多，谁便宜买谁'"
- "Tier 2 supplier differentiation strategy"
- "产业链定位"

**排除：** 此技能 NOT for：
- 技术卖点 → 用 `technical-spec-sell`
- RFQ 响应 → 用 `rfq-response`
- 销售流程 → 用 `industrial-sales-cycle`

## 开始之前

| # | 信息 | 示例 |
|---|------|------|
| 1 | **你的产品在产业链的位置** | "工程塑料 Tier 2 → 汽车 Tier 1 → OEM" |
| 2 | **你的直接客户** | "汽车内饰件 Tier 1 供应商" |
| 3 | **终端客户** | "比亚迪/特斯拉/大众" |
| 4 | **主要竞争对手** | "本土厂商 A（价格低）、国际厂商 B（品牌强）" |
| 5 | **当前被替代的威胁** | "Tier 1 在考虑自己上游整合" |

## 产业链价值传递地图

画出你的材料在整条链中的价值流动：

```
OEM 的需求 ———→ Tier 1 的需求 ———→ 你（Tier 2）能提供什么
"轻量化"          "易加工、良率高"    "密度低 15% + MFR 高 30%"
"降本"            "材料单价低"        "壁厚可减 → 每件 Total Cost 低"
"可持续"          "有 PCR 认证"       "30% PCR 含量 + LCA 报告"
"供应安全"        "不缺货"            "双基地供货 + 安全库存承诺"
```

### 使用这个地图

对 Tier 1 讲"他们的需求"，但要知道这些需求是从 OEM 传导下来的。
如果你的材料能帮 Tier 1 满足 OEM 的轻量化要求，你就不只是在卖材料，
你是在帮你的客户完成他们的 KPI。

## 差异化定位框架

### 四象限竞争定位

```
                   技术优势
                      ↑
         "技术利基者"  │  "全能领导者"
         （小但专）    │  （大而全）
                      │
  价格敏感 ──────────┼──────────→ 价值敏感
                      │
         "成本杀手"    │  "关系依赖者"
         （便宜）      │  （历史关系）
                      │
                   成本优势
```

1. **你在哪个象限？** 诚实定位
2. **你想去哪个象限？** 战略方向
3. **你的头部竞争对手在哪个象限？** 找到你的"不对称优势"

### 不对称竞争策略

当你面对比你大 10 倍的竞争对手时：

| 大公司的弱点 | 你的机会 |
|------------|---------|
| 反应慢，"改个配方要走 6 个月流程" | 快速响应，定制化方案 |
| 对小客户不够重视 | 给 Tier 1 提供大公司不提供的贴身服务 |
| 标准化产品，不灵活 | 按客户产线参数微调配方 |
| 决策链条长 | 客户可以直接和技术负责人沟通 |
| 不差钱，但也不着急 | 你比他们更想在客户身上成功 |

### 对 OEM 的"绕过策略"

有时你需要跳过 Tier 1 直接对 OEM 建立影响力：

**何时使用：**
- Tier 1 不愿意帮你推给 OEM（怕你绕过他们）
- OEM 的材料 Approved Vendor List (AVL) 是关键卡位

**怎么做（不得罪 Tier 1 的前提）：**
- 行业技术研讨会上的联合演讲（"我们和 Tier 1 客户 XX 合作开发了..."）
- 材料技术白皮书（发给 OEM 材料工程师）
- CAE 模拟能力展示（帮 OEM 做虚拟验证）

## 价值主张一句话公式

```
对于 [Tier 1 客户]
他们在 [具体场景] 中面临 [具体痛点]
我们的 [产品/方案] 提供 [独特能力]
不像 [竞争对手]
我们 [关键差异化]
这使 [客户能得到的具体结果]
```

**示例：**
"对于汽车内饰 Tier 1 供应商，
他们在新能源车内饰轻量化趋势下面临'传统材料太重、新金属太贵'的困境，
我们的改性 PP 材料提供密度低 20% + 刚性和韧性的平衡，
不像国际品牌的通用标准料，
我们可以根据客户的模具参数微调流动性和收缩率，
这使客户每辆车减重 3.2kg，同时材料成本降低 12%。"

## 应对"你们和 XX 差不多"的叙事框架

当客户说"你们和 XX 差不多，谁便宜买谁"时，你在打价格战。
说明你之前的沟通没有建立差异化。

**重启差异化对话：**

1. "我理解从表面参数看，我们的材料和 XX 确实接近。
   但我们帮[A 客户]做了一件事，XX 做不到——"

2. "你们现在用 XX 的材料，有什么让你头疼的吗？"
   （找到竞品解决不了的问题，哪怕很小）

3. "如果只看价格，我们确实不是最便宜的。
   如果看过去 12 个月因为批次不稳定导致的停线次数——那个成本可能更高。"

## 质量原则

1. **价值主张不是自嗨** — 不是"我们多牛"，是"我们帮客户解决了什么头疼的问题"
2. **相对竞品定义自己** — 你的差异化必须有参照系
3. **一条链上不能只有一个人觉得你好** — OEM 认可你，Tier 1 才会主动找你
4. **小公司的差异化不是"更便宜"** — 你在成本结构上不可能赢大公司，差异化在服务和灵活度
5. **价值主张每年审视一次** — 市场在变，竞争对手在变，你的定位也要变

## 使用示例

**用户输入：**
> 我们是做改性工程塑料的 Tier 2，客户是汽车 Tier 1。现在面临两个问题：一是国际巨头压价，二是客户的客户（OEM）在推材料的低碳化，我们还没有碳足迹数据。怎么定位？

**AI 执行过程：**
1. 分析产业链：OEM 的低碳需求传导到 Tier 1 → Tier 1 需要供应商提供碳数据
2. 当前定位：成本象限，但价格战打不过巨头
3. 建议转型方向："低碳材料先行者" — 立即开始做 LCA 和碳足迹报告
4. 短期策略：在等待认证期间，用快速响应和定制化服务稳住客户
5. 输出完整价值主张 + 转型路线图
