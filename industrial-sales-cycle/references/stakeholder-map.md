# 工业品多利益相关者管理地图

## Tier 1 客户典型决策链

```
经济买家 (Economic Buyer) —— 有权批预算的人
  └─ VP of Procurement / CPO
     关注：Total Cost、供应风险、战略关系

技术买家 (Technical Buyer) —— 评估技术可行性的人  
  └─ Engineering Manager / Materials Engineer
     关注：技术参数、加工性能、长期可靠性

用户买家 (User Buyer) —— 实际使用你的材料的人
  └─ 产线工程师 / 注塑车间负责人
     关注：好不好加工、会不会堵模、颜色稳不稳定

守门人 (Gatekeeper) —— 控制信息流的人
  └─ 质量经理 / 供应商质量工程师 (SQE)
     关注：认证、PPAP、批次报告、退货处理
```

## 接触策略速查

| 你要见的 | 谁可以引荐 | 敲门砖 |
|---------|----------|--------|
| 工程经理 | 采购经理 → "安排一次技术交流" | 新材料技术资料 + 竞品替代分析 |
| 质量经理 | 工程经理 → "我们的质量体系想请你们审核" | Pre-audit 报告 |
| VP/高管 | 采购经理 → "年度业务回顾" | 行业趋势洞察 + 降本方案 |
| 产线工程师 | 工程经理 → "我们想了解产线实际工况" | 现场试模支持 |

## 红牌信号

以下情况说明你的利益相关者覆盖不足：
- 只认识采购，没见过工程 → 技术关没打通
- 你的 champion 只是采购助理 → 没有实权
- 客户组织架构变动后，你的联系人换了 → 关系归零
- 客户的竞争对手也在接触你 → 利益冲突风险
