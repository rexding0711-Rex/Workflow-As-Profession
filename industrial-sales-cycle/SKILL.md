# Industrial Sales Cycle — 工业品长周期销售

工业品销售不是"聊得好就下单"。从初次接触到批量供货可能要 18 个月，
中间要过认证、试样、audit、谈判。整个过程像打通关游戏——
每一关过不去就从头再来。

## 何时触发

用户提及以下内容时触发：
- "我们的材料卖给 Tier 1 供应商，销售周期好长"
- "客户的采购说好，但工程部否决了"
- "试样做了 3 次还没过"
- "怎么推进 12 个月的销售周期不丢单？"
- "供应商认证卡住了怎么办？"
- "B2B industrial sales cycle management"

**排除：** 此技能 NOT for：
- RFQ/RFP 响应 → 用 `rfq-response`
- 技术卖点提炼 → 用 `technical-spec-sell`
- 供应链价值主张 → 用 `supply-chain-value-prop`

|---|------|------|

## 工业品销售生命周期

### 七阶段通关模型

```
1. 客户识别 (1-2 月)
   └─ 目标客户是否真的需要你的材料？替换成本多高？
2. 初次接触 (1-3 月)
   └─ 技术交流 + 样品提供 + 初步报价
3. 供应商认证 (2-4 月)
   └─ 工厂 audit、质量体系审核、环保合规检查
4. 试样验证 (2-6 月)
   └─ 小样→中样→大样，每一轮都可能挂
5. 商务谈判 (1-3 月)
   └─ 价格、账期、供货保障、质量协议
6. 首次量产 (1-2 月)
   └─ 首批供货，稳定性验证
7. 批量供货 (持续)
   └─ 从"新供应商"变成"战略供应商"
```

### 各阶段的推进信号与卡点

| 阶段 | 推进信号（绿灯） | 卡点（红灯） |
|------|---------------|-----------|
| 客户识别 | 客户正在寻找第二供应商（second source） | "我们现有供应商很好" — 替换动力不足 |
| 初次接触 | 工程部对你的技术资料感兴趣 | 只接触采购、没见到工程（采购有劲使不上） |
| 供应商认证 | Audit 排期已定，客户愿意承担 audit 费用 | "不符合我们的供应商标准" — 技术或合规差距 |
| 试样验证 | 小样通过，客户主动要求中样 | 连续 3 次不过 — 可能根本不是 fit |
| 商务谈判 | 双方交换了合同草案 | 只在价格上纠缠，说明没有足够差异化 |
| 首次量产 | PPAP/FAI 通过，量产订单下达 | 首批良率不达标 — 质量信心崩塌 |
| 批量供货 | 开始谈第二款材料 | 交付延迟、批次差异 — 供应商评级下降 |

## 多利益相关者管理

Tier 1 客户的决策链至少涉及 4 类人：

```
采购部 ─── 关注价格、账期、供应安全
工程部 ─── 关注技术参数、加工性能、一致性
质量部 ─── 关注认证、批次稳定性、退货率
高管层 ─── 关注战略关系、供应风险、Total Cost
```

### 四象限利益相关者策略

| 角色 | 痛点 | 你要展示的 | 接触方式 |
|------|------|----------|---------|
| **采购** | "现有供应商涨价/交期不稳定" | 价格竞争力 + 供应保障 | 报价单 + 供货协议 |
| **工程** | "现有材料加工窗口太窄" | 技术参数 + 试样数据 | 技术交流会 + 现场试模 |
| **质量** | "上一批和这一批不一样" | 质量体系 + 批次一致性数据 | 工厂 audit + 质检报告 |
| **高管** | "供应中断 = 停线" | 供应安全 + Total Cost 降低 | 年度业务回顾 + 战略合作提案 |

### "见不到关键人"怎么办

- **见不到工程部：** 通过采购安排技术交流会（"我们有一些新的材料技术可能对你们的下一代产品有帮助"）
- **见不到质量部：** 主动提出 pre-audit（"我们可以先做一次免费的自评，按你们的审核标准"）
- **见不到高管：** 用行业洞见做敲门砖（"我们观察到新能源车在 XXX 材料上有三个趋势..."）

## 试样推进策略

试样是工业品销售最长的瓶颈。平均需要 3-6 个月，2-3 轮验证。

```
第 1 轮 — 小样（实验室验证，1-2 月）
  "这材料在实验室条件下行不行？"
  关键：技术参数全部达标；如果不过，和工程一起分析原因

第 2 轮 — 中样（产线验证，2-3 月）
  "这材料在真实产线上行不行？"
  关键：工程师的实操反馈比检测报告更重要

第 3 轮 — 大样（批量验证，1-2 月）
  "1000 个零件，良率能不能稳定在 98%+？"
  关键：一致性和稳定性；这一轮通过基本等于量产
```

### 试样加速技巧

1. **让你们的工程师和他们的工程师直接对话** — 跳过采购和商务的来回传话
2. **每次试样前做 pre-check** — 按客户的标准先自测一遍，确保过关
3. **试样失败不要狡辩** — 客户工程师最讨厌供应商说"这没问题啊"。先解决问题，后讨论责任

## 质量原则

1. **长周期不等于没事做** — 每个月至少有一次有价值的客户接触，哪怕试样在进行中
2. **见不到工程 = 没开始卖** — 采购点头只是门票，工程认可才是通行证
3. **试样失败的默认回答是"我们来改"** — 不要争辩"这是你们产线的问题"
4. **和客户的竞争对手保持距离** — 材料行业的客户关系很敏感
5. **Total Cost > Unit Price** — 你的材料贵 10% 但良率高 5%，总成本反而低

## 使用示例

**用户输入：**
> 我们是做工程塑料的，想卖给一家汽车 Tier 1。已经见了采购经理，提供了资料和报价，但两个月了没下文。怎么办？

**AI 执行过程：**
1. 判断当前阶段：初次接触→供应商认证之间
2. 核心问题：只接触了采购，没见到工程部
3. 建议：通过采购约一次技术交流会，让己方工程师和客户工程师直接对技术参数
4. 输出利益相关者地图 + 工程部接触计划 + 技术交流会议程建议
