# 行业分析（Industry Analysis）

## 何时触发
用户需要快速理解一个行业，评估是否进入/投资/合作。例如：
- "帮我分析XXX行业"
- "这个行业值不值得进入"
- "行业趋势怎么看"
- 任何涉及"行业/赛道/市场"的分析

## 核心框架：1小时行业速写

### 行业分析四维度

**维度1：市场规模与增长**
- TAM（总市场）：理论上限
- SAM（可服务市场）：我能触达的
- SOM（可获得市场）：我能拿到的
- 增长率：过去5年CAGR，未来预测
- 关键驱动：谁在推动增长？（技术/政策/人口/消费）
- 生命周期：萌芽→成长→成熟→衰退

**维度2：竞争结构**
- 市场集中度：CR3/CR5/HHI指数
- 竞争格局：领导者/挑战者/跟随者/利基者
- 进入壁垒：资金/技术/牌照/渠道/品牌/网络效应
- 替代品威胁：是否有完全不同的解决方式
- 行业利润率：平均毛利率/净利率，趋势

**维度3：价值链与利润池**
- 价值链环节：上游→研发→生产→渠道→终端→服务
- 利润池分布：哪个环节最赚钱？
- 关键成功因素（KSF）：在这个行业赢需要什么
- 痛点/机会：价值链上哪里效率低、哪里被卡脖子

**维度4：关键趋势与变量**
- 技术趋势：AI/自动化/新材料等如何改变行业
- 政策趋势：监管放松还是收紧？补贴还是征税？
- 消费者趋势：需求变化、偏好迁移、新场景
- 供应链趋势：全球化→本地化？集中→分散？
- 3年后这个行业会怎样？关键变量是什么？

### 行业分析速查表

| 问题 | 数据来源 | 快速获取方式 |
|------|----------|-------------|
| 市场规模 | 行业报告、券商研报 | 搜索引擎、数据平台 |
| 增长率 | 历史数据、预测报告 | 行业白皮书、统计局 |
| 竞争格局 | 企业排名、市场份额 | 行业协会、媒体报道 |
| 利润率 | 上市公司财报、行业平均 | 同花顺/东方财富 |
| 进入壁垒 | 专家访谈、行业研究 | 咨询报告、知乎/36kr |
| 技术趋势 | 专利数据、技术报告 | 学术数据库、TechCrunch |
| 政策趋势 | 政府文件、监管动态 | 国务院/发改委/行业协会 |

## 输出模板

```markdown
# [行业] 1小时行业速写

## 1. 市场规模与增长
- TAM/SAM/SOM：[估算]
- 过去5年CAGR：[X%]
- 未来5年预测：[X%]
- 增长驱动：[技术/政策/人口/消费]
- 生命周期：[阶段]

## 2. 竞争结构
- 市场集中度：[CR3/CR5/HHI]
- 主要玩家：[领导者/挑战者/跟随者/利基者]
- 进入壁垒：[高/中/低，具体是什么]
- 替代品威胁：[是什么，程度]
- 行业平均利润率：[毛利率/净利率]
- 竞争趋势：[集中度提升/分散/稳定]

## 3. 价值链与利润池
- 价值链环节：[列出]
- 利润池分布：[哪个环节最赚钱？]
- 关键成功因素：[KSF列表]
- 行业痛点：[效率低/被卡脖子/变革机会]
- 创新机会：[技术/模式/渠道]

## 4. 关键趋势与变量
- 技术趋势：[AI/自动化/新材料...]
- 政策趋势：[监管方向]
- 消费者趋势：[需求变化]
- 供应链趋势：[全球化/本地化]
- 3年后展望：[情景推演]
- 关键变量：[最可能改变格局的因素]

## 5. 进入/投资评估
- 吸引力：[高/中/低]
- 最佳切入点：[价值链哪个环节/什么模式]
- 关键风险：[列出]
- 成功概率：[高/中/低]
- 如果我是创业者：[建议]
- 如果我是投资人：[建议]
```

## 经典案例

- **麦肯锡"新能源汽车"**：TAM=万亿级，增长=30%+，竞争=特斯拉/比亚迪/新势力，壁垒=技术+品牌+资金，趋势=智能化+全球化
- **BCG"预制菜"**：TAM=千亿级，增长=20%，竞争=分散，壁垒=供应链+品牌，趋势=标准化+年轻化，痛点=口味还原度
- **贝恩"SaaS"**：TAM=千亿级，增长=15%，竞争=微软/ Salesforce/国产替代，壁垒=切换成本+网络效应，趋势=AI集成+垂直化

## 使用示例

**用户输入**：
> "帮我分析预制菜行业，看看能不能进入"

**AI执行**：
1. 市场规模：TAM=5000亿，增长=15-20%
2. 竞争格局：CR5<10%，极度分散，没有绝对龙头
3. 价值链：上游（食材）→中游（加工）→下游（渠道），中游利润最薄
4. 关键趋势：标准化、年轻化、B端（餐饮）→C端（家庭）
5. 给出进入建议：B端切入，绑定连锁餐饮，逐步向C端延伸
