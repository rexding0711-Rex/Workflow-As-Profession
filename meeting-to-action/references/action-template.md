# 行动项输出模板

## 标准输出格式

```markdown
# [会议主题] 纪要

## 基本信息
- **时间**：YYYY-MM-DD HH:MM
- **参会人**：张三、李四、王五（共X人）
- **记录人**：AI助理
- **会议形式**：线上/线下/语音

---

## 议题与决策

### 议题1：[议题名称]
- **讨论内容**：[简要概括]
- **决策**：
  - 定了什么？（如果无决策，写"无明确决策，需进一步讨论"）
  - 条件/限制？
- **行动项**：
  | 事项 | 责任人 | 截止日期 | 优先级 |
  |------|--------|----------|--------|
  | ... | ... | ... | P0/P1/P2 |

### 议题2：...

---

## 信息同步（无行动项）
- [信息1]：大家已知，无需行动
- [信息2]：...

---

## 行动项汇总表

| # | 事项 | 责任人 | 截止日期 | 优先级 | 状态 |
|---|------|--------|----------|--------|------|
| 1 | | | | P0 | 待开始 |
| 2 | | | | P1 | 待开始 |

---

## 待澄清事项
- [事项1]：会上未明确，需会后确认（建议确认人：XX）
- [事项2]：...

---

## 下次会议（如有）
- **时间**：
- **议题**：
- **需准备**：
```

## 优先级定义

| 优先级 | 定义 | 典型场景 |
|--------|------|----------|
| **P0** | 阻塞级，不完成影响其他所有事项 | 前置条件、审批、关键确认 |
| **P1** | 高优先级，本周必须完成 | 客户交付、 deadline 临近 |
| **P2** | 正常优先级，按计划完成 | 常规工作、内部优化 |
| **P3** | 低优先级，有时间再做 | 改进项、调研、非紧急 |

## 分发格式

- **邮件版**：直接粘贴上方Markdown，参会人可回复确认
- **任务系统版**：提取行动项表格，批量导入（如飞书/钉钉/ Jira）
- **消息版**：仅发送行动项汇总表，适合群聊快速同步
