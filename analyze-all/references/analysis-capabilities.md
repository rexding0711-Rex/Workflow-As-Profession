# 分析能力清单

## 数据洞察

| 功能 | 说明 | 输入 | 输出 |
|------|------|------|------|
| 描述统计 | 数据分布、集中趋势、离散度 | 数值数据 | 统计摘要 |
| 趋势分析 | 时间序列趋势、季节性、周期性 | 时间序列数据 | 趋势报告 |
| 相关性分析 | 变量间关联、因果推断 | 多变量数据 | 相关矩阵 |
| 聚类分析 | 数据分组、客户分群 | 特征数据 | 分组结果 |
| 异常检测 | 离群值、异常模式 | 监控数据 | 异常报告 |

## 文档解析

| 功能 | 说明 | 输入 | 输出 |
|------|------|------|------|
| 合同审查 | 条款提取、风险识别、合规检查 | 合同文档 | 审查报告 |
| 政策解读 | 政策要点、影响评估、合规要求 | 政策文件 | 解读简报 |
| 财务分析 | 三表分析、比率计算、趋势评估 | 财务报表 | 财务分析 |
| 技术评估 | 技术路线、架构评审、可行性 | 技术文档 | 评估报告 |
| 专利分析 | 专利布局、侵权风险、技术趋势 | 专利文献 | 专利报告 |

## 多维分析

| 功能 | 说明 | 输入 | 输出 |
|------|------|------|------|
| 竞争分析 | 竞品对比、差异化、SWOT | 竞品数据 | 竞争报告 |
| 客户分析 | 画像、需求、满意度、流失 | 客户数据 | 客户洞察 |
| 市场分析 | 规模、细分、趋势、进入策略 | 市场数据 | 市场报告 |
| 供应链分析 | 供应商评估、成本分析、风险 | 供应链数据 | 供应链报告 |
| 技术路线分析 | 技术成熟度、替代风险、演进 | 技术数据 | 技术路线图 |

## 分析框架

| 框架 | 适用场景 | 输出 |
|------|----------|------|
| SWOT | 战略评估 | 优势/劣势/机会/威胁 |
| PEST | 宏观环境 | 政治/经济/社会/技术 |
| 波特五力 | 行业竞争 | 竞争强度评估 |
| 价值链 | 业务流程 | 价值创造分析 |
| BCG矩阵 | 产品组合 | 明星/现金牛/问号/瘦狗 |
| 安索夫矩阵 | 增长策略 | 市场/产品矩阵 |
| 麦肯锡7S | 组织能力 | 战略/结构/系统/风格/人员/技能/共享价值观 |
| 精益画布 | 商业模式 | 问题/解决方案/关键指标/独特卖点/门槛优势/渠道/客户/成本/收入 |

## 分析模式

| 模式 | 说明 | 触发条件 |
|------|------|----------|
| 探索性分析 | 数据初探，发现模式 | 新数据首次分析 |
| 诊断性分析 | 根因定位，问题诊断 | 出现异常/问题 |
| 预测性分析 | 趋势预测，未来预估 | 需要预测未来 |
| 规范性分析 | 方案推荐，最优决策 | 需要决策建议 |
| 对比分析 | 横向对比，差异分析 | 需要对比评估 |
| 因果分析 | 因果推断，机制分析 | 需要理解因果关系 |
