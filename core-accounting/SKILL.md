# 核心会计 Skill

## 何时触发

用户请求：
- 学习MBA会计课程
- 理解财务报表的编制逻辑
- 准备会计相关考试或面试
- 分析公司财务状况的会计基础

## 核心框架：会计信息架构 + 三表勾稽 + 财务分析

### 步骤1：会计信息基础

**会计恒等式**：
```
资产 = 负债 + 所有者权益
```
这是所有会计的基石。任何交易都保持这个等式平衡。

**会计假设**：
- 会计主体：企业独立于所有者
- 持续经营：假设企业不会马上倒闭
- 会计分期：把连续经营分成报告期
- 货币计量：用货币记录一切

**会计原则**：
- 权责发生制（Accrual）：收入/费用在发生时确认，非现金收付时
- 配比原则：收入与相关费用在同一期间确认
- 历史成本：资产按购买成本入账
- 稳健性：不确认不确定的收入，但确认可能的损失

### 步骤2：资产负债表（B/S）

**结构**：
```
资产
├── 流动资产
│   ├── 现金及等价物
│   ├── 应收账款
│   ├── 存货
│   └── 预付费用
├── 非流动资产
│   ├── 固定资产（PP&E）
│   ├── 无形资产
│   ├── 长期投资
│   └── 商誉

负债
├── 流动负债
│   ├── 应付账款
│   ├── 短期借款
│   ├── 应付工资
│   └── 应交税费
├── 非流动负债
│   ├── 长期借款
│   ├── 债券
│   └── 递延税负债

所有者权益
├── 实收资本/股本
├── 资本公积
├── 留存收益
└── 少数股东权益
```

**关键理解**：
- 资产负债表是**快照**（某一时点）
- 左边是资源，右边是资源的来源
- 留存收益是历年净利润的累积（减去分红）

### 步骤3：利润表（I/S）

**结构**：
```
营业收入
- 营业成本
= 毛利润
- 销售费用
- 管理费用
- 研发费用
- 财务费用
= 营业利润（EBIT）
- 所得税
= 净利润
```

**关键理解**：
- 利润表是**电影**（一段时期）
- 收入确认原则：何时可以确认收入？（控制权转移/服务完成）
- 费用匹配原则：费用与收入匹配
- EBITDA = EBIT + 折旧 + 摊销（剔除非现金成本的经营利润）

### 步骤4：现金流量表（C/F）

**结构**：
```
经营活动现金流
+ 投资活动现金流
+ 筹资活动现金流
= 现金净变动
+ 期初现金
= 期末现金
```

**与净利润的区别**：
- 净利润是权责发生制，现金流是收付实现制
- 核心差异：折旧、应收应付、存货、资本支出
- 经营现金流与净利润的关系是利润质量的关键指标

### 步骤5：三表勾稽关系

```
利润表净利润 → 资产负债表留存收益
利润表折旧 → 现金流量表加回
现金流量表期末现金 → 资产负债表现金
现金流量表资本支出 → 资产负债表固定资产增加
```

**常见错配信号**：
- 利润高但经营现金流低 → 应收账款/存货积压
- 利润低但经营现金流高 → 预收款/应付款增加
- 现金增加但负债也增加 → 借钱度日

### 步骤6：财务分析指标

**盈利能力**：
- ROA = 净利润 / 总资产（资产使用效率）
- ROE = 净利润 / 所有者权益（股东回报）
- ROIC = 税后营业利润 / 投入资本（真正创造价值）

**偿债能力**：
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = (流动资产 - 存货) / 流动负债
- 利息保障倍数 = EBIT / 利息费用

**营运能力**：
- 总资产周转率 = 营收 / 总资产
- 应收账款周转天数 = 平均应收账款 / (营收/365)
- 存货周转天数 = 平均存货 / (营业成本/365)
- 现金周期 = 存货周转天数 + 应收周转天数 - 应付周转天数

**成长能力**：
- 营收增长率
- 净利润增长率
- 可持续增长率 = ROE × 留存收益率

### 步骤7：管理会计（Managerial Accounting）

**成本分类**：
- 固定成本 vs 变动成本
- 直接成本 vs 间接成本
- 沉没成本 vs 增量成本
- 机会成本（MBA最爱考）

**本量利分析（CVP）**：
- 盈亏平衡点 = 固定成本 / (单价 - 单位变动成本)
- 经营杠杆 = 边际贡献 / 营业利润
- 财务杠杆 = EBIT / 税前利润

**预算管理**：
- 零基预算 vs 增量预算
- 滚动预算
- 预算松弛问题

**转移定价**：
- 成本加成法
- 市场价法
- 协商定价法
- 双重定价法

## 经典案例

- **安然（Enron）**：表外融资、SPE、收入确认造假
- **世通（WorldCom）**：将费用资本化，虚增利润
- **乐视**：关联交易、资金占用、表外债务
- **瑞幸咖啡**：虚构交易、伪造凭证
- **苹果**：负营运资本（利用供应商账期）
- **亚马逊**：长期低利润但高现金流（再投资）

## 使用示例

**用户输入**：
> "帮我理解一下苹果公司的资产负债表，为什么它有负的营运资本？"

**AI执行**：
1. 解释营运资本 = 流动资产 - 流动负债
2. 分析苹果的具体数字：现金少，应付账款多，应收账款少
3. 解释原因：强大的议价能力，让供应商给长账期，同时客户预付款或快速付款
4. 对比其他公司（如制造业营运资本为正）
5. 讨论这种模式的可持续性
