# 商业统计 Skill

## 何时触发

用户请求：
- 学习MBA统计学课程
- 数据分析、假设检验、回归分析
- 商业决策中的数据应用
- 准备统计考试或面试
- "怎么判断数据是否显著" / "回归分析怎么做"

## 核心框架：描述统计 + 概率 + 推断统计 + 回归分析

### 步骤1：描述统计

**数据类型**：
- 名义（Nominal）：分类，无顺序（性别、品牌）
- 序数（Ordinal）：有顺序，间距不等（满意度、等级）
- 区间（Interval）：有顺序，间距相等，无绝对零点（温度、IQ）
- 比率（Ratio）：有顺序，间距相等，有绝对零点（收入、销售额）

**集中趋势**：
- 均值（Mean）：平均数，受极端值影响
- 中位数（Median）：中间值，不受极端值影响
- 众数（Mode）：出现频率最高的值
- 选择：对称分布用均值，偏态分布用中位数，分类数据用众数

**离散程度**：
- 极差（Range）：最大值 - 最小值
- 四分位距（IQR）：Q3 - Q1，不受极端值影响
- 方差（Variance）：平均平方偏差
- 标准差（SD）：方差的平方根，与原始数据同单位
- 变异系数（CV）：SD / Mean，用于比较不同尺度的离散度

**分布形态**：
- 正态分布（Normal）：钟形，对称，均值=中位数=众数
- 偏态（Skewness）：
  - 右偏（正偏）：均值 > 中位数，长尾在右（收入分布）
  - 左偏（负偏）：均值 < 中位数，长尾在左（考试分数）
- 峰度（Kurtosis）：尾部厚度，正态为3，>3厚尾，<3薄尾

**可视化**：
- 柱状图：分类数据
- 直方图：连续数据分布
- 箱线图：中位数、四分位、异常值
- 散点图：两个变量关系
- 折线图：时间趋势

### 步骤2：概率基础

**基本概念**：
- 概率：事件发生的可能性，0到1
- 条件概率：P(A|B) = P(A∩B) / P(B)
- 独立事件：P(A∩B) = P(A) × P(B)
- 全概率公式：P(A) = Σ P(A|B_i) × P(B_i)
- 贝叶斯定理：P(A|B) = P(B|A) × P(A) / P(B)
  - 后验概率 = 似然 × 先验 / 证据
  - 商业应用：客户分类、风险评估、预测

**概率分布**：
- 离散分布：
  - 二项分布（Binomial）：n次独立试验，成功概率p，成功次数X ~ B(n,p)
  - 泊松分布（Poisson）：单位时间/空间内事件发生次数，λ = 平均率
- 连续分布：
  - 正态分布（Normal）：X ~ N(μ, σ²)，中心极限定理基础
  - t分布：小样本均值推断，自由度n-1
  - 卡方分布：方差检验、拟合优度检验
  - F分布：方差分析（ANOVA）

**中心极限定理（CLT）**：
- 样本均值分布趋近正态分布，无论总体分布如何
- 样本量越大越接近正态（通常n ≥ 30）
- 基础：几乎所有统计推断都依赖CLT

### 步骤3：统计推断

**抽样**：
- 简单随机抽样：每个样本等概率
- 分层抽样：按层分层，每层随机
- 整群抽样：抽群体，群内全部调查
- 系统抽样：按固定间隔抽样
- 抽样误差：样本统计量与总体参数的差异

**参数估计**：
- 点估计：用样本统计量估计总体参数（如样本均值估计总体均值）
- 区间估计：置信区间
  - 95% CI：有95%的置信度包含总体参数
  - 公式：均值CI = x̄ ± z × (σ/√n) 或 x̄ ± t × (s/√n)
  - 影响因素：样本量（n↑区间↓）、置信水平（↑区间↑）、标准差（↓区间↓）

**假设检验**：
- 原假设（H₀）：通常表示"无效果"、"无差异"
- 备择假设（H₁）：要证明的假设
- 检验步骤：
  1. 建立假设
  2. 选择显著性水平（α，通常0.05）
  3. 计算检验统计量
  4. 确定p值或临界值
  5. 做出决策：p < α → 拒绝H₀
- 两类错误：
  - 第一类错误（α）：拒绝真H₀（假阳性）
  - 第二类错误（β）：接受假H₀（假阴性）
  - 检验力（Power）= 1 - β：正确拒绝假H₀的概率
- 常用检验：
  - 单样本z检验/t检验：均值 vs 已知值
  - 双样本t检验：两组均值比较
  - 配对t检验：前后测比较
  - 卡方检验：分类变量独立性/拟合优度
  - F检验：方差比较
  - ANOVA：多组均值比较

**p值解释**：
- p值：在H₀为真的情况下，观察到当前或更极端结果的概率
- p < 0.05：统计显著（但不一定实际重要）
- p > 0.05：不显著（不能拒绝H₀）
- 注意：p值不是H₀为真的概率，也不衡量效果大小
- 效果大小（Effect Size）：实际意义，比p值更重要

### 步骤4：回归分析

**简单线性回归**：
- 模型：Y = β₀ + β₁X + ε
- Y：因变量，X：自变量，β₀：截距，β₁：斜率，ε：误差
- 最小二乘法（OLS）：最小化误差平方和
- 解释：β₁表示X变化1单位，Y平均变化β₁单位
- 假设：线性关系、误差独立、误差正态、同方差、无多重共线性

**多元回归**：
- 模型：Y = β₀ + β₁X₁ + β₂X₂ + ... + β_kX_k + ε
- 多重共线性：自变量高度相关，导致系数不稳定
- 虚拟变量：将分类变量纳入回归（0/1编码）
- 交互项：X₁×X₂，表示一个变量的效果取决于另一个变量

**回归诊断**：
- R²：解释变异比例，0-1，越接近1越好（但可能过拟合）
- 调整R²：考虑自变量数量，更可靠
- 标准误差：预测精度
- t检验：各系数是否显著（β ≠ 0）
- F检验：模型整体是否显著
- 残差分析：检查假设是否满足
- 异常值：Cook's Distance、杠杆值

**逻辑回归（Logistic）**：
- 因变量是二元（0/1）
- 预测概率：P(Y=1) = 1 / (1 + e^-(β₀+βX))
- 输出 odds ratio（优势比）
- 商业应用：客户流失预测、信用评分、购买预测

**模型选择**：
- 过拟合：模型太复杂，对训练数据好，对新数据差
- 欠拟合：模型太简单，不能捕捉模式
- 调整R²、AIC、BIC：模型选择标准
- 交叉验证：用新数据测试模型性能

### 步骤5：实验设计

**A/B测试基础**：
- 随机分组：实验组 vs 对照组
- 样本量计算：基于统计功效（通常80%）、显著性水平（5%）、最小可检测效果
- 假设检验：两组均值/比例差异
- 执行注意：一次只改一个变量、运行完整周期、避免偷看数据

**实验设计类型**：
- 完全随机设计：随机分配到各组
- 随机区组设计：先分组（区组），再随机
- 因子设计：多个因素同时测试，交互效应
- 拉丁方设计：控制两个干扰变量

**统计功效分析**：
- 功效：正确拒绝假H₀的概率
- 影响因素：效果大小、样本量、显著性水平、变异度
- 通常要求功效 ≥ 80%
- 功效不足：可能错过真实效果（第二类错误）

## 使用示例

**用户输入**：
> "我们做了A/B测试，实验组转化率12%，对照组10%，样本量各1000，这个结果可信吗？"

**AI执行**：
1. 计算比例差异检验：z检验
2. 计算标准误差：SE = √[p(1-p)(1/n₁+1/n₂)]
3. 计算z值和p值
4. 判断：p < 0.05？是否显著？
5. 计算置信区间：差异的95% CI
6. 计算效果大小：实际提升2个百分点，相对提升20%
7. 给出结论：统计显著 + 实际意义 + 建议是否推广
