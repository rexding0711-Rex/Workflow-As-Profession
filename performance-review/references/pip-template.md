# PIP 模板 — 绩效改进计划

## 使用说明

PIP 是一个**帮助工具**，不是"解雇的前奏"。好的 PIP 有 50-70% 的成功率。
在启动 PIP 之前确认：
- 员工已明确知道期望（≥2 次正式反馈）
- 绩效差距不是外部因素导致（资源、角色变更、市场变化）
- Manager 愿意投入时间和资源帮助改进

|---------|---------|---------|
| [具体的期望行为/结果] | [具体的实际行为/结果] | [差距的影响] |

## PIP 之后

### 如果成功
- 庆祝（但不需要大张旗鼓，真诚的认可即可）
- 恢复正常绩效管理周期
- Manager 继续定期 1-on-1 确保不会回退

### 如果不成功
- 保持尊重，不要让对方感到"被羞辱"
- 离职过渡尽量体面
- 团队沟通：保护隐私，仅说"XXX 决定离开"即可
