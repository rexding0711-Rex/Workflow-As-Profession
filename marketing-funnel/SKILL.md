# 市场营销 Skill

## 何时触发

用户请求：
- 设计或优化营销/增长策略
- 分析某个品牌的营销打法
- 提升获客/转化/留存
- 评估广告投放或渠道策略
- "怎么让更多人买" / "这个营销方案怎么样"

## 核心框架：AARRR + 4P/4C + 品牌定位三角

### 步骤1：用户旅程诊断（AARRR漏斗）

```
获客(Acquisition) → 激活(Activation) → 留存(Retention) → 收入(Revenue) → 推荐(Referral)
```

诊断问题：
- 哪个环节是最大瓶颈？（用数据说话）
- 用户在哪一步流失最多？
- 当前投入资源集中在哪个环节？是否匹配瓶颈？

**常见错误**：在漏斗顶部疯狂投广告，但留存率惨不忍睹，导致CAC永远无法回收。

### 步骤2：营销组合设计（4P → 4C）

传统4P（企业视角）：
- Product（产品）
- Price（价格）
- Place（渠道）
- Promotion（推广）

现代4C（用户视角）：
- Customer Solution（用户真正要什么）
- Cost to Customer（用户总成本，不只是价格）
- Convenience（获取便利性）
- Communication（双向沟通，不只是单向广告）

**关键转化**：不要问"我们怎么卖"，要问"用户为什么买"。

### 步骤3：品牌定位（定位三角）

```
        目标用户
       /        \
      /          \
   差异化        信任状
      \          /
       \        /
        品牌主张
```

- **目标用户**：为谁解决什么问题？（越具体越好）
- **差异化**：和竞品相比，有什么不同？（可验证的差异）
- **信任状**：凭什么让用户相信？（数据/背书/体验）
- **品牌主张**：一句话说清楚"选择我，你会得到什么"

**定位检验**：用特劳特定位四问
1. 你是什么？（品类）
2. 有何不同？（差异点）
3. 何以见得？（信任状）
4. 与我何干？（用户利益）

### 步骤4：增长引擎选择

三种可持续增长引擎：

| 引擎类型 | 核心机制 | 典型指标 | 适用场景 |
|---------|---------|----------|----------|
| **病毒式增长** | 用户带来用户 | K因子（每个用户带来新用户数） | 社交/协作/UGC产品 |
| **付费增长** | 买量赚钱 | LTV/CAC > 3，回收期 < 12月 | 高毛利/复购业务 |
| **粘性增长** | 老用户留存高 | 月流失率 < 5%，NPS > 50 | 高频/刚需/订阅业务 |

**关键判断**：
- 你的业务天然适合哪种引擎？
- 当前资源投入和引擎类型是否匹配？
- 单一引擎是否足够？还是需要组合？

### 步骤5：渠道策略与CAC优化

渠道评估矩阵：

| 渠道 | 流量规模 | 精准度 | 成本 | 可控性 | 可持续性 |
|------|----------|--------|------|--------|----------|
|  organic搜索 | 中 | 高 | 低 | 中 | 高 |
| 付费广告 | 高 | 中 | 高 | 高 | 中 |
| 内容营销 | 中 | 高 | 低 | 中 | 高 |
| KOL/口碑 | 中 | 高 | 中 | 低 | 高 |
| 地推/直销 | 低 | 极高 | 高 | 高 | 中 |
| 渠道分销 | 高 | 中 | 中 | 低 | 中 |

**CAC优化原则**：
- 不要把鸡蛋放一个篮子（渠道集中度风险）
- 免费/低成本渠道优先验证PMF
- 付费渠道要在LTV/CAC > 3时才大规模投放

## 输出模板

```markdown
# [业务/品牌] 营销策略分析

## 1. 漏斗诊断
- 当前瓶颈环节：[AARRR环节]
- 关键数据：[各环节转化率/留存率]
- 资源错配：[是否投入和瓶颈匹配]

## 2. 用户洞察
- 目标用户画像：[描述]
- 核心需求/痛点：[是什么]
- 购买决策因素：[优先级排序]
- 现有解决方案不满：[和竞品比差在哪]

## 3. 品牌定位
- 品类：[你是什么]
- 差异点：[有何不同]
- 信任状：[何以见得]
- 用户利益：[与我何干]
- 定位语：[一句话版本]

## 4. 增长引擎
- 推荐引擎：[病毒/付费/粘性/组合]
- 引擎健康度：[指标评估]
- 优化方向：[具体建议]

## 5. 渠道策略
- 现有渠道组合：[列表]
- CAC/LTV比：[各渠道]
- 优化建议：[增加/减少/调整]

## 6. 行动清单
- 立即执行（本周）：[1-2项]
- 短期优化（本月）：[2-3项]
- 中期建设（本季）：[1-2项]
```

## 营销案例库

- **瑞幸**：裂变+补贴→数据驱动→品牌升级（付费增长→粘性增长迁移）
- **完美日记**：DTC+KOL矩阵→私域沉淀（付费增长+病毒增长组合）
- **Shein**：供应链快反+算法推荐+社媒引流（付费增长+粘性增长）
- **飞书**：PLG（产品驱动增长）+ 口碑扩散（病毒式增长）
- **华为ToB**：铁三角+解决方案销售（高触达+高信任）

## 使用示例

**用户输入**：
> "我做了个SaaS工具，帮助企业自动化报销，现在怎么获客？"

**AI执行**：
1. 判断产品类型：B2B SaaS，中高客单价，决策链长
2. 推荐增长引擎：付费增长（内容营销+SEM）+ 粘性增长（产品体验）
3. 不建议病毒式增长（B2B天然K因子低）
4. 设计渠道策略：内容营销建立信任 → SEM捕获需求 → 销售跟进 → 客户成功留存
5. 输出完整策略文档
