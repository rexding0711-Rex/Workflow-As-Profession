# 财务分析 Skill

## 何时触发

用户请求：
- 分析某家公司财务状况
- 读懂财报/三表（资产负债表、利润表、现金流量表）
- 评估投资标的或合作方财务健康度
- 建立财务模型或做预测
- "这家公司有没有雷" / "这个生意赚不赚钱"

## 核心能力：10分钟财报速写

### 步骤1：利润表速读（赚不赚钱）

关键指标：
- **毛利率** = (营收 - 营业成本) / 营收 → 产品竞争力
- **营业利润率** = 营业利润 / 营收 → 运营效率
- **净利率** = 净利润 / 营收 → 最终盈利能力
- **费用率** = (销售+管理+研发+财务) / 营收 → 成本结构

**红线检查**：
- 毛利率是否持续下降？（产品竞争力恶化）
- 费用率是否异常高于同行？（管理失控）
- 净利润与经营现金流是否严重背离？（利润质量差）

### 步骤2：资产负债表速读（安不安全）

关键指标：
- **资产负债率** = 总负债 / 总资产 → 财务杠杆
- **流动比率** = 流动资产 / 流动负债 → 短期偿债能力
- **速动比率** = (流动资产 - 存货) / 流动负债 → 更严格的短期偿债
- **应收账款周转天数** = 平均应收账款 / (营收/365) → 回款能力
- **存货周转天数** = 平均存货 / (营业成本/365) → 库存管理

**红线检查**：
- 应收账款暴增但营收没同步增长？（向渠道压货）
- 存货积压但毛利率上升？（可能虚增利润）
- 商誉占总资产比例过高？（并购风险）
- 有息负债是否过度依赖短债？（流动性风险）

### 步骤3：现金流量表速读（钱从哪来，到哪去）

三流分析：
- **经营活动现金流** → 生意本身是否造血
- **投资活动现金流** → 是否在扩张或收缩
- **筹资活动现金流** → 融资还是还债

**红线检查**：
- 经营现金流持续为负？（生意模式有问题）
- 利润为正但经营现金流为负？（盈利质量极低）
- 投资现金流大量流出但无明确方向？（盲目投资）
- 筹资现金流持续流入？（依赖输血）

### 步骤4：杜邦分析（ROE拆解）

ROE = 净利率 × 总资产周转率 × 权益乘数

| 驱动因素 | 含义 | 高ROE来源 |
|---------|------|----------|
| 净利率高 | 产品暴利 | 品牌/技术/垄断 |
| 周转率高 | 周转快 | 零售/平台/轻资产 |
| 杠杆高 | 借钱生钱 | 金融/地产/重资本 |

**关键判断**：这家公司的ROE是靠什么驱动的？可持续吗？

### 步骤5：同行对比（相对位置）

必须对比的维度：
- 毛利率 vs 行业均值（产品竞争力）
- 费用率 vs 行业均值（运营效率）
- ROE vs 行业均值（综合回报）
- 负债率 vs 行业均值（风险偏好）
- 现金流/利润比 vs 行业均值（盈利质量）

## 现金流模式识别

| 模式 | 经营 | 投资 | 筹资 | 含义 | 典型公司 |
|------|------|------|------|------|----------|
| 奶牛型 | + | - | - | 成熟企业，自给自足分红 | 茅台、水电 |
| 扩张型 | + | - | +/- | 成长期，投资扩张 | 优秀制造业 |
| 输血型 | - | - | + | 持续烧钱，依赖融资 | 多数初创公司 |
| 衰退型 | - | + | - | 卖资产度日，收缩 | 转型期企业 |
| 套利型 | + | + | - | 投资赚钱，主业一般 | 多元化集团 |

## 财务造假预警信号

1. 毛利率远高于同行且无法解释
2. 应收账款增长 > 营收增长，且周转天数持续增加
3. 经营现金流与净利润长期背离
4. 存货增长与营收增长不匹配
5. 在建工程长期挂账不转固
6. 其他应收款/其他应付款异常
7. 频繁更换审计师
8. 大股东高比例质押

## 使用示例

**用户输入**：
> "帮我看看比亚迪的财务情况，值不值得投资"

**AI执行**：
1. 通过 kimi_finance_v2 或 kimi_datasource_call_v2 获取比亚迪财报数据
2. 按5个步骤完成财务速写
3. 输出分析文档
4. 提示用户："需要我对比亚迪和同行（特斯拉、蔚来、小鹏）做横向对比吗？"
