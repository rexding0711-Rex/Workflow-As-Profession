# 战略管理 Skill

## 何时触发

用户请求分析以下任一问题时：
- 某个业务/公司/产品的竞争格局
- 行业趋势判断或赛道选择
- 商业模式设计或评估
- 战略定位或差异化策略
- "这个生意能不能做" / "怎么破局"

## 核心框架：五力 + 价值链 + 战略时钟

### 步骤1：行业吸引力判断（五力模型）

用5个维度评估行业是否值得进入：

1. **现有竞争者强度**
   - 市场集中度（CR3/CR5）
   - 竞争方式（价格战/差异化/服务）
   - 退出壁垒高低

2. **潜在进入者威胁**
   - 进入壁垒（资金/技术/牌照/渠道/品牌）
   - 现有玩家的反击能力
   - 新进入者的差异化可能

3. **替代品威胁**
   - 满足相同需求的其他方式
   - 替代品的性价比趋势
   - 用户切换成本

4. **供应商议价能力**
   - 供应商集中度
   - 转换成本
   - 前向一体化威胁

5. **买方议价能力**
   - 买方集中度
   - 产品标准化程度
   - 买方信息透明度

**输出格式**：每个维度用 ✅ 高吸引力 / ⚠️ 中等 / ❌ 低吸引力 标注，给出总体判断。

### 步骤2：价值链定位（价值链分析）

识别行业价值链上的关键增值环节：

```
上游 → 研发/采购 → 生产/组装 → 营销/品牌 → 渠道/分销 → 服务/售后 → 下游
```

关键问题：
- 哪个环节利润率最高？
- 哪个环节被卡脖子？
- 哪个环节最容易建立壁垒？
- 目标公司目前在哪个环节？想往哪个环节迁移？

### 步骤3：竞争战略选择（战略时钟）

根据产品价值 vs 价格定位，选择战略路径：

| 路径 | 策略 | 适用场景 |
|------|------|----------|
| 路径1 | 低价低值（集中成本领先） | 下沉市场、标准化产品 |
| 路径2 | 低价同值（成本领先） | 规模效应明显、效率驱动 |
| 路径3 | 同价高值（差异化） | 品牌/技术/体验可溢价 |
| 路径4 | 高价高值（聚焦差异化） | 高端市场、利基市场 |
| 路径5 | 高价低值（收割/寻租） | 有垄断/牌照壁垒 |

**红线**：不要走中间路线（同价同值），那是绞肉机。

### 步骤4：动态视角（竞争演化）

- 行业处于什么生命周期？（萌芽/成长/成熟/衰退）
- 技术变革是否正在重塑行业？
- 政策/监管趋势是友好还是收紧？
- 3年后这个行业会是什么样子？

## 输出模板

每次分析必须包含：

```markdown
# [业务/公司] 战略分析

## 1. 行业吸引力（五力模型）
- 现有竞争者：[强度等级 + 关键证据]
- 潜在进入者：[威胁等级 + 壁垒分析]
- 替代品：[威胁等级 + 替代路径]
- 供应商：[议价能力 + 依赖度]
- 买方：[议价能力 + 转换成本]
- **综合判断**：[值得进入/谨慎观望/不建议]

## 2. 价值链定位
- 核心增值环节：[环节名称]
- 目标公司当前位置：[位置]
- 迁移方向建议：[上游/下游/整合]

## 3. 竞争战略建议
- 推荐路径：[路径编号 + 名称]
- 关键假设：[需要什么条件成立]
- 风险点：[如果假设不成立会怎样]

## 4. 动态视角
- 行业生命周期：[阶段]
- 关键变量：[未来3年最可能改变格局的因素]
- 情景推演：[乐观/中性/悲观三种情景]

## 5. 行动清单（So What?）
- 如果我是创业者：[具体建议]
- 如果我是投资人：[判断角度]
- 如果我是从业者：[能力储备方向]
```

## 案例库（内置）

分析以下经典案例时，参考已知结论但鼓励重新推导：
- 拼多多下沉战略（路径2→路径3迁移）
- 苹果垂直整合（价值链上移）
- 小米生态链（成本领先 + 网络效应）
- 比亚迪一体化（应对供应商议价能力）
- Shein快时尚（价值链重构）

## 使用示例

**用户输入**：
> "我想分析一下预制菜行业，看看能不能进入"

**AI执行**：
1. 先问用户："你想从哪个角度切入？创业者、投资人还是从业者？"
2. 按五力模型逐步分析预制菜行业
3. 输出完整战略分析文档
4. 将分析结果保存到用户的记忆文件 `my-business-cases.md`
