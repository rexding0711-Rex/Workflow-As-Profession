# 社交媒体运营（Social Media）

## 核心能力

社交媒体 = 内容 × 平台算法 × 用户关系。
本 Skill 覆盖多平台内容策略、分发节奏、数据分析和增长方法论。

## 工作流

### Step 1：平台选择与定位

| 平台 | 内容形态 | 适合场景 | 算法偏好 |
|------|---------|---------|---------|
| 微信公众号 | 长图文 | 深度内容、品牌建设、B2B | 订阅关系 + 社交推荐 |
| 小红书 | 图文笔记 | 种草、生活方式、女性用户 | 内容质量 + 互动率 |
| 抖音 | 短视频 | 泛娱乐、品牌曝光、带货 | 完播率 + 互动率 |
| 微博 | 短图文 + 话题 | 热点、明星、公关 | 时效性 + 转发链 |
| B站 | 中长视频 | 深度内容、Z世代、知识区 | 一键三连 + 弹幕 |
| LinkedIn | 专业内容 | B2B、职场、招聘 | 职业相关性 |
| Twitter/X | 短文本 | 科技、金融、国际传播 | 时效性 + 引用 |

**一句话定位**：你的目标用户在哪个平台花时间，就去哪个平台。

### Step 2：内容策略

**内容矩阵**：
```
40% 价值内容（干货、教程、行业洞察） — 建立信任
30% 互动内容（话题讨论、投票、问答） — 提升互动
20% 品牌内容（案例、故事、文化） — 建立认知
10% 转化内容（限时优惠、白皮书下载） — 驱动转化
```

**内容日历模板**：
| 日期 | 平台 | 内容主题 | 形式 | 标签 | 负责 |
|------|------|---------|------|------|------|
| 周一 | 公众号 | 行业周报 | 长图文 | #行业趋势 | |
| 周三 | 小红书 | 产品测评 | 图文笔记 | #好物推荐 | |
| 周五 | 抖音 | 幕后花絮 | 短视频 | #公司日常 | |

### Step 3：增长策略

**自然增长**：
- SEO 优化（标题含关键词、标签精准）
- 跨平台引流（公众号→小红书，抖音→微信）
- 社群裂变（转发有礼、邀请机制）
- UGC 激励（用户投稿、话题挑战）

**付费增长**：
- 信息流广告（按 ROI 调优）
- KOL/KOC 合作（金字塔：头部造势 + 腰部种草 + 素人铺量）
- 平台挑战赛/话题活动

### Step 4：数据复盘

**核心指标看板**：

| 指标类型 | 指标 | 健康值参考 |
|---------|------|-----------|
| 曝光 | 阅读量/播放量 | 对比同类内容 |
| 互动 | 点赞率、评论率、转发率 | 点赞率>3% |
| 转化 | 点击率、下载量、留资量 | CTR>1% |
| 增长 | 新增关注、净增粉 | 月增>5% |
| 效率 | 单粉成本（CPF） | 持续下降趋势 |

**复盘模板**：
- 本月 Top 3 内容 & 成功原因
- 本月 Bottom 3 & 失败教训
- 下月内容方向调整
- 预算使用率与 ROI

## 注意事项
- 不要每个平台都做——先做好 1-2 个
- 内容质量 > 发布频率（周更一篇好文 > 日更水文）
- 算法会变，内容为王不变
- 私域（微信）是长期资产，公域是流量入口
